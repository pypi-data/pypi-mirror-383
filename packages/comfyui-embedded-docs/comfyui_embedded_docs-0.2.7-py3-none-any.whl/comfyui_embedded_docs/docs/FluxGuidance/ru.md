## Входы

| Параметр | Тип данных | Описание |
|----------------|-----------|-------------|
| `условие` | CONDITIONING | Входные данные кондиционирования, обычно полученные на предыдущих шагах кодирования или обработки |
| `руководство` | FLOAT | Управляет влиянием текстовых промптов на генерацию изображения. Диапазон регулировки от 0.0 до 100.0 |

## Выходы

| Параметр | Тип данных | Описание |
|----------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Обновленные данные кондиционирования, содержащие новое значение `руководство` |
