Узел Rodin 3D Smooth генерирует 3D-объекты с использованием Rodin API, обрабатывая входные изображения и преобразуя их в сглаженные 3D-модели. Он принимает несколько изображений на вход и создает загружаемый файл 3D-модели. Узел автоматически обрабатывает весь процесс генерации, включая создание задачи, опрос статуса и загрузку файла.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | Да | - | Входные изображения, используемые для генерации 3D-модели |
| `Seed` | INT | Да | - | Значение случайного сида для обеспечения воспроизводимости генерации |
| `Material_Type` | STRING | Да | - | Тип материала, применяемого к 3D-модели |
| `Polygon_count` | STRING | Да | - | Целевое количество полигонов для генерируемой 3D-модели |

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | Путь к файлу загруженной 3D-модели |
