Узел ModelMergeAuraflow позволяет объединять две различные модели, регулируя специфические веса смешивания для различных компонентов модели. Он обеспечивает детальный контроль над тем, как различные части моделей объединяются, от начальных слоёв до финальных выходных данных. Этот узел особенно полезен для создания пользовательских комбинаций моделей с точным контролем над процессом слияния.
