Узел Mahiro изменяет функцию управления, чтобы больше фокусироваться на направлении позитивного промпта, а не на разнице между позитивными и негативными промптами. Он создает модифицированную модель, которая применяет пользовательский подход масштабирования управления с использованием косинусного сходства между нормализованными условными и безусловными денойзенными выходами. Этот экспериментальный узел помогает более сильно направлять генерацию в сторону задуманного направления позитивного промпта.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | | Модель, которая будет модифицирована с измененной функцией управления |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `patched_model` | MODEL | Модифицированная модель с примененной функцией управления Mahiro |
