Узел StableCascade_SuperResolutionControlnet подготавливает входные данные для обработки супер-разрешения Stable Cascade. Он принимает входное изображение и кодирует его с использованием VAE для создания входных данных controlnet, одновременно генерируя заполняющие латентные представления для стадии C и стадии B конвейера Stable Cascade.

## Входные параметры

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для обработки с целью супер-разрешения |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования входного изображения |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `controlnet_input` | IMAGE | Закодированное представление изображения, подходящее для входных данных controlnet |
| `stage_c` | LATENT | Заполняющее латентное представление для стадии C обработки Stable Cascade |
| `stage_b` | LATENT | Заполняющее латентное представление для стадии B обработки Stable Cascade |
