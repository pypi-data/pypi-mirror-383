Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/vae`, а также читает модели из дополнительных путей, настроенных в файле extra_model_paths.yaml. Иногда может потребоваться **обновить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей из соответствующей папки.

Узел VAELoader предназначен для загрузки моделей вариационных автоэнкодеров (VAE) и специально адаптирован для работы как со стандартными, так и с аппроксимированными VAE. Он поддерживает загрузку VAE по имени, включая специализированную обработку моделей 'taesd' и 'taesdxl', и динамически настраивается в зависимости от конкретной конфигурации VAE.

## Входы

| Поле   | Тип данных Comfy | Описание                                                                                   |
|---------|-------------------|-----------------------------------------------------------------------------------------------|
| `название_vae` | `COMBO[STRING]`    | Определяет имя загружаемого VAE, указывая, какая модель VAE должна быть получена и загружена. Поддерживает диапазон предопределённых имён VAE, включая 'taesd' и 'taesdxl'. |

## Выходы

| Поле | Тип данных | Описание                                                              |
|-------|-------------|--------------------------------------------------------------------------|
| `vae`  | `VAE`       | Возвращает загруженную модель VAE, готовую для дальнейших операций, таких как кодирование или декодирование. Выходные данные представляют собой объект модели, инкапсулирующий состояние загруженной модели. |
