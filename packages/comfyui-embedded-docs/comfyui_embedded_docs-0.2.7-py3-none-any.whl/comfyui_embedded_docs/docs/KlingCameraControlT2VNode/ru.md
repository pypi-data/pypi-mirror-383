Узел Kling Text to Video Camera Control преобразует текст в кинематографические видео с профессиональными движениями камеры, имитирующими реальную киносъёмку. Этот узел поддерживает управление виртуальными действиями камеры, включая масштабирование, вращение, панорамирование, наклон и вид от первого лица, сохраняя фокус на вашем исходном тексте. Длительность, режим и название модели жёстко заданы, поскольку управление камерой поддерживается только в профессиональном режиме с моделью kling-v1-5 и длительностью 5 секунд.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Положительный текстовый промпт |
| `negative_prompt` | STRING | Да | - | Негативный текстовый промпт |
| `cfg_scale` | FLOAT | Нет | 0.0-1.0 | Определяет, насколько точно выходные данные следуют промпту (по умолчанию: 0.75) |
| `aspect_ratio` | COMBO | Нет | "16:9"<br>"9:16"<br>"1:1"<br>"21:9"<br>"3:4"<br>"4:3" | Соотношение сторон для генерируемого видео (по умолчанию: "16:9") |
| `camera_control` | CAMERA_CONTROL | Нет | - | Может быть создан с помощью узла Kling Camera Controls. Управляет движением и перемещением камеры во время генерации видео. |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео с эффектами управления камерой |
| `video_id` | STRING | Уникальный идентификатор для сгенерированного видео |
| `duration` | STRING | Длительность сгенерированного видео |
