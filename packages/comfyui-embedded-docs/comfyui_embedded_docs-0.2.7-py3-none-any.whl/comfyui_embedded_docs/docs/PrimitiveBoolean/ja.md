Booleanノードは、ワークフロー内でブール値（true/false）を渡すためのシンプルな方法を提供します。ブール値の入力を受け取り、同じ値を変更せずに出力するため、他のノードのブールパラメータを制御することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `値` | BOOLEAN | はい | true<br>false | ノードを通じて渡すブール値 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `output` | BOOLEAN | 入力として提供されたものと同じブール値 |
