Узел LoadAudio загружает аудиофайлы из входной директории и преобразует их в формат, который может обрабатываться другими аудиоузлами в ComfyUI. Он считывает аудиофайлы и извлекает как данные формы волны, так и частоту дискретизации, делая их доступными для последующих задач обработки аудио.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `аудио` | AUDIO | Да | Все поддерживаемые аудио/видеофайлы во входной директории | Аудиофайл для загрузки из входной директории |

**Примечание:** Узел принимает только аудио и видеофайлы, которые находятся во входной директории ComfyUI. Файл должен существовать и быть доступным для успешной загрузки.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | Аудиоданные, содержащие информацию о форме волны и частоте дискретизации |
