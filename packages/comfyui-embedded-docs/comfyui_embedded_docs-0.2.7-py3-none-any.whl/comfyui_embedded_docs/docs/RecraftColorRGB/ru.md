Создайте цвет Recraft, выбирая конкретные значения RGB. Этот узел позволяет определить цвет, указывая отдельные значения красного, зелёного и синего, которые затем преобразуются в формат цвета Recraft, пригодный для использования в других операциях Recraft.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `r` | INT | Да | 0-255 | Значение красного канала цвета (по умолчанию: 0) |
| `g` | INT | Да | 0-255 | Значение зелёного канала цвета (по умолчанию: 0) |
| `b` | INT | Да | 0-255 | Значение синего канала цвета (по умолчанию: 0) |
| `recraft_color` | COLOR | Нет | - | Опциональный существующий цвет Recraft для расширения |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `recraft_color` | COLOR | Созданный объект цвета Recraft, содержащий указанные значения RGB |
