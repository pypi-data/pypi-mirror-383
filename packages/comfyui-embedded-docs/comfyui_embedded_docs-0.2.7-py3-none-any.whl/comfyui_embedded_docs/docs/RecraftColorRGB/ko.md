특정 RGB 값을 선택하여 Recraft 색상을 생성합니다. 이 노드를 사용하면 개별적인 빨강, 녹색, 파랑 값을 지정하여 색상을 정의할 수 있으며, 이 값들은 다른 Recraft 작업에서 사용할 수 있는 Recraft 색상 형식으로 변환됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `r` | INT | 예 | 0-255 | 색상의 빨강 값 (기본값: 0) |
| `g` | INT | 예 | 0-255 | 색상의 녹색 값 (기본값: 0) |
| `b` | INT | 예 | 0-255 | 색상의 파랑 값 (기본값: 0) |
| `recraft 색` | COLOR | 아니오 | - | 확장할 선택적 기존 Recraft 색상 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `recraft 색` | COLOR | 지정된 RGB 값을 포함하는 생성된 Recraft 색상 객체 |
