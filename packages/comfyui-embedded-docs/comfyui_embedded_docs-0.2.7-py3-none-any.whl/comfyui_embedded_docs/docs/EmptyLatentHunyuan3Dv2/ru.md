Узел EmptyLatentHunyuan3Dv2 создает пустые латентные тензоры, специально отформатированные для 3D-моделей генерации Hunyuan3Dv2. Он генерирует пустые латентные пространства с правильными размерностями и структурой, требуемыми архитектурой Hunyuan3Dv2, позволяя вам начинать рабочие процессы 3D-генерации с нуля. Узел производит латентные тензоры, заполненные нулями, которые служат основой для последующих процессов 3D-генерации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `разрешение` | INT | Да | 1 - 8192 | Размерность разрешения для латентного пространства (по умолчанию: 3072) |
| `размер_пакета` | INT | Да | 1 - 4096 | Количество латентных изображений в пакете (по умолчанию: 1) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Возвращает латентный тензор, содержащий пустые сэмплы, отформатированные для 3D-генерации Hunyuan3Dv2 |
