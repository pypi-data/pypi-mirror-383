Узел BasicGuider создаёт простой механизм управления процессом сэмплирования. Он принимает модель и данные кондиционирования в качестве входных данных и создаёт объект-направитель, который может использоваться для управления процессом генерации во время сэмплирования. Этот узел предоставляет базовую функциональность управления, необходимую для контролируемой генерации.

## Входные данные

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `модель` | MODEL | обязательный | - | - | Модель, которая будет использоваться для управления |
| `кондиционирование` | CONDITIONING | обязательный | - | - | Данные кондиционирования, которые направляют процесс генерации |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | Объект-направитель, который может использоваться в процессе сэмплирования для управления генерацией |
