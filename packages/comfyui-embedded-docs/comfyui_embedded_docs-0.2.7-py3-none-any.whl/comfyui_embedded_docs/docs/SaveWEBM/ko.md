SaveWEBM 노드는 일련의 이미지들을 WEBM 비디오 파일로 저장합니다. 여러 개의 입력 이미지를 가져와서 VP9 또는 AV1 코덱을 사용하여 구성 가능한 품질 설정과 프레임 속도로 비디오로 인코딩합니다. 결과 비디오 파일은 프롬프트 정보를 포함한 메타데이터와 함께 출력 디렉토리에 저장됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `이미지` | IMAGE | 예 | - | 비디오 프레임으로 인코딩할 입력 이미지 시퀀스 |
| `파일명_접두사` | STRING | 아니오 | - | 출력 파일 이름의 접두사 (기본값: "ComfyUI") |
| `코덱` | COMBO | 예 | "vp9"<br>"av1" | 인코딩에 사용할 비디오 코덱 |
| `fps` | FLOAT | 아니오 | 0.01-1000.0 | 출력 비디오의 프레임 속도 (기본값: 24.0) |
| `crf` | FLOAT | 아니오 | 0-63.0 | 품질 설정으로, 높은 crf는 더 작은 파일 크기와 더 낮은 품질을 의미하고, 낮은 crf는 더 큰 파일 크기와 더 높은 품질을 의미합니다 (기본값: 32.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `ui` | PREVIEW | 저장된 WEBM 파일을 보여주는 비디오 미리보기 |
