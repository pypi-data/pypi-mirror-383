Узел LatentOperationSharpen применяет эффект повышения резкости к латентным представлениям с использованием гауссова ядра. Он работает путем нормализации латентных данных, применения свертки с пользовательским ядром повышения резкости и последующего восстановления исходной яркости. Это усиливает детали и границы в латентном представлении.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `радиус_резкости` | INT | Нет | 1-31 | Радиус ядра повышения резкости (по умолчанию: 9) |
| `сигма` | FLOAT | Нет | 0.1-10.0 | Стандартное отклонение для гауссова ядра (по умолчанию: 1.0) |
| `альфа` | FLOAT | Нет | 0.0-5.0 | Коэффициент интенсивности повышения резкости (по умолчанию: 0.1) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `operation` | LATENT_OPERATION | Возвращает операцию повышения резкости, которая может быть применена к латентным данным |
