LatentOperationSharpen 노드는 가우시안 커널을 사용하여 잠재 표현에 샤프닝 효과를 적용합니다. 이 노드는 잠재 데이터를 정규화하고, 사용자 정의 샤프닝 커널로 컨볼루션을 적용한 후 원래의 휘도를 복원하는 방식으로 작동합니다. 이를 통해 잠재 공간 표현의 세부 사항과 경계를 강화합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `선명화 반경` | INT | 아니요 | 1-31 | 샤프닝 커널의 반지름 (기본값: 9) |
| `시그마` | FLOAT | 아니요 | 0.1-10.0 | 가우시안 커널의 표준 편차 (기본값: 1.0) |
| `알파` | FLOAT | 아니요 | 0.0-5.0 | 샤프닝 강도 계수 (기본값: 0.1) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `operation` | LATENT_OPERATION | 잠재 데이터에 적용할 수 있는 샤프닝 연산을 반환합니다 |
