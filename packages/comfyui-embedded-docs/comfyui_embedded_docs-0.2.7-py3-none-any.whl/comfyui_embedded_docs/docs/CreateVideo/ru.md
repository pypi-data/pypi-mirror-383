Узел Create Video создает видеофайл из последовательности изображений. Вы можете указать скорость воспроизведения с помощью кадров в секунду и при необходимости добавить аудио к видео. Узел объединяет ваши изображения в видеоформат, который можно воспроизводить с указанной частотой кадров.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображения` | IMAGE | Да | - | Изображения, из которых создается видео. |
| `кадров в секунду` | FLOAT | Да | 1.0 - 120.0 | Кадры в секунду для скорости воспроизведения видео (по умолчанию: 30.0). |
| `аудио` | AUDIO | Нет | - | Аудиодорожка для добавления к видео. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл, содержащий входные изображения и опциональное аудио. |
