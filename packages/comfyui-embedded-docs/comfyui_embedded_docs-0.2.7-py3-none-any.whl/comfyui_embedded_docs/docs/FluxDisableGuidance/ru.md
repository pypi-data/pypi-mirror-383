Этот узел полностью отключает функциональность направляющих эмбеддингов для моделей Flux и аналогичных. Он принимает на вход данные кондиционирования и удаляет компонент направляющих, устанавливая его в значение None, что эффективно отключает кондиционирование на основе направляющих в процессе генерации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `conditioning` | CONDITIONING | Да | - | Данные кондиционирования, из которых удаляются направляющие |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | Модифицированные данные кондиционирования с отключенными направляющими |
