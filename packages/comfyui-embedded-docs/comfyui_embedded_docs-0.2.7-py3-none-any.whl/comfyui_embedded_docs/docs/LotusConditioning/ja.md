LotusConditioningノードは、Lotusモデル用に事前計算された条件付け埋め込みを提供します。このノードは、ヌル条件付けを使用した凍結されたエンコーダを利用し、推論や大きなテンソルファイルの読み込みを必要とせずに、リファレンス実装との互換性を達成するためにハードコードされたプロンプト埋め込みを返します。このノードは、生成パイプラインで直接使用できる固定の条件付けテンソルを出力します。
