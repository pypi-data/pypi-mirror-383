Упрощенная версия узла SkipLayerGuidanceDiT, которая изменяет только безусловный проход в процессе денойзинга. Этот узел применяет skip layer guidance к определенным слоям трансформера в моделях DiT (Diffusion Transformer), выборочно пропуская определенные слои во время безусловного прохода на основе заданных параметров времени и слоев.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой применяется skip layer guidance |
| `double_layers` | STRING | Да | - | Индексы слоев с двойными блоками для пропуска, разделенные запятыми (по умолчанию: "7, 8, 9") |
| `single_layers` | STRING | Да | - | Индексы слоев с одинарными блоками для пропуска, разделенные запятыми (по умолчанию: "7, 8, 9") |
| `start_percent` | FLOAT | Да | 0.0 - 1.0 | Начальный процент процесса денойзинга, когда начинается skip layer guidance (по умолчанию: 0.0) |
| `end_percent` | FLOAT | Да | 0.0 - 1.0 | Конечный процент процесса денойзинга, когда прекращается skip layer guidance (по умолчанию: 1.0) |

**Примечание:** Skip layer guidance применяется только когда и `double_layers`, и `single_layers` содержат допустимые индексы слоев. Если оба параметра пусты, узел возвращает исходную модель без изменений.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с примененным skip layer guidance к указанным слоям |
