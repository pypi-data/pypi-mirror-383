Узел ByteDance Image Reference создает видео с использованием текстового запроса и от одной до четырех эталонных изображений. Он отправляет изображения и запрос во внешний API-сервис, который создает видео, соответствующее вашему описанию, и при этом включает визуальный стиль и содержимое ваших эталонных изображений. Узел предоставляет различные элементы управления для разрешения видео, соотношения сторон, длительности и других параметров генерации.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `model` | MODEL | COMBO | seedance_1_lite | seedance_1_lite | Название модели |
| `prompt` | STRING | STRING | - | - | Текстовый запрос, используемый для генерации видео. |
| `images` | IMAGE | IMAGE | - | - | От одного до четырех изображений. |
| `resolution` | STRING | COMBO | - | 480p, 720p | Разрешение выходного видео. |
| `aspect_ratio` | STRING | COMBO | - | adaptive, 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 | Соотношение сторон выходного видео. |
| `duration` | INT | INT | 5 | 3-12 | Длительность выходного видео в секундах. |
| `seed` | INT | INT | 0 | 0-2147483647 | Начальное значение (сид) для использования при генерации. |
| `watermark` | BOOLEAN | BOOLEAN | True | - | Определяет, добавлять ли водяной знак "Сгенерировано ИИ" на видео. |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл на основе входного запроса и эталонных изображений. |
