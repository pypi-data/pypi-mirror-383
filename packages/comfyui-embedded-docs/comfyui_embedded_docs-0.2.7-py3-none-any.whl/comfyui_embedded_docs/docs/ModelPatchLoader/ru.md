## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `name` | STRING | Да | Все доступные файлы патчей моделей из папки model_patches | Имя файла патча модели для загрузки из директории model_patches |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `MODEL_PATCH` | MODEL_PATCH | Загруженный патч модели, обёрнутый в ModelPatcher для использования в рабочем процессе |
