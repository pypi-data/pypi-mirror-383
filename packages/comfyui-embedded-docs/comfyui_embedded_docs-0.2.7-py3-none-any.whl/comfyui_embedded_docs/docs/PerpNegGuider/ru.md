Узел PerpNegGuider создает систему управления для контроля генерации изображений с использованием перпендикулярного негативного кондиционирования. Он принимает входные данные позитивного, негативного и пустого кондиционирования и применяет специализированный алгоритм управления для направления процесса генерации. Этот узел предназначен для тестирования и обеспечивает точный контроль над силой управления и масштабированием негативного воздействия.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, используемая для генерации управления |
| `положительный` | CONDITIONING | Да | - | Позитивное кондиционирование, которое направляет генерацию к желаемому содержимому |
| `отрицательный` | CONDITIONING | Да | - | Негативное кондиционирование, которое отдаляет генерацию от нежелательного содержимого |
| `пустое_условие` | CONDITIONING | Да | - | Пустое или нейтральное кондиционирование, используемое в качестве базового ориентира |
| `cfg` | FLOAT | Нет | 0.0 - 100.0 | Коэффициент управления без классификатора, который контролирует, насколько сильно кондиционирование влияет на генерацию (по умолчанию: 8.0) |
| `масштаб_отриц` | FLOAT | Нет | 0.0 - 100.0 | Коэффициент масштабирования негативного воздействия, который регулирует силу негативного кондиционирования (по умолчанию: 1.0) |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `guider` | GUIDER | Настроенная система управления, готовая к использованию в конвейере генерации |
