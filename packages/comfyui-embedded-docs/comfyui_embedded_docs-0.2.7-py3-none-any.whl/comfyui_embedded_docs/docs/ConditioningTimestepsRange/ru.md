Узел ConditioningTimestepsRange создаёт три отдельных диапазона временных шагов для управления моментом применения эффектов кондиционирования в процессе генерации. Он принимает начальное и конечное процентные значения и разделяет весь диапазон временных шагов (от 0.0 до 1.0) на три сегмента: основной диапазон между указанными процентами, диапазон до начального процента и диапазон после конечного процента.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `процент_начала` | FLOAT | Да | 0.0 - 1.0 | Начальный процент диапазона временных шагов (по умолчанию: 0.0) |
| `процент_конца` | FLOAT | Да | 0.0 - 1.0 | Конечный процент диапазона временных шагов (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `ДО_ДИАПАЗОНА` | TIMESTEPS_RANGE | Основной диапазон временных шагов, определённый start_percent и end_percent |
| `ПОСЛЕ_ДИАПАЗОНА` | TIMESTEPS_RANGE | Диапазон временных шагов от 0.0 до start_percent |
| `AFTER_RANGE` | TIMESTEPS_RANGE | Диапазон временных шагов от end_percent до 1.0 |
