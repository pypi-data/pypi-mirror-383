Узел SaveAudioOpus сохраняет аудиоданные в файл формата Opus. Он принимает аудиовход и экспортирует его как сжатый Opus-файл с настраиваемыми параметрами качества. Узел автоматически обрабатывает присвоение имён файлам и сохраняет результат в указанный выходной каталог.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | Да | - | Аудиоданные для сохранения в файл формата Opus |
| `filename_prefix` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "audio/ComfyUI") |
| `quality` | COMBO | Нет | "64k"<br>"96k"<br>"128k"<br>"192k"<br>"320k" | Настройка качества аудио для Opus-файла (по умолчанию: "128k") |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| - | - | Этот узел не возвращает никаких выходных значений. Его основной функцией является сохранение аудиофайла на диск. |
