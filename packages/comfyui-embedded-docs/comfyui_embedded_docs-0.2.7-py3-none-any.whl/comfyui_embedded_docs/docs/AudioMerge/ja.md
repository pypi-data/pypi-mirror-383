AudioMergeノードは、2つのオーディオトラックを波形を重ね合わせて結合します。両方のオーディオ入力のサンプルレートを自動的に一致させ、長さを等しく調整してからマージを行います。このノードは、オーディオ信号を結合するためのいくつかの数学的手法を提供し、出力が許容可能な音量レベル内に収まるようにします。

## 入力

| パラメータ | データ型 | 入力タイプ | デフォルト | 範囲 | 説明 |
|-----------|-----------|------------|---------|-------|-------------|
| `audio1` | AUDIO | required | - | - | マージする最初のオーディオ入力 |
| `audio2` | AUDIO | required | - | - | マージする2番目のオーディオ入力 |
| `merge_method` | COMBO | required | - | ["add", "mean", "subtract", "multiply"] | オーディオ波形を結合するために使用される方法 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | 結合された波形とサンプルレートを含むマージ済みオーディオ出力 |
