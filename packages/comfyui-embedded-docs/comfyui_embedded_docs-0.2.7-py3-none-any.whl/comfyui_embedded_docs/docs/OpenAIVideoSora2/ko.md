OpenAIVideoSora2 노드는 OpenAI의 Sora 모델을 사용하여 비디오를 생성합니다. 텍스트 프롬프트와 선택적 입력 이미지를 기반으로 비디오 콘텐츠를 생성한 후 생성된 비디오 출력을 반환합니다. 이 노드는 선택된 모델에 따라 다양한 비디오 길이와 해상도를 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | "sora-2"<br>"sora-2-pro" | 비디오 생성에 사용할 OpenAI Sora 모델 (기본값: "sora-2") |
| `prompt` | STRING | 예 | - | 안내 텍스트; 입력 이미지가 있는 경우 비워둘 수 있음 (기본값: 빈 값) |
| `size` | COMBO | 예 | "720x1280"<br>"1280x720"<br>"1024x1792"<br>"1792x1024" | 생성된 비디오의 해상도 (기본값: "1280x720") |
| `duration` | COMBO | 예 | 4<br>8<br>12 | 생성된 비디오의 길이(초 단위) (기본값: 8) |
| `image` | IMAGE | 아니오 | - | 비디오 생성을 위한 선택적 입력 이미지 |
| `seed` | INT | 아니오 | 0부터 2147483647까지 | 노드 재실행 여부를 결정하는 시드; 실제 결과는 시드 값과 관계없이 비결정적임 (기본값: 0) |

**제약 사항 및 한계:**

- "sora-2" 모델은 "720x1280" 및 "1280x720" 해상도만 지원합니다
- image 매개변수 사용 시 하나의 입력 이미지만 지원됩니다
- 시드 값과 관계없이 결과는 비결정적입니다

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 출력 |
