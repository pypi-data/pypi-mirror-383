Узел T5TokenizerOptions позволяет настроить параметры токенизатора для различных типов моделей T5. Он устанавливает параметры минимального заполнения и минимальной длины для нескольких вариантов моделей T5, включая t5xxl, pile_t5xl, t5base, mt5xl и umt5xxl. Узел принимает входные данные CLIP и возвращает модифицированную модель CLIP с применёнными указанными настройками токенизатора.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | - | Модель CLIP, для которой настраиваются параметры токенизатора |
| `минимальное_дополнение` | INT | Нет | 0-10000 | Значение минимального заполнения, устанавливаемое для всех типов моделей T5 (по умолчанию: 0) |
| `минимальная_длина` | INT | Нет | 0-10000 | Значение минимальной длины, устанавливаемое для всех типов моделей T5 (по умолчанию: 0) |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | CLIP | Модифицированная модель CLIP с обновлёнными параметрами токенизатора, применёнными ко всем вариантам T5 |
