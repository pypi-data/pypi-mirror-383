Этот узел устанавливает направляющий латент для модели редактирования. Он принимает данные кондиционирования и опциональный латентный вход, затем модифицирует кондиционирование, чтобы включить информацию о референсном латенте. Если модель поддерживает эту функцию, вы можете объединять несколько узлов ReferenceLatent для установки нескольких референсных изображений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `conditioning` | CONDITIONING | Да | - | Данные кондиционирования, которые будут модифицированы с информацией о референсном латенте |
| `latent` | LATENT | Нет | - | Опциональные латентные данные, используемые в качестве референса для модели редактирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | CONDITIONING | Модифицированные данные кондиционирования, содержащие информацию о референсном латенте |
