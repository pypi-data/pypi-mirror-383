Узел CLIPTextEncodeControlnet обрабатывает текстовый ввод с помощью модели CLIP и объединяет его с существующими данными кондиционирования для создания улучшенного выходного кондиционирования для приложений controlnet. Он токенизирует входной текст, кодирует его через модель CLIP и добавляет полученные эмбеддинги к предоставленным данным кондиционирования в качестве параметров cross-attention controlnet.

## Входы

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `clip` | CLIP | Обязательный | - | - | Модель CLIP, используемая для токенизации и кодирования текста |
| `кондиционирование` | CONDITIONING | Обязательный | - | - | Существующие данные кондиционирования, которые будут улучшены параметрами controlnet |
| `текст` | STRING | Многострочный, Динамические промпты | - | - | Текстовый ввод для обработки моделью CLIP |

**Примечание:** Для работы этого узла требуются оба входных параметра `clip` и `conditioning`. Вход `text` поддерживает динамические промпты и многострочный текст для гибкой обработки текста.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Улучшенные данные кондиционирования с добавленными параметрами cross-attention controlnet |
