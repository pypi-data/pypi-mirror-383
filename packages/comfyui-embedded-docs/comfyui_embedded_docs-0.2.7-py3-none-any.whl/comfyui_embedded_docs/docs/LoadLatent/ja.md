LoadLatentノードは、入力ディレクトリ内の.latentファイルから以前に保存された潜在表現を読み込みます。ファイルから潜在テンソルデータを読み取り、他のノードで使用できるようにする前に必要なスケーリング調整を適用してから、潜在データを返します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `latent` | STRING | はい | 入力ディレクトリ内のすべての.latentファイル | 入力ディレクトリ内の利用可能なファイルから、どの.latentファイルを読み込むかを選択します |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | 選択したファイルから読み込まれた潜在表現データを返します |
