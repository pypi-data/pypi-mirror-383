Узел LoadLatent загружает ранее сохранённые латентные представления из файлов .latent во входной директории. Он считывает данные латентного тензора из файла и применяет необходимые корректировки масштабирования перед возвратом латентных данных для использования в других узлах.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `скрытый` | STRING | Да | Все .latent файлы во входной директории | Выбирает, какой .latent файл загрузить из доступных файлов во входной директории |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Возвращает загруженные данные латентного представления из выбранного файла |
