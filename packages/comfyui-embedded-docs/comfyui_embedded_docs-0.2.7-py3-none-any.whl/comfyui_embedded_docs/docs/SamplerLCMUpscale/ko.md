SamplerLCMUpscale 노드는 잠재 일관성 모델(LCM) 샘플링과 이미지 업스케일링 기능을 결합한 특수한 샘플링 방법을 제공합니다. 다양한 보간 방법을 사용하여 샘플링 과정에서 이미지를 업스케일할 수 있어, 이미지 품질을 유지하면서 더 높은 해상도의 출력물을 생성하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `확대율` | FLOAT | 아니오 | 0.1 - 20.0 | 업스케일링 과정에서 적용할 스케일링 비율 (기본값: 1.0) |
| `확대 스텝 수` | INT | 아니오 | -1 - 1000 | 업스케일링 과정에 사용할 단계 수. 자동 계산을 위해 -1을 사용합니다 (기본값: -1) |
| `업스케일 방법` | COMBO | 예 | "bislerp"<br>"nearest-exact"<br>"bilinear"<br>"area"<br>"bicubic" | 이미지 업스케일링에 사용되는 보간 방법 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | 샘플링 파이프라인에서 사용할 수 있도록 구성된 샘플러 객체를 반환합니다 |
