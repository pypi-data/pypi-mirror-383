SamplerLCMUpscaleノードは、Latent Consistency Model（LCM）サンプリングと画像アップスケーリング機能を組み合わせた特殊なサンプリング方法を提供します。様々な補間方法を使用してサンプリングプロセス中に画像をアップスケールすることができ、画質を維持しながら高解像度の出力を生成するのに役立ちます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `スケール比` | FLOAT | いいえ | 0.1 - 20.0 | アップスケーリング中に適用するスケーリング係数（デフォルト: 1.0） |
| `スケールステップ` | INT | いいえ | -1 - 1000 | アップスケーリングプロセスに使用するステップ数。-1を指定すると自動計算されます（デフォルト: -1） |
| `アップスケール方法` | COMBO | はい | "bislerp"<br>"nearest-exact"<br>"bilinear"<br>"area"<br>"bicubic" | 画像のアップスケーリングに使用する補間方法 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | サンプリングパイプラインで使用できる設定済みサンプラーオブジェクトを返します |
