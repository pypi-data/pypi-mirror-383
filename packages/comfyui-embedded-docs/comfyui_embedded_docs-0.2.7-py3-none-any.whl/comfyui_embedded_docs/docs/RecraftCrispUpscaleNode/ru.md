Масштабирует изображение синхронно. Улучшает заданное растровое изображение с помощью инструмента 'crisp upscale', увеличивая разрешение изображения, делая его более чётким и чистым.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для масштабирования |
| `auth_token` | STRING | Нет | - | Токен аутентификации для Recraft API |
| `comfy_api_key` | STRING | Нет | - | API-ключ для сервисов Comfy.org |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Масштабированное изображение с улучшенным разрешением и чёткостью |
