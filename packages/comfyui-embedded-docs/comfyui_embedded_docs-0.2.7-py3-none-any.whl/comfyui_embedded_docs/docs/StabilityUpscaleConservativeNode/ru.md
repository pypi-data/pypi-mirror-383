Масштабирование изображения с минимальными изменениями до разрешения 4K. Этот узел использует консервативное масштабирование от Stability AI для увеличения разрешения изображения с сохранением исходного содержимого и внесением лишь незначительных изменений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Исходное изображение для масштабирования |
| `промпт` | STRING | Да | - | То, что вы хотите видеть в итоговом изображении. Четкий, описательный промт, который ясно определяет элементы, цвета и объекты, позволит достичь лучших результатов. (по умолчанию: пустая строка) |
| `креативность` | FLOAT | Да | 0.2-0.5 | Определяет вероятность создания дополнительных деталей, не строго обусловленных исходным изображением. (по умолчанию: 0.35) |
| `seed` | INT | Да | 0-4294967294 | Случайное начальное значение (сид), используемое для генерации шума. (по умолчанию: 0) |
| `негативный промпт` | STRING | Нет | - | Ключевые слова, описывающие то, что вы не хотите видеть в итоговом изображении. Это расширенная функция. (по умолчанию: пустая строка) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Масштабированное изображение в разрешении 4K |
