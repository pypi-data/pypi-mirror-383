Этот узел содержит изображение и значение веса для использования с узлом Luma Generate Image. Он создаёт цепочку ссылок, которая может быть передана другим узлам Luma для влияния на генерацию изображения. Узел может либо начать новую цепочку ссылок, либо добавить к существующей.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Изображение, используемое в качестве ссылки. |
| `weight` | FLOAT | Да | 0.0 - 1.0 | Вес ссылки на изображение (по умолчанию: 1.0). |
| `luma_ref` | LUMA_REF | Нет | - | Необязательная существующая цепочка ссылок Luma, к которой можно добавить. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `luma_ref` | LUMA_REF | Цепочка ссылок Luma, содержащая изображение и вес. |
