BetaSamplingSchedulerノードは、ベータスケジューリングアルゴリズムを使用して、サンプリングプロセス用のノイズレベル（シグマ）のシーケンスを生成します。モデルと設定パラメータを受け取り、画像生成時のノイズ除去プロセスを制御するカスタマイズされたノイズスケジュールを作成します。このスケジューラは、アルファとベータのパラメータを通じてノイズ低減の軌跡を微調整することができます。

## 入力

| パラメータ | データ型 | 入力タイプ | デフォルト | 範囲 | 説明 |
|-----------|-----------|------------|---------|-------|-------------|
| `モデル` | MODEL | 必須 | - | - | サンプリングに使用されるモデル。モデルサンプリングオブジェクトを提供します |
| `ステップ` | INT | 必須 | 20 | 1-10000 | シグマを生成するサンプリングステップ数 |
| `アルファ` | FLOAT | 必須 | 0.6 | 0.0-50.0 | ベータスケジューラのアルファパラメータ。スケジューリング曲線を制御します |
| `ベータ` | FLOAT | 必須 | 0.6 | 0.0-50.0 | ベータスケジューラのベータパラメータ。スケジューリング曲線を制御します |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `SIGMAS` | SIGMAS | サンプリングプロセスで使用されるノイズレベル（シグマ）のシーケンス |
