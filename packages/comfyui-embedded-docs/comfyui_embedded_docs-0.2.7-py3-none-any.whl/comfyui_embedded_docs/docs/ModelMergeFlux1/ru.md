Узел ModelMergeFlux1 объединяет две диффузионные модели путем смешивания их компонентов с использованием взвешенной интерполяции. Он обеспечивает детальный контроль над тем, как различные части моделей комбинируются, включая блоки обработки изображений, слои временного эмбеддинга, механизмы управления, векторные входы, текстовые энкодеры и различные трансформер-блоки. Это позволяет создавать гибридные модели с настроенными характеристиками из двух исходных моделей.
