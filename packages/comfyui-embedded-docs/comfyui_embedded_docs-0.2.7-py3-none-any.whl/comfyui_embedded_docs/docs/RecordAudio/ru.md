Узел RecordAudio загружает аудиофайлы, которые были записаны или выбраны через интерфейс аудиозаписи. Он обрабатывает аудиофайл и преобразует его в формат волновой формы, который может использоваться другими узлами обработки аудио в рабочем процессе. Узел автоматически определяет частоту дискретизации и подготавливает аудиоданные для дальнейшей обработки.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO_RECORD | Да | N/A | Вход аудиозаписи из интерфейса аудиозаписи |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | Обработанные аудиоданные, содержащие информацию о волновой форме и частоте дискретизации |
