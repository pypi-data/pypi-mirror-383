Узел VAEDecode предназначен для декодирования латентных представлений в изображения с использованием заданного вариационного автоэнкодера (VAE). Он служит для генерации изображений из сжатых представлений данных, обеспечивая реконструкцию изображений из их латентных пространственных кодировок.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `образцы` | `LATENT`    | Параметр 'samples' представляет латентные представления, которые должны быть декодированы в изображения. Он критически важен для процесса декодирования, поскольку предоставляет сжатые данные, из которых реконструируются изображения. |
| `vae`     | VAE       | Параметр 'vae' определяет модель вариационного автоэнкодера, которая будет использоваться для декодирования латентных представлений в изображения. Он необходим для определения механизма декодирования и качества реконструированных изображений. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | Выходное изображение, реконструированное из предоставленного латентного представления с использованием указанной модели VAE. |
