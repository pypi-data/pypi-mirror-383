
이 노드는 합성 작업을 위해 설계되었으며, 특히 이미지와 해당 알파 마스크를 결합하여 단일 출력 이미지를 생성합니다. 이는 시각적 콘텐츠와 투명성 정보를 효과적으로 결합하여 특정 영역이 투명하거나 반투명한 이미지를 생성할 수 있게 합니다.

## 입력

| 매개변수 | 데이터 유형 | 설명                                                                                                                           |
| -------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `이미지`  | `IMAGE`     | 알파 마스크와 결합될 주요 시각 콘텐츠입니다. 투명성 정보가 없는 이미지를 나타냅니다.                                           |
| `알파`  | `MASK`      | 해당 이미지의 투명성을 정의하는 알파 마스크입니다. 이미지의 어느 부분이 투명하거나 반투명해야 하는지를 결정하는 데 사용됩니다. |

## 출력

| 매개변수 | 데이터 유형 | 설명                                                                                                |
| -------- | ----------- | --------------------------------------------------------------------------------------------------- |
| `이미지`  | `IMAGE`     | 입력 이미지와 알파 마스크를 결합하여 투명성 정보를 시각적 콘텐츠에 통합한 단일 이미지를 출력합니다. |
