Узел PrimitiveFloat создает числовое значение с плавающей точкой, которое может использоваться в вашем рабочем процессе. Он принимает единственное числовое значение на вход и выдает это же значение на выход, позволяя вам определять и передавать значения с плавающей точкой между различными узлами в вашем конвейере ComfyUI.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `значение` | FLOAT | Да | -sys.maxsize до sys.maxsize | Числовое значение с плавающей точкой для вывода |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | FLOAT | Входное числовое значение с плавающей точкой |
