StringContains 노드는 주어진 문자열에 특정 부분 문자열이 포함되어 있는지 확인합니다. 대소문자를 구분하거나 구분하지 않는 검색을 수행할 수 있으며, 부분 문자열이 메인 문자열 내에서 발견되었는지 여부를 나타내는 불리언 결과를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | - | 검색을 수행할 메인 텍스트 문자열 |
| `substring` | STRING | 예 | - | 메인 문자열 내에서 검색할 텍스트 |
| `case_sensitive` | BOOLEAN | 아니오 | - | 검색 시 대소문자를 구분할지 여부를 결정합니다 (기본값: true) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `contains` | BOOLEAN | 부분 문자열이 문자열 내에서 발견되면 true를 반환하고, 그렇지 않으면 false를 반환합니다 |
