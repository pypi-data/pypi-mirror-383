FreeU_V2 노드는 U-Net 아키텍처를 수정하여 확산 모델에 주파수 기반 향상을 적용합니다. 추가 학습 없이 이미지 생성 품질을 개선하기 위해 구성 가능한 매개변수를 사용하여 서로 다른 특징 채널의 크기를 조정합니다. 이 노드는 특정 채널 차원에 스케일링 인자를 적용하기 위해 모델의 출력 블록을 패치하는 방식으로 작동합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | FreeU 향상을 적용할 확산 모델 |
| `b1` | FLOAT | 예 | 0.0 - 10.0 | 첫 번째 블록에 대한 백본 특징 스케일링 인자 (기본값: 1.3) |
| `b2` | FLOAT | 예 | 0.0 - 10.0 | 두 번째 블록에 대한 백본 특징 스케일링 인자 (기본값: 1.4) |
| `s1` | FLOAT | 예 | 0.0 - 10.0 | 첫 번째 블록에 대한 스킵 특징 스케일링 인자 (기본값: 0.9) |
| `s2` | FLOAT | 예 | 0.0 - 10.0 | 두 번째 블록에 대한 스킵 특징 스케일링 인자 (기본값: 0.2) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | FreeU 수정이 적용된 향상된 확산 모델 |
