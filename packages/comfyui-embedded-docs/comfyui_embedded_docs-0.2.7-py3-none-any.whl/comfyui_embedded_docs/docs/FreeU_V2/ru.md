Узел FreeU_V2 применяет частотное улучшение для диффузионных моделей путем модификации архитектуры U-Net. Он масштабирует различные каналы характеристик с использованием настраиваемых параметров для повышения качества генерации изображений без необходимости дополнительного обучения. Узел работает путем модификации выходных блоков модели для применения коэффициентов масштабирования к определенным размерностям каналов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Диффузионная модель, к которой применяется улучшение FreeU |
| `b1` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования характеристик основного потока для первого блока (по умолчанию: 1.3) |
| `b2` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования характеристик основного потока для второго блока (по умолчанию: 1.4) |
| `s1` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования характеристик обходного пути для первого блока (по умолчанию: 0.9) |
| `s2` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования характеристик обходного пути для второго блока (по умолчанию: 0.2) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Улучшенная диффузионная модель с примененными модификациями FreeU |
