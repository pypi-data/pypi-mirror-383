SaveGLBノードは、3DメッシュデータをGLBファイルとして保存するノードです。GLBは3Dモデルの一般的なフォーマットです。このノードはメッシュデータを入力として受け取り、指定されたファイル名プレフィックスを使用して出力ディレクトリにエクスポートします。入力に複数のメッシュオブジェクトが含まれている場合は複数のメッシュを保存でき、メタデータが有効な場合はファイルに自動的にメタデータを追加します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `メッシュ` | MESH | はい | - | GLBファイルとして保存する3Dメッシュデータ |
| `ファイル名のプレフィックス` | STRING | いいえ | - | 出力ファイル名のプレフィックス（デフォルト: "mesh/ComfyUI"） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `ui` | UI | 保存されたGLBファイルをファイル名とサブフォルダ情報と共にユーザーインターフェースに表示します |
