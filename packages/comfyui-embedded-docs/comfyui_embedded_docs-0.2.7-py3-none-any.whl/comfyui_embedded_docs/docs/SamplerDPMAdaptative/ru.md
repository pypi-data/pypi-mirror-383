Узел SamplerDPMAdaptative реализует адаптивный сэмплер DPM (Diffusion Probabilistic Model), который автоматически регулирует размеры шагов в процессе сэмплирования. Он использует контроль ошибок на основе допусков для определения оптимальных размеров шагов, балансируя между вычислительной эффективностью и точностью сэмплирования. Этот адаптивный подход помогает сохранять качество, потенциально уменьшая количество необходимых шагов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `порядок` | INT | Да | 2-3 | Порядок метода сэмплера (по умолчанию: 3) |
| `rtol` | FLOAT | Да | 0.0-100.0 | Относительный допуск для контроля ошибок (по умолчанию: 0.05) |
| `atol` | FLOAT | Да | 0.0-100.0 | Абсолютный допуск для контроля ошибок (по умолчанию: 0.0078) |
| `h_init` | FLOAT | Да | 0.0-100.0 | Начальный размер шага (по умолчанию: 0.05) |
| `pcoeff` | FLOAT | Да | 0.0-100.0 | Пропорциональный коэффициент для контроля размера шага (по умолчанию: 0.0) |
| `icoeff` | FLOAT | Да | 0.0-100.0 | Интегральный коэффициент для контроля размера шага (по умолчанию: 1.0) |
| `dcoeff` | FLOAT | Да | 0.0-100.0 | Дифференциальный коэффициент для контроля размера шага (по умолчанию: 0.0) |
| `accept_safety` | FLOAT | Да | 0.0-100.0 | Коэффициент безопасности для принятия шага (по умолчанию: 0.81) |
| `eta` | FLOAT | Да | 0.0-100.0 | Параметр стохастичности (по умолчанию: 0.0) |
| `s_noise` | FLOAT | Да | 0.0-100.0 | Коэффициент масштабирования шума (по умолчанию: 1.0) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Возвращает настроенный экземпляр адаптивного сэмплера DPM |
