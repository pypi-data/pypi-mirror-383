SetClipHooks 노드를 사용하면 CLIP 모델에 사용자 정의 훅을 적용하여 고급 동작 수정이 가능합니다. 이 노드는 조건화 출력에 훅을 적용하고, 선택적으로 클립 스케줄링 기능을 활성화할 수 있습니다. 입력된 CLIP 모델의 복제본을 생성하여 지정된 훅 설정을 적용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | - | 훅을 적용할 CLIP 모델 |
| `조건에 적용` | BOOLEAN | 예 | - | 조건화 출력에 훅을 적용할지 여부 (기본값: True) |
| `clip 스케쥴 사용` | BOOLEAN | 예 | - | 클립 스케줄링을 활성화할지 여부 (기본값: False) |
| `후크` | HOOKS | 아니오 | - | CLIP 모델에 적용할 선택적 훅 그룹 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `clip` | CLIP | 지정된 훅이 적용된 복제된 CLIP 모델 |
