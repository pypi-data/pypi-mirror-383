TrimVideoLatentノードは、ビデオの潜在表現から先頭のフレームを削除します。このノードは潜在ビデオサンプルを受け取り、指定された数のフレームを先頭から切り取り、残りの部分を返します。これにより、初期フレームを削除してビデオシーケンスを短縮することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `サンプル` | LATENT | はい | - | トリム対象のビデオフレームを含む入力潜在ビデオ表現 |
| `トリム量` | INT | いいえ | 0 から 99999 | ビデオの先頭から削除するフレーム数（デフォルト: 0） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `output` | LATENT | 指定された数のフレームが先頭から削除された、トリム済みの潜在ビデオ表現 |
