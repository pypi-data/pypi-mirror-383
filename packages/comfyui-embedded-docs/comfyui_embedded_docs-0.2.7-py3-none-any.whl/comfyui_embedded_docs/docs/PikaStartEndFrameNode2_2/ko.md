PikaFrames v2.2 노드는 첫 번째 프레임과 마지막 프레임을 결합하여 비디오를 생성합니다. 시작점과 끝점을 정의하는 두 개의 이미지를 업로드하면, AI가 두 이미지 사이를 부드럽게 전환하는 완성된 비디오를 만들어냅니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `시작 이미지` | IMAGE | 예 | - | 결합할 첫 번째 이미지입니다. |
| `끝 이미지` | IMAGE | 예 | - | 결합할 마지막 이미지입니다. |
| `프롬프트 텍스트` | STRING | 예 | - | 원하는 비디오 내용을 설명하는 텍스트 프롬프트입니다. |
| `부정 프롬프트` | STRING | 예 | - | 비디오에서 피해야 할 내용을 설명하는 텍스트입니다. |
| `시드` | INT | 예 | - | 생성 일관성을 위한 랜덤 시드 값입니다. |
| `해상도` | STRING | 예 | - | 출력 비디오 해상도입니다. |
| `길이` | INT | 예 | - | 생성된 비디오의 지속 시간입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 시작 및 종료 프레임을 AI 전환 효과와 함께 결합하여 생성된 비디오입니다. |
