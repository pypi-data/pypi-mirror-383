Узел ImageScaleToMaxDimension изменяет размер изображений, чтобы они помещались в пределах заданного максимального измерения, сохраняя при этом исходное соотношение сторон. Он определяет, является ли изображение портретным или альбомным, затем масштабирует большее измерение до целевого размера, пропорционально изменяя меньшее измерение. Узел поддерживает несколько методов увеличения масштаба для различных требований к качеству и производительности.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение для масштабирования |
| `upscale_method` | STRING | Да | "area"<br>"lanczos"<br>"bilinear"<br>"nearest-exact"<br>"bicubic" | Метод интерполяции, используемый для масштабирования изображения |
| `largest_size` | INT | Да | 0 до 16384 | Максимальное измерение для масштабированного изображения (по умолчанию: 512) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Масштабированное изображение, у которого наибольшее измерение соответствует указанному размеру |
