Имя узла: Reroute Node  
Назначение узла: В основном используется для организации логики чрезмерно длинных линий соединений в рабочем процессе ComfyUI.

## Как использовать узлы Reroute

| Опции меню | Описание |
| --- | --- |
| Show Type | Отобразить свойство типа узла |
| Hide Type By Default | Скрывать свойство типа узла по умолчанию |
| Set Vertical | Установить вертикальное направление проводки |
| Set Horizontal | Установить горизонтальное направление проводки |

Когда логика ваших соединений становится слишком длинной и сложной, и вы хотите упорядочить интерфейс, вы можете вставить узел `Reroute` между двумя точками соединения. Вход и выход этого узла не имеют ограничений по типу, а стиль по умолчанию — горизонтальный. Вы можете изменить направление проводки на вертикальное через контекстное меню (правой кнопкой мыши).
