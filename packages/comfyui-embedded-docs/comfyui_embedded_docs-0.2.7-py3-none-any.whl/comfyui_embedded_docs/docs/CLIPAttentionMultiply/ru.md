Узел CLIPAttentionMultiply позволяет настраивать механизм внимания в моделях CLIP путем применения множителей к различным компонентам слоев самовнимания. Он работает за счет модификации весов и смещений проекций запроса, ключа, значения и выходной проекции в механизме внимания модели CLIP. Этот экспериментальный узел создает модифицированную копию входной модели CLIP с применением указанных коэффициентов масштабирования.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `clip` | CLIP | обязательный | - | - | Модель CLIP для модификации |
| `q` | FLOAT | обязательный | 1.0 | 0.0 - 10.0 | Множитель для весов и смещений проекции запроса |
| `k` | FLOAT | обязательный | 1.0 | 0.0 - 10.0 | Множитель для весов и смещений проекции ключа |
| `v` | FLOAT | обязательный | 1.0 | 0.0 - 10.0 | Множитель для весов и смещений проекции значения |
| `выход` | FLOAT | обязательный | 1.0 | 0.0 - 10.0 | Множитель для весов и смещений выходной проекции |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `CLIP` | CLIP | Возвращает модифицированную модель CLIP с применением указанных коэффициентов масштабирования внимания |
