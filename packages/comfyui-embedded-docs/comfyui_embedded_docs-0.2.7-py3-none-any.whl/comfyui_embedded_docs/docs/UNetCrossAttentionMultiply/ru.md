Узел UNetCrossAttentionMultiply применяет коэффициенты умножения к механизму перекрёстного внимания в модели UNet. Он позволяет масштабировать компоненты запроса, ключа, значения и выхода в слоях перекрёстного внимания для экспериментирования с различными вариантами поведения и эффектами внимания.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель UNet для модификации с помощью коэффициентов масштабирования внимания |
| `q` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов запроса в перекрёстном внимании (по умолчанию: 1.0) |
| `k` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов ключа в перекрёстном внимании (по умолчанию: 1.0) |
| `v` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов значения в перекрёстном внимании (по умолчанию: 1.0) |
| `выход` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов выхода в перекрёстном внимании (по умолчанию: 1.0) |

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель UNet с масштабированными компонентами перекрёстного внимания |
