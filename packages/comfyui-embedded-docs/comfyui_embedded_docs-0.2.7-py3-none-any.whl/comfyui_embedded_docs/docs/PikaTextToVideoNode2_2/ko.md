Pika Text2Video v2.2 노드는 텍스트 프롬프트를 Pika API 버전 2.2로 전송하여 비디오를 생성합니다. 이 노드는 사용자의 텍스트 설명을 Pika의 AI 비디오 생성 서비스를 이용해 비디오로 변환합니다. 비디오 생성 과정의 다양한 측면, 예를 들어 화면비, 지속 시간, 해상도 등을 사용자 정의할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `프롬프트 텍스트` | STRING | 예 | - | 생성하려는 비디오 내용을 설명하는 주요 텍스트 설명 |
| `부정 프롬프트` | STRING | 예 | - | 생성된 비디오에 나타나지 않기를 바라는 내용을 설명하는 텍스트 |
| `시드` | INT | 예 | - | 재현 가능한 결과를 위해 생성의 무작위성을 제어하는 숫자 |
| `해상도` | STRING | 예 | - | 출력 비디오의 해상도 설정 |
| `길이` | INT | 예 | - | 비디오의 길이(초 단위) |
| `종횡비` | FLOAT | 아니오 | 0.4 - 2.5 | 화면비(너비 / 높이) (기본값: 1.7777777777777777) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | Pika API에서 반환된 생성된 비디오 파일 |
