Pika Text2Video v2.2 ノードは、テキストプロンプトを Pika API バージョン 2.2 に送信して動画を生成します。このノードは、Pika の AI 動画生成サービスを使用して、テキストによる説明を動画に変換します。アスペクト比、長さ、解像度など、動画生成プロセスの様々な側面をカスタマイズすることができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `プロンプトテキスト` | STRING | はい | - | 動画で生成したい内容を説明する主要なテキスト記述 |
| `ネガティブプロンプト` | STRING | はい | - | 生成された動画に出現してほしくない内容を説明するテキスト |
| `シード` | INT | はい | - | 再現性のある結果を得るために、生成のランダム性を制御する数値 |
| `解像度` | STRING | はい | - | 出力動画の解像度設定 |
| `再生時間` | INT | はい | - | 動画の長さ（秒単位） |
| `アスペクト比` | FLOAT | いいえ | 0.4 - 2.5 | アスペクト比（幅 / 高さ）（デフォルト: 1.7777777777777777） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `output` | VIDEO | Pika API から返された生成済み動画ファイル |
