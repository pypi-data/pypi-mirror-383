Узел SamplerEulerAncestral создаёт сэмплер Эйлера-Анцестрала для генерации изображений. Этот сэмплер использует специфический математический подход, сочетающий интегрирование Эйлера с техниками анцестрального сэмплинга для создания вариаций изображений. Узел позволяет настраивать поведение сэмплинга путём регулировки параметров, контролирующих случайность и размер шага в процессе генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `eta` | FLOAT | Да | 0.0 - 100.0 | Управляет размером шага и стохастичностью процесса сэмплинга (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время сэмплинга (по умолчанию: 1.0) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Возвращает настроенный сэмплер Эйлера-Анцестрала, который может быть использован в конвейере сэмплинга |
