SaveVideo 노드는 입력된 비디오 콘텐츠를 ComfyUI 출력 디렉토리에 저장합니다. 저장할 파일의 파일명 접두사, 비디오 형식 및 코덱을 지정할 수 있습니다. 이 노드는 카운터 증분을 통한 파일 이름 지정을 자동으로 처리하며, 저장된 비디오에 워크플로우 메타데이터를 포함할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `비디오` | VIDEO | 예 | - | 저장할 비디오입니다. |
| `파일명 접두사` | STRING | 아니요 | - | 저장할 파일의 접두사입니다. %date:yyyy-MM-dd% 또는 %Empty Latent Image.width%와 같은 형식 정보를 포함하여 노드의 값을 포함할 수 있습니다 (기본값: "video/ComfyUI"). |
| `포맷` | COMBO | 아니요 | 사용 가능한 여러 옵션 | 비디오를 저장할 형식입니다 (기본값: "auto"). |
| `코덱` | COMBO | 아니요 | 사용 가능한 여러 옵션 | 비디오에 사용할 코덱입니다 (기본값: "auto"). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| *출력 없음* | - | 이 노드는 어떠한 출력 데이터도 반환하지 않습니다. |
