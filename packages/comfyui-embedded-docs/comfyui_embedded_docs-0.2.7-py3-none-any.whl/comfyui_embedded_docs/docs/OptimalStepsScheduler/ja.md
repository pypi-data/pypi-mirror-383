OptimalStepsSchedulerノードは、選択されたモデルタイプとステップ設定に基づいて、拡散モデルのノイズスケジュールシグマ値を計算します。このノードはデノイズパラメータに応じて総ステップ数を調整し、要求されたステップ数に合わせてノイズレベルを補間します。拡散サンプリングプロセスで使用されるノイズレベルを決定する一連のシグマ値を返します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `model_type` | COMBO | はい | "FLUX"<br>"Wan"<br>"Chroma" | ノイズレベル計算に使用する拡散モデルのタイプ |
| `ステップ数` | INT | はい | 3-1000 | 計算するサンプリングステップの総数（デフォルト: 20） |
| `ノイズ除去` | FLOAT | いいえ | 0.0-1.0 | デノイズ強度を制御し、実効ステップ数を調整します（デフォルト: 1.0） |

**注記:** `denoise`が1.0未満に設定されている場合、ノードは実効ステップ数を`steps * denoise`として計算します。`denoise`が0.0に設定されている場合、ノードは空のテンソルを返します。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | 拡散サンプリング用のノイズスケジュールを表す一連のシグマ値 |
