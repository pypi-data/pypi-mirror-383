Узел OptimalStepsScheduler вычисляет сигмы шумового расписания для диффузионных моделей на основе выбранного типа модели и конфигурации шагов. Он корректирует общее количество шагов в соответствии с параметром денойзинга и интерполирует уровни шума, чтобы соответствовать запрошенному количеству шагов. Узел возвращает последовательность значений сигмы, которые определяют уровни шума, используемые в процессе сэмплирования диффузии.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model_type` | COMBO | Да | "FLUX"<br>"Wan"<br>"Chroma" | Тип диффузионной модели, используемый для расчёта уровней шума |
| `шаги` | INT | Да | 3-1000 | Общее количество шагов сэмплирования для расчёта (по умолчанию: 20) |
| `шумоподавление` | FLOAT | Нет | 0.0-1.0 | Управляет силой денойзинга, которая корректирует эффективное количество шагов (по умолчанию: 1.0) |

**Примечание:** Когда параметр `denoise` установлен меньше 1.0, узел вычисляет эффективные шаги как `steps * denoise`. Если `denoise` установлен в 0.0, узел возвращает пустой тензор.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | Последовательность значений сигмы, представляющая шумовое расписание для сэмплирования диффузии |
