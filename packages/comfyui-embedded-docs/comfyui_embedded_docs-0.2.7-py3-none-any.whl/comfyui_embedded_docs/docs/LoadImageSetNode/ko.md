LoadImageSetNode는 배치 처리 및 학습 목적으로 입력 디렉터리에서 여러 이미지를 불러옵니다. 다양한 이미지 형식을 지원하며, 선택적으로 다른 방법을 사용하여 이미지 크기를 조정할 수 있습니다. 이 노드는 선택된 모든 이미지를 배치로 처리하고 단일 텐서로 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | 여러 이미지 파일 | 입력 디렉터리에서 여러 이미지를 선택합니다. PNG, JPG, JPEG, WEBP, BMP, GIF, JPE, APNG, TIF, TIFF 형식을 지원합니다. 이미지의 배치 선택이 가능합니다. |
| `resize_method` | STRING | 아니오 | "None"<br>"Stretch"<br>"Crop"<br>"Pad" | 불러온 이미지의 크기를 조정하는 선택적 방법입니다 (기본값: "None"). 원본 크기를 유지하려면 "None", 강제로 크기를 조정하려면 "Stretch", 종횡비를 유지하며 자르려면 "Crop", 종횡비를 유지하며 패딩을 추가하려면 "Pad"를 선택합니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 불러온 모든 이미지를 배치로 포함하여 후속 처리를 위한 텐서입니다. |
