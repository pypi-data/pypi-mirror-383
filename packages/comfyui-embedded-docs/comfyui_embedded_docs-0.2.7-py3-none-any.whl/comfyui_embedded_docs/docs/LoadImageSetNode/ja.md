LoadImageSetNodeは、バッチ処理やトレーニングの目的で入力ディレクトリから複数の画像を読み込みます。様々な画像フォーマットをサポートし、オプションで異なる方法を使用して画像のサイズを変更することができます。このノードは選択されたすべての画像をバッチとして処理し、単一のテンソルとして返します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | はい | 複数の画像ファイル | 入力ディレクトリから複数の画像を選択します。PNG、JPG、JPEG、WEBP、BMP、GIF、JPE、APNG、TIF、TIFFフォーマットをサポートします。画像のバッチ選択が可能です。 |
| `resize_method` | STRING | いいえ | "None"<br>"Stretch"<br>"Crop"<br>"Pad" | 読み込んだ画像のサイズを変更するオプションの方法（デフォルト: "None"）。"None"で元のサイズを維持、"Stretch"で強制的にリサイズ、"Crop"でアスペクト比を維持してクロップ、"Pad"でアスペクト比を維持してパディングを追加します。 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 読み込まれたすべての画像をバッチとして含むテンソル。さらなる処理のために使用されます。 |
