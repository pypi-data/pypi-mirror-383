EmptySD3LatentImageノードは、Stable Diffusion 3モデル用に特別にフォーマットされた空白の潜在画像テンソルを作成します。SD3パイプラインで期待される正しい次元と構造を持つ、ゼロで埋められたテンソルを生成します。これは画像生成ワークフローの開始点として一般的に使用されます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `幅` | INT | はい | 16 から MAX_RESOLUTION (ステップ: 16) | 出力潜在画像の幅（ピクセル単位）（デフォルト: 1024） |
| `高さ` | INT | はい | 16 から MAX_RESOLUTION (ステップ: 16) | 出力潜在画像の高さ（ピクセル単位）（デフォルト: 1024） |
| `バッチサイズ` | INT | はい | 1 から 4096 | バッチで生成する潜在画像の数（デフォルト: 1） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | SD3互換の次元を持つ空白サンプルを含む潜在テンソル |
