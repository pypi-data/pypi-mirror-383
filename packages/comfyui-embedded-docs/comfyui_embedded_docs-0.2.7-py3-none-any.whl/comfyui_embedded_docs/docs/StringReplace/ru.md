Узел StringReplace выполняет операции замены текста во входных строках. Он ищет указанную подстроку во входном тексте и заменяет все её вхождения другой подстрокой. Этот узел возвращает изменённую строку со всеми применёнными заменами.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | - | Входная текстовая строка, в которой будут выполняться замены |
| `find` | STRING | Да | - | Подстрока для поиска во входном тексте |
| `replace` | STRING | Да | - | Текст замены, который заменит все найденные вхождения |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Изменённая строка со всеми вхождениями искомого текста, заменёнными на текст замены |
