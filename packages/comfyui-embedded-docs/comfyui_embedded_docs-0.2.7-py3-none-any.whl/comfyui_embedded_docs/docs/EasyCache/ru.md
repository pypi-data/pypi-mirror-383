Узел EasyCache реализует встроенную систему кэширования моделей для повышения производительности путем повторного использования ранее вычисленных шагов в процессе сэмплирования. Он добавляет функциональность EasyCache к модели с настраиваемыми порогами для определения момента начала и окончания использования кэша в ходе сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой добавляется EasyCache. |
| `reuse_threshold` | FLOAT | Нет | 0.0 - 3.0 | Порог повторного использования кэшированных шагов (по умолчанию: 0.2). |
| `start_percent` | FLOAT | Нет | 0.0 - 1.0 | Относительный шаг сэмплирования для начала использования EasyCache (по умолчанию: 0.15). |
| `end_percent` | FLOAT | Нет | 0.0 - 1.0 | Относительный шаг сэмплирования для окончания использования EasyCache (по умолчанию: 0.95). |
| `verbose` | BOOLEAN | Нет | - | Включение вывода подробной информации в журнал (по умолчанию: False). |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модель с добавленной функциональностью EasyCache. |
