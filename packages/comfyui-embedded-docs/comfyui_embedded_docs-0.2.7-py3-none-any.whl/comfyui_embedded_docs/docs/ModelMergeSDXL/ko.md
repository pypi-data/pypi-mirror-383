ModelMergeSDXL 노드를 사용하면 두 개의 SDXL 모델을 아키텍처의 다양한 부분에 대한 각 모델의 영향을 조정하여 혼합할 수 있습니다. 시간 임베딩, 레이블 임베딩 및 모델 구조 내 다양한 블록에 각 모델이 얼마나 기여하는지 제어할 수 있습니다. 이를 통해 두 입력 모델의 특성을 결합한 하이브리드 모델이 생성됩니다.
