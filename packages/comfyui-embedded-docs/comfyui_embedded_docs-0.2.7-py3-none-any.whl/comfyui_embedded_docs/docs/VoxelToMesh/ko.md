VoxelToMesh 노드는 3D 복셀 데이터를 다양한 알고리즘을 사용하여 메시 형상으로 변환합니다. 복셀 그리드를 처리하고 3D 메시 표현을 형성하는 정점과 면을 생성합니다. 이 노드는 여러 변환 알고리즘을 지원하며 표면 추출을 제어하기 위한 임계값 조정이 가능합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `복셀` | VOXEL | 예 | - | 메시 형상으로 변환할 입력 복셀 데이터 |
| `알고리즘` | COMBO | 예 | "surface net"<br>"basic" | 복셀 데이터에서 메시 변환에 사용되는 알고리즘 |
| `임계값` | FLOAT | 예 | -1.0 ~ 1.0 | 표면 추출을 위한 임계값 (기본값: 0.6) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MESH` | MESH | 정점과 면을 포함하는 생성된 3D 메시 |
