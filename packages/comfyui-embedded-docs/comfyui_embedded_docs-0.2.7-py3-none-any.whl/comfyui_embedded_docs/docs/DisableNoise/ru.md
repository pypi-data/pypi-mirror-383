Узел DisableNoise предоставляет пустую конфигурацию шума, которая может использоваться для отключения генерации шума в процессах сэмплирования. Он возвращает специальный объект шума, не содержащий данных о шуме, что позволяет другим узлам пропускать операции, связанные с шумом, при подключении к этому выходу.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| *Нет входных параметров* | - | - | - | Этот узел не требует каких-либо входных параметров. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `NOISE` | NOISE | Возвращает пустую конфигурацию шума, которая может использоваться для отключения генерации шума в процессах сэмплирования. |
