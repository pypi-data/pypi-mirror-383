Создает изображения синхронно через конечную точку GPT Image 1 от OpenAI. Этот узел может создавать новые изображения по текстовым промптам или редактировать существующие изображения, если предоставлено входное изображение и опциональная маска.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовый промпт для GPT Image 1 (по умолчанию: "") |
| `seed` | INT | Нет | от 0 до 2147483647 | Случайное зерно для генерации (по умолчанию: 0) - пока не реализовано в бэкенде |
| `quality` | COMBO | Нет | "low"<br>"medium"<br>"high" | Качество изображения, влияет на стоимость и время генерации (по умолчанию: "low") |
| `background` | COMBO | Нет | "opaque"<br>"transparent" | Возвращать изображение с фоном или без (по умолчанию: "opaque") |
| `size` | COMBO | Нет | "auto"<br>"1024x1024"<br>"1024x1536"<br>"1536x1024" | Размер изображения (по умолчанию: "auto") |
| `n` | INT | Нет | от 1 до 8 | Сколько изображений сгенерировать (по умолчанию: 1) |
| `image` | IMAGE | Нет | - | Опорное изображение для редактирования (по умолчанию: None) |
| `mask` | MASK | Нет | - | Маска для инпейнтинга (белые области будут заменены) (по умолчанию: None) |

**Ограничения параметров:**

- При предоставлении `image` узел переключается в режим редактирования изображений
- `mask` можно использовать только при наличии `image`
- При использовании `mask` поддерживаются только одиночные изображения (размер пакета должен быть равен 1)
- `mask` и `image` должны быть одного размера

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Сгенерированные или отредактированные изображения |
