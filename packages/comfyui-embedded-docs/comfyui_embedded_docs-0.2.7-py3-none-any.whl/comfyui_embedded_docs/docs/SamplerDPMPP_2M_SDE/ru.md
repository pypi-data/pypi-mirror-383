Узел SamplerDPMPP_2M_SDE создаёт сэмплер DPM++ 2M SDE для диффузионных моделей. Этот сэмплер использует решатели дифференциальных уравнений второго порядка со стохастическими дифференциальными уравнениями для генерации сэмплов. Он предоставляет различные типы решателей и опции управления шумом для контроля процесса сэмплирования.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `тип_решателя` | STRING | Да | `"midpoint"`<br>`"heun"` | Тип решателя дифференциальных уравнений, используемый в процессе сэмплирования |
| `eta` | FLOAT | Да | 0.0 - 100.0 | Управляет стохастичностью процесса сэмплирования (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время сэмплирования (по умолчанию: 1.0) |
| `устройство_шума` | STRING | Да | `"gpu"`<br>`"cpu"` | Устройство, на котором выполняются вычисления, связанные с шумом |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Настроенный объект сэмплера, готовый к использованию в конвейере сэмплирования |
