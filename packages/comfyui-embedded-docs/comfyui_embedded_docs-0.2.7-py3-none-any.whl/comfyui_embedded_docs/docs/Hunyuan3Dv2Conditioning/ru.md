Узел Hunyuan3Dv2Conditioning обрабатывает выходные данные CLIP vision для генерации кондиционирующих данных видеомоделей. Он извлекает эмбеддинги последнего скрытого состояния из выходных данных vision и создает как положительные, так и отрицательные пары кондиционирования. Положительное кондиционирование использует фактические эмбеддинги, тогда как отрицательное кондиционирование использует эмбеддинги с нулевыми значениями той же формы.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `выход_clip_vision` | CLIP_VISION_OUTPUT | Да | - | Выходные данные модели CLIP vision, содержащие визуальные эмбеддинги |

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `отрицательный` | CONDITIONING | Положительные кондиционирующие данные, содержащие эмбеддинги CLIP vision |
| `negative` | CONDITIONING | Отрицательные кондиционирующие данные, содержащие эмбеддинги с нулевыми значениями, соответствующие форме положительных эмбеддингов |
