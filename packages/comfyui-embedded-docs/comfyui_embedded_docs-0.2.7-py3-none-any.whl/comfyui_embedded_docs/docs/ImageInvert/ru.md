Узел `ImageInvert` предназначен для инвертирования цветов изображения, эффективно преобразуя цветовое значение каждого пикселя в его дополнительный цвет на цветовом круге. Эта операция полезна для создания негативных изображений или визуальных эффектов, требующих инвертирования цветов.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `изображение`   | `IMAGE`     | Параметр 'image' представляет входное изображение для инвертирования. Он важен для указания целевого изображения, цвета которого должны быть инвертированы, что влияет на выполнение узла и визуальный результат процесса инвертирования. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `изображение`   | `IMAGE`     | На выходе получается инвертированная версия входного изображения, где цветовое значение каждого пикселя преобразовано в его дополнительный цвет. |
