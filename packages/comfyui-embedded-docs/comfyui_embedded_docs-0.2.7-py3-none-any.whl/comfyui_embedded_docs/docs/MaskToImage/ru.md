Узел `MaskToImage` предназначен для преобразования маски в формат изображения. Это преобразование позволяет визуализировать маски и обрабатывать их как изображения, обеспечивая связь между операциями на основе масок и приложениями для работы с изображениями.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `маска`    | `MASK`      | Входная маска является основой для процесса преобразования, выступая в качестве исходных данных, которые будут трансформированы в формат изображения. Этот входной параметр определяет форму и содержимое итогового изображения. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | Выходные данные представляют собой изображение, созданное на основе входной маски, что позволяет проводить визуальную проверку и дальнейшие манипуляции, характерные для работы с изображениями. |
