UNetSelfAttentionMultiply 노드는 UNet 모델의 자기 주의 메커니즘 내 쿼리, 키, 값, 출력 구성 요소에 곱셈 계수를 적용합니다. 주의 가중치가 모델 동작에 미치는 영향을 실험하기 위해 주의 계산의 다양한 부분을 조절할 수 있도록 합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | 주의 스케일링 계수로 수정할 UNet 모델 |
| `q` | FLOAT | 아니오 | 0.0 - 10.0 | 쿼리 구성 요소에 대한 곱셈 계수 (기본값: 1.0) |
| `k` | FLOAT | 아니오 | 0.0 - 10.0 | 키 구성 요소에 대한 곱셈 계수 (기본값: 1.0) |
| `v` | FLOAT | 아니오 | 0.0 - 10.0 | 값 구성 요소에 대한 곱셈 계수 (기본값: 1.0) |
| `out` | FLOAT | 아니오 | 0.0 - 10.0 | 출력 구성 요소에 대한 곱셈 계수 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 주의 구성 요소가 조절된 수정된 UNet 모델 |
