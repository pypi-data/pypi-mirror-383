Узел PairConditioningSetDefaultAndCombine устанавливает значения кондиционирования по умолчанию и объединяет их с входными данными кондиционирования. Он принимает позитивные и негативные входные данные кондиционирования вместе с их аналогами по умолчанию, затем обрабатывает их через систему хуков ComfyUI для получения итоговых выходных данных кондиционирования, которые включают значения по умолчанию.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Основной позитивный вход кондиционирования для обработки |
| `negative` | CONDITIONING | Да | - | Основной негативный вход кондиционирования для обработки |
| `positive_DEFAULT` | CONDITIONING | Да | - | Значения позитивного кондиционирования по умолчанию, используемые как резервные |
| `negative_DEFAULT` | CONDITIONING | Да | - | Значения негативного кондиционирования по умолчанию, используемые как резервные |
| `hooks` | HOOKS | Нет | - | Опциональная группа хуков для пользовательской логики обработки |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Обработанное позитивное кондиционирование со включенными значениями по умолчанию |
| `negative` | CONDITIONING | Обработанное негативное кондиционирование со включенными значениями по умолчанию |
