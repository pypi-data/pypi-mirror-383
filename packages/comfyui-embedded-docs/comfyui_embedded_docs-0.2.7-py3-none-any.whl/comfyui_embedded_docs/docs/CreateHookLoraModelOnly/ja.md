このノードは、モデルコンポーネントにのみ適用されるLoRA（Low-Rank Adaptation）フックを作成します。これにより、CLIPコンポーネントに影響を与えることなくモデルの動作を変更できます。LoRAファイルを読み込み、指定された強度でモデルに適用し、CLIPコンポーネントは変更しません。このノードは、以前のフックと連鎖させて複雑な変更パイプラインを作成することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `lora_name` | STRING | はい | 複数オプション利用可能 | lorasフォルダから読み込むLoRAファイルの名前 |
| `strength_model` | FLOAT | はい | -20.0 から 20.0 | モデルコンポーネントにLoRAを適用する際の強度乗数（デフォルト: 1.0） |
| `prev_hooks` | HOOKS | いいえ | - | このフックと連鎖させるためのオプションの以前のフック |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `hooks` | HOOKS | モデル処理に適用可能な作成されたLoRAフック |
