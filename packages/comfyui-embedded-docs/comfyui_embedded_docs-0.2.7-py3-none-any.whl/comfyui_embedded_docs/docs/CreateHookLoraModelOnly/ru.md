Этот узел создаёт LoRA (Low-Rank Adaptation) хук, который применяется только к компоненту модели, позволяя изменять поведение модели без воздействия на компонент CLIP. Он загружает файл LoRA и применяет его с заданной силой к модели, оставляя компонент CLIP неизменным. Узел может объединяться в цепочку с предыдущими хуками для создания сложных конвейеров модификации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `название_lora` | STRING | Да | Доступно несколько вариантов | Имя файла LoRA для загрузки из папки loras |
| `сила_модели` | FLOAT | Да | -20.0 до 20.0 | Множитель силы применения LoRA к компоненту модели (по умолчанию: 1.0) |
| `предыдущие_хуки` | HOOKS | Нет | - | Опциональные предыдущие хуки для объединения в цепочку с этим хуком |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `hooks` | HOOKS | Созданный LoRA хук, который может быть применён к обработке модели |
