Узел SolidMask создаёт однородную маску с указанным значением по всей своей площади. Он предназначен для создания масок определённых размеров и интенсивности, что полезно в различных задачах обработки изображений и работы с масками.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `значение`   | FLOAT       | Определяет значение интенсивности маски, влияя на её общий вид и полезность в последующих операциях. |
| `ширина`   | INT         | Определяет ширину генерируемой маски, напрямую влияя на её размер и соотношение сторон. |
| `высота`  | INT         | Задаёт высоту генерируемой маски, влияя на её размер и соотношение сторон. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `mask`    | MASK        | Выводит однородную маску с указанными размерами и значением. |
