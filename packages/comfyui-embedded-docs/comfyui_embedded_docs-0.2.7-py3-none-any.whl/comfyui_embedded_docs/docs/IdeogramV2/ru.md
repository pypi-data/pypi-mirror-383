Узел Ideogram V2 генерирует изображения с использованием модели искусственного интеллекта Ideogram V2. Он принимает текстовые промпты и различные настройки генерации для создания изображений через API-сервис. Узел поддерживает различные соотношения сторон, разрешения и параметры стиля для настройки выходных изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Промпт для генерации изображения (по умолчанию: пустая строка) |
| `turbo` | BOOLEAN | Нет | - | Использовать ли турбо-режим (более быстрая генерация, потенциально более низкое качество) (по умолчанию: False) |
| `aspect_ratio` | COMBO | Нет | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | Соотношение сторон для генерации изображения. Игнорируется, если разрешение не установлено в AUTO. (по умолчанию: "1:1") |
| `resolution` | COMBO | Нет | "Auto"<br>"1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | Разрешение для генерации изображения. Если не установлено в AUTO, переопределяет настройку aspect_ratio. (по умолчанию: "Auto") |
| `magic_prompt_option` | COMBO | Нет | "AUTO"<br>"ON"<br>"OFF" | Определяет, следует ли использовать MagicPrompt в генерации (по умолчанию: "AUTO") |
| `seed` | INT | Нет | 0-2147483647 | Случайное зерно для генерации (по умолчанию: 0) |
| `style_type` | COMBO | Нет | "AUTO"<br>"GENERAL"<br>"REALISTIC"<br>"DESIGN"<br>"RENDER_3D"<br>"ANIME" | Тип стиля для генерации (только V2) (по умолчанию: "NONE") |
| `negative_prompt` | STRING | Нет | - | Описание того, что следует исключить из изображения (по умолчанию: пустая строка) |
| `num_images` | INT | Нет | 1-8 | Количество генерируемых изображений (по умолчанию: 1) |

**Примечание:** Когда `resolution` не установлен в "Auto", он переопределяет настройку `aspect_ratio`. Параметр `num_images` имеет максимальное ограничение в 8 изображений за одну генерацию.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное(ые) изображение(я) из модели Ideogram V2 |
