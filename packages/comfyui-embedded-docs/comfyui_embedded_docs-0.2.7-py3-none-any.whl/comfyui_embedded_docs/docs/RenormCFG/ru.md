Узел RenormCFG модифицирует процесс классификаторно-свободного управления (CFG) в диффузионных моделях, применяя условное масштабирование и нормализацию. Он корректирует процесс удаления шума на основе заданных пороговых значений временного шага и факторов перенормировки для управления влиянием условных и безусловных предсказаний во время генерации изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Диффузионная модель, к которой применяется перенормированное CFG |
| `cfg_trunc` | FLOAT | Нет | 0.0 - 100.0 | Пороговое значение временного шага для применения масштабирования CFG (по умолчанию: 100.0) |
| `renorm_cfg` | FLOAT | Нет | 0.0 - 100.0 | Фактор перенормировки для управления силой условного управления (по умолчанию: 1.0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с применённой функцией перенормированного CFG |
