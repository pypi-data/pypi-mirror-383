Узел Moonvalley Marey Video to Video преобразует входное видео в новое видео на основе текстового описания. Он использует API Moonvalley для генерации видео, соответствующих вашему промпту, сохраняя при этом характеристики движения или позы из исходного видео. Вы можете управлять стилем и содержанием выходного видео с помощью текстовых промптов и различных параметров генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Описывает видео для генерации (многострочный ввод) |
| `negative_prompt` | STRING | Нет | - | Текст негативного промпта (по умолчанию: расширенный список негативных дескрипторов) |
| `seed` | INT | Да | 0-4294967295 | Значение случайного сида (по умолчанию: 9) |
| `video` | VIDEO | Да | - | Референсное видео, используемое для генерации выходного видео. Должно быть длиной не менее 5 секунд. Видео длиннее 5 секунд будут автоматически обрезаны. Поддерживается только формат MP4. |
| `control_type` | COMBO | Нет | "Motion Transfer"<br>"Pose Transfer" | Выбор типа контроля (по умолчанию: "Motion Transfer") |
| `motion_intensity` | INT | Нет | 0-100 | Используется только если control_type установлен в 'Motion Transfer' (по умолчанию: 100) |
| `steps` | INT | Да | 1-100 | Количество шагов инференса (по умолчанию: 33) |

**Примечание:** Параметр `motion_intensity` применяется только когда `control_type` установлен в "Motion Transfer". При использовании "Pose Transfer" этот параметр игнорируется.

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное выходное видео |
