Узел SplitSigmasDenoise разделяет последовательность значений сигма на две части на основе параметра силы шумоподавления. Он разделяет входные сигма-значения на последовательности высоких и низких сигм, где точка разделения определяется умножением общего количества шагов на коэффициент шумоподавления. Это позволяет разделить график шума на диапазоны различной интенсивности для специализированной обработки.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `сигмы` | SIGMAS | Да | - | Входная последовательность значений сигма, представляющая график шума |
| `шумоподавление` | FLOAT | Да | 0.0 - 1.0 | Коэффициент силы шумоподавления, определяющий место разделения последовательности сигма (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `низкие_сигмы` | SIGMAS | Первая часть последовательности сигма, содержащая более высокие значения сигма |
| `low_sigmas` | SIGMAS | Вторая часть последовательности сигма, содержащая более низкие значения сигма |
