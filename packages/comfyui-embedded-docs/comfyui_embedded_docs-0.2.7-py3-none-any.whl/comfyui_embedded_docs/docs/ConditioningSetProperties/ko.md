ConditioningSetProperties 노드는 조건화 데이터의 속성을 강도, 영역 설정 조정 및 선택적 마스크나 타임스텝 범위 적용을 통해 수정합니다. 이 노드를 사용하면 이미지 생성 과정에서 조건화 데이터가 적용되는 방식을 영향을 미치는 특정 매개변수를 설정하여 조건화가 생성 과정에 미치는 영향을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 타입 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `새 조건` | CONDITIONING | 필수 | - | - | 수정할 조건화 데이터 |
| `강도` | FLOAT | 필수 | 1.0 | 0.0-10.0 | 조건화 효과의 강도를 제어합니다 |
| `조건 영역 설정` | STRING | 필수 | default | ["default", "mask bounds"] | 조건화 영역이 적용되는 방식을 결정합니다 |
| `마스크` | MASK | 선택사항 | - | - | 조건화가 적용되는 영역을 제한하는 선택적 마스크 |
| `후크` | HOOKS | 선택사항 | - | - | 사용자 정의 처리를 위한 선택적 훅 함수 |
| `타임스텝 범위` | TIMESTEPS_RANGE | 선택사항 | - | - | 조건화가 활성화되는 시기를 제한하는 선택적 타임스텝 범위 |

**참고:** `mask`가 제공될 때, `set_cond_area` 매개변수를 "mask bounds"로 설정하면 조건화 적용을 마스크된 영역으로만 제한할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 업데이트된 속성을 가진 수정된 조건화 데이터 |
