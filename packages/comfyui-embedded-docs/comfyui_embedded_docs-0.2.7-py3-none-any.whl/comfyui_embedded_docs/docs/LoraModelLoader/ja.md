LoraModelLoaderノードは、学習済みのLoRA（Low-Rank Adaptation）の重みを拡散モデルに適用します。学習済みのLoRAモデルからLoRAの重みを読み込み、その影響力の強度を調整することで、ベースモデルを修正します。これにより、拡散モデルを最初から再学習させることなく、その動作をカスタマイズすることができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | はい | - | LoRAを適用する拡散モデル。 |
| `lora` | LORA_MODEL | はい | - | 拡散モデルに適用するLoRAモデル。 |
| `strength_model` | FLOAT | はい | -100.0 ～ 100.0 | 拡散モデルを変更する強度。この値は負の値も設定可能です（デフォルト: 1.0）。 |

**注意:** `strength_model` が0に設定されている場合、ノードはLoRAの変更を一切適用せずに元のモデルを返します。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `model` | MODEL | LoRAの重みが適用された、修正後の拡散モデル。 |
