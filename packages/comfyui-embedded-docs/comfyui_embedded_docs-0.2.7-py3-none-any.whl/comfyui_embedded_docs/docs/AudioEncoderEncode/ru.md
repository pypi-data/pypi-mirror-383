Узел AudioEncoderEncode обрабатывает аудиоданные, кодируя их с помощью модели аудиоэнкодера. Он принимает аудиовход и преобразует его в закодированное представление, которое может быть использовано для дальнейшей обработки в конвейере кондиционирования. Этот узел преобразует исходные аудиоволновые формы в формат, пригодный для машинного обучения на основе аудио.

## Входы

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `audio_encoder` | AUDIO_ENCODER | Обязательный | - | - | Модель аудиоэнкодера, используемая для обработки аудиовхода |
| `audio` | AUDIO | Обязательный | - | - | Аудиоданные, содержащие информацию о волновой форме и частоте дискретизации |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | AUDIO_ENCODER_OUTPUT | Закодированное аудиопредставление, сгенерированное аудиоэнкодером |
