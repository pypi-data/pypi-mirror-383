Создает изображения синхронно через конечную точку OpenAI DALL·E 3. Этот узел принимает текстовый промт и создает соответствующие изображения с использованием модели OpenAI DALL·E 3, позволяя указать качество, стиль и размеры изображения.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовый промт для DALL·E (по умолчанию: "") |
| `seed` | INT | Нет | 0 до 2147483647 | пока не реализовано в бэкенде (по умолчанию: 0) |
| `качество` | COMBO | Нет | "standard"<br>"hd" | Качество изображения (по умолчанию: "standard") |
| `стиль` | COMBO | Нет | "natural"<br>"vivid" | Vivid заставляет модель склоняться к созданию гиперреалистичных и драматичных изображений. Natural заставляет модель создавать более естественные, менее гиперреалистичные изображения. (по умолчанию: "natural") |
| `размер` | COMBO | Нет | "1024x1024"<br>"1024x1792"<br>"1792x1024" | Размер изображения (по умолчанию: "1024x1024") |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Сгенерированное изображение от DALL·E 3 |
