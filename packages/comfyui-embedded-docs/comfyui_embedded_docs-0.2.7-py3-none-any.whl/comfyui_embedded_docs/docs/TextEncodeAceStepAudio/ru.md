Узел TextEncodeAceStepAudio обрабатывает текстовые входные данные для аудио-кондиционирования, объединяя теги и тексты песен в токены с последующим кодированием с регулируемой силой влияния текста. Он принимает модель CLIP вместе с текстовыми описаниями и текстами песен, токенизирует их вместе и генерирует данные кондиционирования, подходящие для задач генерации аудио. Узел позволяет точно настраивать влияние текста песен через параметр силы, который контролирует их воздействие на конечный результат.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | - | Модель CLIP, используемая для токенизации и кодирования |
| `tags` | STRING | Да | - | Текстовые теги или описания для аудио-кондиционирования (поддерживает многострочный ввод и динамические промпты) |
| `lyrics` | STRING | Да | - | Текст песен для аудио-кондиционирования (поддерживает многострочный ввод и динамические промпты) |
| `lyrics_strength` | FLOAT | Нет | 0.0 - 10.0 | Контролирует силу влияния текста песен на выходные данные кондиционирования (по умолчанию: 1.0, шаг: 0.01) |

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | Закодированные данные кондиционирования, содержащие обработанные текстовые токены с примененной силой влияния текста песен |
