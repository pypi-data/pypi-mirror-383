Узел TripoTextureNode генерирует текстурированные 3D-модели с использованием Tripo API. Он принимает идентификатор задачи модели и применяет генерацию текстур с различными опциями, включая PBR-материалы, настройки качества текстур и методы выравнивания. Узел взаимодействует с Tripo API для обработки запроса на генерацию текстур и возвращает результирующий файл модели и идентификатор задачи.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model_task_id` | MODEL_TASK_ID | Да | - | Идентификатор задачи модели, к которой применяются текстуры |
| `texture` | BOOLEAN | Нет | - | Следует ли генерировать текстуры (по умолчанию: True) |
| `pbr` | BOOLEAN | Нет | - | Следует ли генерировать PBR-материалы (Physically Based Rendering) (по умолчанию: True) |
| `texture_seed` | INT | Нет | - | Случайное начальное значение для генерации текстур (по умолчанию: 42) |
| `texture_quality` | COMBO | Нет | "standard"<br>"detailed" | Уровень качества для генерации текстур (по умолчанию: "standard") |
| `texture_alignment` | COMBO | Нет | "original_image"<br>"geometry" | Метод выравнивания текстур (по умолчанию: "original_image") |

*Примечание: Этот узел требует токены аутентификации и API-ключи, которые автоматически обрабатываются системой.*

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | STRING | Сгенерированный файл модели с примененными текстурами |
| `model task_id` | MODEL_TASK_ID | Идентификатор задачи для отслеживания процесса генерации текстур |
