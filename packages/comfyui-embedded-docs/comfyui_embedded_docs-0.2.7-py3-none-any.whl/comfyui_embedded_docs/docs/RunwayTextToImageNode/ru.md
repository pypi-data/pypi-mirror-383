Узел Runway Text to Image генерирует изображения из текстовых запросов с использованием модели Gen 4 от Runway. Вы можете предоставить текстовое описание и, при желании, включить эталонное изображение для управления процессом генерации изображения. Узел обрабатывает взаимодействие с API и возвращает сгенерированное изображение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовый запрос для генерации (по умолчанию: "") |
| `ratio` | COMBO | Да | "16:9"<br>"1:1"<br>"21:9"<br>"2:3"<br>"3:2"<br>"4:5"<br>"5:4"<br>"9:16"<br>"9:21" | Соотношение сторон для генерируемого изображения |
| `reference_image` | IMAGE | Нет | - | Опциональное эталонное изображение для управления генерацией |

**Примечание:** Эталонное изображение должно иметь размеры, не превышающие 7999x7999 пикселей, и соотношение сторон в диапазоне от 0,5 до 2,0. Когда эталонное изображение предоставлено, оно направляет процесс генерации изображения.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное изображение на основе текстового запроса и опционального эталонного изображения |
