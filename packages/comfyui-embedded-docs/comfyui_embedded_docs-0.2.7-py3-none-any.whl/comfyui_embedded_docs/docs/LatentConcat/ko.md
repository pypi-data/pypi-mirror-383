LatentConcat 노드는 두 개의 잠재 샘플을 지정된 차원을 따라 결합합니다. 두 개의 잠재 입력을 받아 선택한 축(x, y 또는 t 차원)을 따라 연결합니다. 이 노드는 연결 작업을 수행하기 전에 두 번째 입력의 배치 크기를 첫 번째 입력과 일치하도록 자동으로 조정합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `samples1` | LATENT | 예 | - | 연결할 첫 번째 잠재 샘플 |
| `samples2` | LATENT | 예 | - | 연결할 두 번째 잠재 샘플 |
| `dim` | COMBO | 예 | `"x"`<br>`"-x"`<br>`"y"`<br>`"-y"`<br>`"t"`<br>`"-t"` | 잠재 샘플을 연결할 차원. 양수 값은 samples1을 samples2 앞에 연결하고, 음수 값은 samples2를 samples1 앞에 연결합니다 |

**참고:** 두 번째 잠재 샘플(`samples2`)은 연결 전에 첫 번째 잠재 샘플(`samples1`)의 배치 크기와 일치하도록 자동으로 조정됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | LATENT | 지정된 차원을 따라 두 입력 샘플을 결합한 결과인 연결된 잠재 샘플 |
