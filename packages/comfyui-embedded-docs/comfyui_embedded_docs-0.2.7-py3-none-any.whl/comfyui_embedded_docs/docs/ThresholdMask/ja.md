ThresholdMaskノードは、閾値を適用してマスクをバイナリマスクに変換します。入力マスクの各ピクセルを指定された閾値と比較し、閾値を超えるピクセルを1（白）、閾値以下または等しいピクセルを0（黒）にした新しいマスクを作成します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `マスク` | MASK | はい | - | 処理対象の入力マスク |
| `値` | FLOAT | はい | 0.0 - 1.0 | 二値化のための閾値（デフォルト: 0.5） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `マスク` | MASK | 閾値処理後のバイナリマスク |
