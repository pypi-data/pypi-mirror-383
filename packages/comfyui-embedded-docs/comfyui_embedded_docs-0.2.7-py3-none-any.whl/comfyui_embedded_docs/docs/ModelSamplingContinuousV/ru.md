Узел ModelSamplingContinuousV изменяет поведение сэмплинга модели, применяя параметры непрерывного V-предиктивного сэмплинга. Он создает клон входной модели и настраивает его с пользовательскими параметрами диапазона сигмы для расширенного управления сэмплингом. Это позволяет пользователям точно настраивать процесс сэмплинга с определенными минимальными и максимальными значениями сигмы.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Входная модель для модификации с непрерывным V-предиктивным сэмплингом |
| `выборка` | STRING | Да | "v_prediction" | Метод сэмплинга для применения (в настоящее время поддерживается только V-предикция) |
| `sigma_max` | FLOAT | Да | 0.0 - 1000.0 | Максимальное значение сигмы для сэмплинга (по умолчанию: 500.0) |
| `sigma_min` | FLOAT | Да | 0.0 - 1000.0 | Минимальное значение сигмы для сэмплинга (по умолчанию: 0.03) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с примененным непрерывным V-предиктивным сэмплингом |
