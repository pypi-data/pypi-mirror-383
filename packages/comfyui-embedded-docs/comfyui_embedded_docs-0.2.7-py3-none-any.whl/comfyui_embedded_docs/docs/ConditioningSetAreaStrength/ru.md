Данный узел предназначен для изменения атрибута силы заданного набора кондиционирования, позволяя регулировать степень влияния или интенсивности кондиционирования на процесс генерации.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | Набор кондиционирования, который требуется изменить; представляет текущее состояние кондиционирования, влияющее на процесс генерации. |
| `сила` | `FLOAT` | Значение силы, применяемое к набору кондиционирования, которое определяет интенсивность его влияния. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | Измененный набор кондиционирования с обновленными значениями силы для каждого элемента. |
