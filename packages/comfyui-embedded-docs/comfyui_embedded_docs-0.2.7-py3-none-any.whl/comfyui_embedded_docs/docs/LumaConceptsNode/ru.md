Хранит одну или несколько концепций камеры для использования с узлами Luma Text to Video и Luma Image to Video. Этот узел позволяет выбрать до четырех концепций камеры и при необходимости объединить их с существующими цепочками концепций.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `concept1` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор первой концепции камеры из доступных концепций Luma |
| `concept2` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор второй концепции камеры из доступных концепций Luma |
| `concept3` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор третьей концепции камеры из доступных концепций Luma |
| `concept4` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор четвертой концепции камеры из доступных концепций Luma |
| `luma_concepts` | LUMA_CONCEPTS | Нет | N/A | Дополнительные концепции камеры для добавления к выбранным здесь |

**Примечание:** Все параметры концепций (с `concept1` по `concept4`) могут быть установлены в "None", если вы не хотите использовать все четыре слота для концепций. Узел объединит любые предоставленные `luma_concepts` с выбранными концепциями для создания комбинированной цепочки концепций.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `luma_concepts` | LUMA_CONCEPTS | Комбинированная цепочка концепций камеры, содержащая все выбранные концепции |
