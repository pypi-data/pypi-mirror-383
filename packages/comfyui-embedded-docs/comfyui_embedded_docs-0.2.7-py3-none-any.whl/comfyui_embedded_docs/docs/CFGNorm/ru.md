Узел CFGNorm применяет технику нормализации к процессу классификатор-свободного управления (CFG) в диффузионных моделях. Он корректирует масштаб обесшумленного предсказания, сравнивая нормы условного и безусловного выходов, а затем применяет множитель силы для управления эффектом. Это помогает стабилизировать процесс генерации, предотвращая экстремальные значения в масштабировании управления.

## Входы

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `model` | MODEL | обязательный | - | - | Диффузионная модель, к которой применяется нормализация CFG |
| `strength` | FLOAT | обязательный | 1.0 | 0.0 - 100.0 | Управляет интенсивностью эффекта нормализации, применяемого к масштабированию CFG |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `patched_model` | MODEL | Возвращает модифицированную модель с примененной нормализацией CFG к ее процессу сэмплирования |
