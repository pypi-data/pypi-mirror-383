Узел StableCascade_StageB_Conditioning подготавливает данные кондиционирования для генерации Stage B модели Stable Cascade, объединяя существующую информацию кондиционирования с априорными латентными представлениями из Stage C. Он модифицирует данные кондиционирования, включая в них латентные сэмплы из Stage C, что позволяет процессу генерации использовать априорную информацию для получения более согласованных результатов.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `кондиционирование` | CONDITIONING | Да | - | Данные кондиционирования, которые будут модифицированы с априорной информацией из Stage C |
| `стадия_c` | LATENT | Да | - | Латентное представление из Stage C, содержащее априорные сэмплы для кондиционирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Модифицированные данные кондиционирования с интегрированной априорной информацией из Stage C |
