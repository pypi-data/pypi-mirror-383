Узел TripleCLIPLoader загружает три различные модели текстового кодировщика одновременно и объединяет их в единую модель CLIP. Это полезно для сложных сценариев текстового кодирования, где требуется несколько текстовых кодировщиков, например, в рабочих процессах SD3, которые требуют совместной работы моделей clip-l, clip-g и t5.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip_name1` | STRING | Да | Доступно несколько вариантов | Первая модель текстового кодировщика для загрузки из доступных текстовых кодировщиков |
| `clip_name2` | STRING | Да | Доступно несколько вариантов | Вторая модель текстового кодировщика для загрузки из доступных текстовых кодировщиков |
| `clip_name3` | STRING | Да | Доступно несколько вариантов | Третья модель текстового кодировщика для загрузки из доступных текстовых кодировщиков |

**Примечание:** Все три параметра текстового кодировщика должны быть выбраны из доступных моделей текстовых кодировщиков в вашей системе. Узел загрузит все три модели и объединит их в единую модель CLIP для обработки.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `CLIP` | CLIP | Комбинированная модель CLIP, содержащая все три загруженных текстовых кодировщика |
