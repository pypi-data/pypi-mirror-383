TrimAudioDuration 노드를 사용하면 오디오 파일에서 특정 시간 세그먼트를 잘라낼 수 있습니다. 트림을 시작할 시점과 결과 오디오 클립의 길이를 지정할 수 있습니다. 이 노드는 시간 값을 오디오 프레임 위치로 변환하고 해당 오디오 파형의 일부를 추출하는 방식으로 작동합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | 예 | - | 트림할 오디오 입력 |
| `start_index` | FLOAT | 예 | -0xffffffffffffffff ~ 0xffffffffffffffff | 시작 시간(초 단위), 음수 값을 사용하면 끝에서부터 역으로 계산합니다(소수 단위 초 지원). 기본값: 0.0 |
| `duration` | FLOAT | 예 | 0.0 ~ 0xffffffffffffffff | 지속 시간(초 단위). 기본값: 60.0 |

**참고:** 시작 시간은 종료 시간보다 작아야 하며 오디오 길이 내에 있어야 합니다. 음수 시작 값은 오디오의 끝에서부터 역방향으로 계산합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 지정된 시작 시간과 지속 시간으로 트림된 오디오 세그먼트 |
