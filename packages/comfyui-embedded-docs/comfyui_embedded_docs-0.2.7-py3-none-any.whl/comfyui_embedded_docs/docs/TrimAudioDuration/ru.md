Узел TrimAudioDuration позволяет вырезать определённый временной сегмент из аудиофайла. Вы можете указать, с какого момента начать обрезку и какой продолжительности должен быть итоговый аудиоклип. Узел работает путём преобразования временных значений в позиции аудиокадров и извлечения соответствующей части аудиоволны.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | Да | - | Аудиовход для обрезки |
| `start_index` | FLOAT | Да | -0xffffffffffffffff до 0xffffffffffffffff | Время начала в секундах, может быть отрицательным для отсчёта с конца (поддерживает доли секунд). По умолчанию: 0.0 |
| `duration` | FLOAT | Да | 0.0 до 0xffffffffffffffff | Продолжительность в секундах. По умолчанию: 60.0 |

**Примечание:** Время начала должно быть меньше времени окончания и находиться в пределах длины аудио. Отрицательные значения начала отсчитываются назад от конца аудио.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `audio` | AUDIO | Обрезанный аудиосегмент с указанным временем начала и продолжительностью |
