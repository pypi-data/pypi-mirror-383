Узел SaveAudioMP3 сохраняет аудиоданные в файл формата MP3. Он принимает аудиовход и экспортирует его в указанный выходной каталог с настраиваемыми параметрами имени файла и качества. Узел автоматически обрабатывает присвоение имен файлам и преобразование формата для создания воспроизводимого MP3-файла.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | Да | - | Аудиоданные для сохранения в формате MP3 |
| `filename_prefix` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "audio/ComfyUI") |
| `quality` | STRING | Нет | "V0"<br>"128k"<br>"320k" | Настройка качества аудио для MP3-файла (по умолчанию: "V0") |
| `prompt` | PROMPT | Нет | - | Внутренние данные промпта (автоматически предоставляются системой) |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Дополнительная PNG-информация (автоматически предоставляется системой) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| *Нет* | - | Этот узел не возвращает выходных данных, но сохраняет аудиофайл в выходной каталог |
