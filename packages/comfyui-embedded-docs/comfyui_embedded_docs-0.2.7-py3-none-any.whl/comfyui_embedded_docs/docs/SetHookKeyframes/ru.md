Узел Set Hook Keyframes позволяет применять планирование по ключевым кадрам к существующим группам хуков. Он принимает группу хуков и опционально применяет информацию о времени ключевых кадров для управления моментом выполнения различных хуков в процессе генерации. При предоставлении ключевых кадров узел клонирует группу хуков и устанавливает временные параметры ключевых кадров для всех хуков внутри группы.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `хуки` | HOOKS | Да | - | Группа хуков, к которой будет применено планирование по ключевым кадрам |
| `ключевые_кадры_хука` | HOOK_KEYFRAMES | Нет | - | Опциональная группа ключевых кадров, содержащая информацию о времени выполнения хуков |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `хуки` | HOOKS | Модифицированная группа хуков с примененным планированием по ключевым кадрам (клонированная, если были предоставлены ключевые кадры) |
