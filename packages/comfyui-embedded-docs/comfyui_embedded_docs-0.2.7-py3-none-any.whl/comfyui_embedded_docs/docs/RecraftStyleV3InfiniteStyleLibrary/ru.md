Этот узел позволяет выбрать стиль из библиотеки бесконечных стилей Recraft с использованием предустановленного UUID. Он получает информацию о стиле на основе предоставленного идентификатора и возвращает её для использования в других узлах Recraft.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `style_id` | STRING | Да | Любой допустимый UUID | UUID стиля из библиотеки бесконечных стилей. |

**Примечание:** Входной параметр `style_id` не может быть пустым. Если предоставлена пустая строка, узел вызовет исключение.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `recraft_style` | STYLEV3 | Выбранный объект стиля из библиотеки бесконечных стилей Recraft |
