Combine Hooks [2] 노드는 두 개의 훅 그룹을 단일 결합 훅 그룹으로 병합합니다. 두 개의 선택적 훅 입력을 받아 ComfyUI의 훅 결합 기능을 사용하여 결합합니다. 이를 통해 여러 훅 구성을 통합하여 효율적인 처리를 할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 타입 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `hooks_A` | HOOKS | 선택적 | 없음 | - | 결합할 첫 번째 훅 그룹 |
| `hooks_B` | HOOKS | 선택적 | 없음 | - | 결합할 두 번째 훅 그룹 |

**참고:** 두 입력 모두 선택적이지만, 노드가 기능하려면 최소한 하나의 훅 그룹이 제공되어야 합니다. 하나의 훅 그룹만 제공된 경우 변경 없이 반환됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `hooks` | HOOKS | 두 입력 그룹의 모든 훅을 포함하는 결합된 훅 그룹 |
