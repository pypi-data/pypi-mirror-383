ModelSaveノードは、学習済みまたは修正されたモデルをコンピュータのストレージに保存します。モデルを入力として受け取り、指定したファイル名でファイルに書き込みます。これにより、作業内容を保存し、将来のプロジェクトでモデルを再利用することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `モデル` | MODEL | はい | - | ディスクに保存するモデル |
| `ファイル名プレフィックス` | STRING | はい | - | 保存するモデルファイルのファイル名とパスのプレフィックス（デフォルト: "diffusion_models/ComfyUI"） |
| `prompt` | PROMPT | いいえ | - | ワークフローのプロンプト情報（自動的に提供されます） |
| `extra_pnginfo` | EXTRA_PNGINFO | いいえ | - | 追加のワークフローメタデータ（自動的に提供されます） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| *なし* | - | このノードは出力値を返しません |
