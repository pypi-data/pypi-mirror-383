Узел LatentOperationTonemapReinhard применяет тональное отображение Рейнхарда к латентным векторам. Этот метод нормализует латентные векторы и регулирует их величину с использованием статистического подхода на основе среднего значения и стандартного отклонения, при этом интенсивность контролируется параметром множителя.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `множитель` | FLOAT | Нет | 0.0 до 100.0 | Управляет интенсивностью эффекта тонального отображения (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `operation` | LATENT_OPERATION | Возвращает операцию тонального отображения, которая может быть применена к латентным векторам |
