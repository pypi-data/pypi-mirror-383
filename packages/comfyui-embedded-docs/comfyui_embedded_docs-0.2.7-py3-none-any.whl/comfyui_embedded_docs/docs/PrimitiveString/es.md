El nodo String proporciona una forma sencilla de introducir y transmitir datos de texto en tu flujo de trabajo. Toma una cadena de texto como entrada y produce la misma cadena sin cambios, lo que lo hace útil para proporcionar entradas de texto a otros nodos que requieren parámetros de cadena.

## Entradas

| Parámetro | Tipo de Datos | Requerido | Rango | Descripción |
|-----------|---------------|-----------|-------|-------------|
| `valor` | STRING | Sí | Cualquier texto | La cadena de texto que se transmitirá a través del nodo |

## Salidas

| Nombre de Salida | Tipo de Datos | Descripción |
|------------------|---------------|-------------|
| `output` | STRING | La misma cadena de texto que se proporcionó como entrada |
