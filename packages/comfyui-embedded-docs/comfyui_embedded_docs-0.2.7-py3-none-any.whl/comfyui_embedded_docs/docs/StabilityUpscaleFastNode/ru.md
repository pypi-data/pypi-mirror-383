Быстро увеличивает изображение до 4x от исходного размера через вызов API Stability. Этот узел специально предназначен для апскейла изображений низкого качества или сжатых изображений путем их отправки в сервис быстрого увеличения разрешения от Stability AI.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для увеличения разрешения |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Увеличенное изображение, возвращенное из API Stability AI |
