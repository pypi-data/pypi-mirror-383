Узел CLIP Text Encode for Lumina2 кодирует системный промпт и пользовательский промпт с помощью модели CLIP в эмбеддинг, который может направлять диффузионную модель для генерации конкретных изображений. Он объединяет предопределённый системный промпт с вашим пользовательским текстовым промптом и обрабатывает их через модель CLIP для создания данных кондиционирования для генерации изображений.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `system_prompt` | STRING | COMBO | - | "superior", "alignment" | Lumina2 предоставляет два типа системных промптов: Superior (Превосходный): Вы — помощник, созданный для генерации превосходных изображений с высочайшей степенью соответствия изображения и текста на основе текстовых промптов или пользовательских промптов. Alignment (Соответствие): Вы — помощник, созданный для генерации высококачественных изображений с наивысшей степенью соответствия изображения и текста на основе текстовых промптов. |
| `user_prompt` | STRING | STRING | - | - | Текст для кодирования. |
| `clip` | CLIP | CLIP | - | - | Модель CLIP, используемая для кодирования текста. |

**Примечание:** Входной параметр `clip` является обязательным и не может быть None. Если входной параметр clip недействителен, узел выдаст ошибку, указывающую, что контрольная точка может не содержать действительную модель CLIP или текстовый кодировщик.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Кондиционирование, содержащее встроенный текст, используемый для направления диффузионной модели. |
