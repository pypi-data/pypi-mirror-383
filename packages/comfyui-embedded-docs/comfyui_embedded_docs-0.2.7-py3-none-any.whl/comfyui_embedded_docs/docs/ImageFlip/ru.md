Узел ImageFlip переворачивает изображения по различным осям. Он может переворачивать изображения вертикально по оси x или горизонтально по оси y. Узел использует операции torch.flip для выполнения переворота в зависимости от выбранного метода.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение для переворота |
| `flip_method` | STRING | Да | "x-axis: vertically"<br>"y-axis: horizontally" | Направление переворота для применения |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Перевёрнутое выходное изображение |
