CheckpointLoader 노드는 사전 훈련된 모델 체크포인트와 해당 구성 파일을 로드합니다. 구성 파일과 체크포인트 파일을 입력으로 받아 워크플로우에서 사용할 수 있도록 메인 모델, CLIP 모델, VAE 모델을 포함한 로드된 모델 구성 요소를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 타입 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `설정 이름` | STRING | COMBO | - | 사용 가능한 구성 파일 | 모델 아키텍처와 설정을 정의하는 구성 파일 |
| `체크포인트 파일명` | STRING | COMBO | - | 사용 가능한 체크포인트 파일 | 훈련된 모델 가중치와 매개변수를 포함하는 체크포인트 파일 |

**참고:** 이 노드는 구성 파일과 체크포인트 파일 모두가 선택되어야 합니다. 구성 파일은 로드되는 체크포인트 파일의 아키텍처와 일치해야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 추론 준비가 된 로드된 메인 모델 구성 요소 |
| `CLIP` | CLIP | 텍스트 인코딩을 위한 로드된 CLIP 모델 구성 요소 |
| `VAE` | VAE | 이미지 인코딩과 디코딩을 위한 로드된 VAE 모델 구성 요소 |

**중요 참고:** 이 노드는 더 이상 사용되지 않는 것으로 표시되었으며 향후 버전에서 제거될 수 있습니다. 새로운 워크플로우에는 대체 로딩 노드 사용을 고려해 주십시오.
