Узел ThresholdMask преобразует маску в бинарную маску путём применения порогового значения. Он сравнивает каждый пиксель входной маски с указанным пороговым значением и создаёт новую маску, в которой пиксели выше порога становятся 1 (белыми), а пиксели ниже или равные порогу становятся 0 (чёрными).

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `маска` | MASK | Да | - | Входная маска для обработки |
| `значение` | FLOAT | Да | 0.0 - 1.0 | Пороговое значение для бинаризации (по умолчанию: 0.5) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `маска` | MASK | Результирующая бинарная маска после применения порога |
