Сохраняет файлы SVG на диск. Этот узел принимает данные SVG в качестве входных данных и сохраняет их в вашу выходную директорию с возможностью встраивания дополнительных метаданных. Узел автоматически обрабатывает именование файлов с суффиксами-счётчиками и может встраивать информацию о промпте рабочего процесса непосредственно в файл SVG.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `svg` | SVG | Да | - | Данные SVG для сохранения на диск |
| `filename_prefix` | STRING | Да | - | Префикс для сохраняемого файла. Может включать информацию о форматировании, такую как `%date:yyyy-MM-dd%` или `%Empty Latent Image.width%`, для включения значений из узлов. (по умолчанию: "svg/ComfyUI") |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `ui` | DICT | Возвращает информацию о файле, включая имя файла, подпапку и тип для отображения в интерфейсе ComfyUI |

**Примечание:** Этот узел автоматически встраивает метаданные рабочего процесса (промпт и дополнительную PNG-информацию) в файл SVG, когда они доступны. Метаданные вставляются как раздел CDATA внутри элемента метаданных SVG.
