Узел StringContains проверяет, содержит ли заданная строка указанную подстроку. Он может выполнять эту проверку с учетом или без учета регистра, возвращая логический результат, указывающий, была ли подстрока найдена в основной строке.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | - | Основная текстовая строка, в которой выполняется поиск |
| `substring` | STRING | Да | - | Текст, который нужно найти в основной строке |
| `case_sensitive` | BOOLEAN | Нет | - | Определяет, должен ли поиск учитывать регистр (по умолчанию: true) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `contains` | BOOLEAN | Возвращает true, если подстрока найдена в строке, и false в противном случае |
