WanFunInpaintToVideoノードは、開始画像と終了画像の間をインペイントすることでビデオシーケンスを作成します。ポジティブおよびネガティブなコンディショニングと、オプションのフレーム画像を受け取り、ビデオの潜在表現を生成します。このノードは、設定可能な寸法と長さのパラメータでビデオ生成を処理します。
