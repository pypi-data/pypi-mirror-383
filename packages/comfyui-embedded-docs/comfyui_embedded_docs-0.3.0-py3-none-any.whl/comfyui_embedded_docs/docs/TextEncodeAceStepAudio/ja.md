TextEncodeAceStepAudioノードは、タグと歌詞をトークンに結合し、調整可能な歌詞強度でエンコードすることで、オーディオコンディショニング用のテキスト入力を処理します。CLIPモデルとテキスト記述、歌詞を受け取り、それらを一緒にトークン化し、オーディオ生成タスクに適したコンディショニングデータを生成します。このノードは、最終出力への影響を制御する強度パラメータを通じて、歌詞の影響を微調整することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | はい | - | トークン化とエンコードに使用されるCLIPモデル |
| `tags` | STRING | はい | - | オーディオコンディショニング用のテキストタグまたは記述（複数行入力と動的プロンプトをサポート） |
| `lyrics` | STRING | はい | - | オーディオコンディショニング用の歌詞テキスト（複数行入力と動的プロンプトをサポート） |
| `lyrics_strength` | FLOAT | いいえ | 0.0 - 10.0 | コンディショニング出力に対する歌詞の影響の強度を制御します（デフォルト: 1.0、ステップ: 0.01） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | 処理されたテキストトークンと適用された歌詞強度を含む、エンコードされたコンディショニングデータ |
