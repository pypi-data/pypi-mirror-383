Узел CLIPMergeSubtract выполняет слияние моделей путем вычитания весов одной CLIP-модели из другой. Он создает новую CLIP-модель путем клонирования первой модели и последующего вычитания ключевых патчей из второй модели с регулируемым множителем для управления силой вычитания. Это позволяет выполнять тонкую настройку смешивания моделей путем удаления определенных характеристик из базовой модели.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip1` | CLIP | Да | - | Базовая CLIP-модель, которая будет клонирована и изменена |
| `clip2` | CLIP | Да | - | CLIP-модель, чьи ключевые патчи будут вычтены из базовой модели |
| `множитель` | FLOAT | Да | -10.0 до 10.0 | Управляет силой операции вычитания (по умолчанию: 1.0) |

**Примечание:** Узел исключает параметры `.position_ids` и `.logit_scale` из операции вычитания, независимо от значения множителя.

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `clip` | CLIP | Результирующая CLIP-модель после вычитания весов второй модели из первой |
