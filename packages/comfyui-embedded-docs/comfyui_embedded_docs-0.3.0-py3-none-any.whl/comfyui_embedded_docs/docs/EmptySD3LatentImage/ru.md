Узел EmptySD3LatentImage создает пустой латентный тензор изображения, специально отформатированный для моделей Stable Diffusion 3. Он генерирует тензор, заполненный нулями, который имеет правильные размеры и структуру, ожидаемые конвейерами SD3. Обычно используется в качестве отправной точки для рабочих процессов генерации изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `ширина` | INT | Да | от 16 до MAX_RESOLUTION (шаг: 16) | Ширина выходного латентного изображения в пикселях (по умолчанию: 1024) |
| `высота` | INT | Да | от 16 до MAX_RESOLUTION (шаг: 16) | Высота выходного латентного изображения в пикселях (по умолчанию: 1024) |
| `размер_пакета` | INT | Да | от 1 до 4096 | Количество латентных изображений для генерации в пакете (по умолчанию: 1) |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Латентный тензор, содержащий пустые сэмплы с размерами, совместимыми с SD3 |
