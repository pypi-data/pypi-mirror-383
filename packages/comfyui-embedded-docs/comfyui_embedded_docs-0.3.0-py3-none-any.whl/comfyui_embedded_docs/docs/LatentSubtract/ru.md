Узел LatentSubtract предназначен для вычитания одного латентного представления из другого. Эта операция может использоваться для манипуляции или модификации характеристик выходных данных генеративных моделей путём эффективного удаления признаков или атрибутов, представленных в одном латентном пространстве, из другого.

## Входы

| Параметр    | Тип данных | Описание |
|-------------|------------|----------|
| `образцы1`  | `LATENT`   | Первый набор латентных сэмплов, из которого производится вычитание. Служит основой для операции вычитания. |
| `образцы2`  | `LATENT`   | Второй набор латентных сэмплов, который будет вычитаться из первого набора. Эта операция может изменять выходные данные генеративной модели путём удаления атрибутов или признаков. |

## Выходы

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `latent` | `LATENT`   | Результат вычитания второго набора латентных сэмплов из первого. Это модифицированное латентное представление может быть использовано для последующих генеративных задач. |
