EmptyMochiLatentVideoノードは、指定された次元で空の潜在動画テンソルを作成します。このノードは、動画生成ワークフローの開始点として使用できるゼロ埋めされた潜在表現を生成します。潜在動画テンソルの幅、高さ、長さ、バッチサイズを定義することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `幅` | INT | はい | 16 から MAX_RESOLUTION | 潜在動画の幅（ピクセル単位）（デフォルト: 848、16で割り切れる必要があります） |
| `高さ` | INT | はい | 16 から MAX_RESOLUTION | 潜在動画の高さ（ピクセル単位）（デフォルト: 480、16で割り切れる必要があります） |
| `長さ` | INT | はい | 7 から MAX_RESOLUTION | 潜在動画のフレーム数（デフォルト: 25） |
| `バッチサイズ` | INT | いいえ | 1 から 4096 | バッチで生成する潜在動画の数（デフォルト: 1） |

**注記:** 実際の潜在次元は幅/8および高さ/8として計算され、時間次元は((length - 1) // 6) + 1として計算されます。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `samples` | LATENT | 指定された次元を持つ空の潜在動画テンソル（すべてゼロで埋められています） |
