Узел ModelMergeSDXL позволяет объединить две модели SDXL, регулируя влияние каждой модели на различные части архитектуры. Вы можете управлять тем, насколько каждая модель влияет на временные эмбеддинги, эмбеддинги меток и различные блоки в структуре модели. Это создаёт гибридную модель, сочетающую характеристики обеих исходных моделей.
