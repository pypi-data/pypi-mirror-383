Узел ModelSamplingSD3 применяет параметры сэмплинга Stable Diffusion 3 к модели. Он изменяет поведение сэмплинга модели путем настройки параметра смещения, который управляет характеристиками распределения сэмплинга. Узел создает модифицированную копию входной модели с примененной указанной конфигурацией сэмплинга.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Входная модель, к которой применяются параметры сэмплинга SD3 |
| `сдвиг` | FLOAT | Да | 0.0 - 100.0 | Управляет параметром смещения сэмплинга (по умолчанию: 3.0) |

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с примененными параметрами сэмплинга SD3 |
