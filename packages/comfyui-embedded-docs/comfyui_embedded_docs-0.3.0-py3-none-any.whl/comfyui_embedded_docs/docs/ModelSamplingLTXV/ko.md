ModelSamplingLTXV 노드는 토큰 수를 기반으로 모델에 고급 샘플링 매개변수를 적용합니다. 입력 잠재 변수의 토큰 수에 따라 기본 시프트 값과 최대 시프트 값 사이의 선형 보간을 사용하여 시프트 값을 계산합니다. 그런 다음 특수화된 모델 샘플링 구성을 생성하고 입력 모델에 적용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | 샘플링 매개변수를 적용할 입력 모델 |
| `최대 시프트` | FLOAT | 아니오 | 0.0 ~ 100.0 | 계산에 사용되는 최대 시프트 값 (기본값: 2.05) |
| `기반 시프트` | FLOAT | 아니오 | 0.0 ~ 100.0 | 계산에 사용되는 기본 시프트 값 (기본값: 0.95) |
| `잠재 비디오` | LATENT | 아니오 | - | 시프트 계산을 위한 토큰 수 결정에 사용되는 선택적 잠재 변수 입력 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | 적용된 샘플링 매개변수가 포함된 수정된 모델 |
