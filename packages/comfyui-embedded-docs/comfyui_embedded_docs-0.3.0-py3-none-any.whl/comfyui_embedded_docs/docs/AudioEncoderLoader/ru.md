Узел AudioEncoderLoader загружает модели аудио-энкодеров из доступных файлов аудио-энкодеров. Он принимает имя файла аудио-энкодера в качестве входных данных и возвращает загруженную модель аудио-энкодера, которая может использоваться для задач обработки аудио в вашем рабочем процессе.

## Входные данные

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `audio_encoder_name` | STRING | COMBO | - | Доступные файлы аудио-энкодеров | Выбирает, какой файл модели аудио-энкодера загрузить из вашей папки audio_encoders |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `audio_encoder` | AUDIO_ENCODER | Возвращает загруженную модель аудио-энкодера для использования в рабочих процессах обработки аудио |
