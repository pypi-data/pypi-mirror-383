## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | 수직 부정 가이던스를 적용할 모델 |
| `빈 조건` | CONDITIONING | 예 | - | 부정 가이던스 계산에 사용되는 빈 조건화 |
| `부정 스케일` | FLOAT | 아니오 | 0.0 - 100.0 | 부정 가이던스의 스케일링 계수 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | 수직 부정 가이던스가 적용된 수정된 모델 |

**참고**: 이 노드는 더 이상 사용되지 않으며(deprecated) PerpNegGuider 노드로 대체되었습니다. 실험적으로 표시되어 있으며 프로덕션 워크플로에서 사용해서는 안 됩니다.
