Генерирует SVG синхронно на основе промпта и разрешения. Этот узел создает векторные иллюстрации, отправляя текстовые промпты в API Recraft и возвращает сгенерированное SVG-содержимое.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Промпт для генерации изображения. (по умолчанию: "") |
| `substyle` | COMBO | Да | Доступно несколько опций | Конкретный стиль иллюстрации, используемый для генерации. Опции определяются доступными подстилями векторных иллюстраций в RecraftStyleV3. |
| `size` | COMBO | Да | Доступно несколько опций | Размер генерируемого изображения. (по умолчанию: 1024x1024) |
| `n` | INT | Да | 1-6 | Количество генерируемых изображений. (по умолчанию: 1, мин.: 1, макс.: 6) |
| `seed` | INT | Да | 0-18446744073709551615 | Сид для определения, должен ли узел перезапускаться; фактические результаты недетерминированы независимо от сида. (по умолчанию: 0, мин.: 0, макс.: 18446744073709551615) |
| `negative_prompt` | STRING | Нет | - | Необязательное текстовое описание нежелательных элементов на изображении. (по умолчанию: "") |
| `recraft_controls` | CONTROLS | Нет | - | Необязательные дополнительные элементы управления генерацией через узел Recraft Controls. |

**Примечание:** Параметр `seed` управляет только тем, когда узел перезапускается, но не делает результаты генерации детерминированными.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `SVG` | SVG | Сгенерированная векторная иллюстрация в формате SVG |
