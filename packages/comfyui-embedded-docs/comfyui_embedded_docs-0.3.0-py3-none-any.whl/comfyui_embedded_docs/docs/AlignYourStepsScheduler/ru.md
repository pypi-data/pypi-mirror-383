Узел AlignYourStepsScheduler генерирует значения сигма для процесса удаления шума на основе различных типов моделей. Он вычисляет подходящие уровни шума для каждого шага процесса сэмплирования и корректирует общее количество шагов в соответствии с параметром деннойза. Это помогает согласовать шаги сэмплирования с конкретными требованиями различных моделей диффузии.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `тип_модели` | STRING | COMBO | - | SD1, SDXL, SVD | Определяет тип модели, используемый для расчета сигма |
| `шаги` | INT | INT | 10 | 1-10000 | Общее количество шагов сэмплирования для генерации |
| `шумоподавление` | FLOAT | FLOAT | 1.0 | 0.0-1.0 | Определяет степень удаления шума с изображения, где 1.0 использует все шаги, а меньшие значения используют меньше шагов |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | Возвращает рассчитанные значения сигма для процесса удаления шума |
