Узел ImageOnlyCheckpointSave сохраняет файл контрольной точки, содержащий модель, энкодер CLIP vision и VAE. Он создает файл safetensors с указанным префиксом имени и сохраняет его в выходном каталоге. Этот узел специально предназначен для совместного сохранения компонентов модели, связанных с изображениями, в одном файле контрольной точки.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель для сохранения в контрольной точке |
| `clip_vision` | CLIP_VISION | Да | - | Энкодер CLIP vision для сохранения в контрольной точке |
| `vae` | VAE | Да | - | VAE (Вариационный автоэнкодер) для сохранения в контрольной точке |
| `префикс_названия_файла` | STRING | Да | - | Префикс для имени выходного файла (по умолчанию: "checkpoints/ComfyUI") |
| `prompt` | PROMPT | Нет | - | Скрытый параметр для данных промпта рабочего процесса |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Скрытый параметр для дополнительных метаданных PNG |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| - | - | Этот узел не возвращает никаких выходных данных |
