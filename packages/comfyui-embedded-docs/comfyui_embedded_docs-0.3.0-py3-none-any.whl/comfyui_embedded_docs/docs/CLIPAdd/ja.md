CLIPAddノードは、2つのCLIPモデルのキーパッチを統合することで結合します。最初のCLIPモデルのコピーを作成し、2番目のモデルのキーパッチの大部分を追加します（位置IDとロジットスケールパラメータは除く）。これにより、最初のモデルの構造を保持しながら、異なるCLIPモデルの特徴をブレンドすることができます。

## 入力

| パラメータ | データ型 | 入力タイプ | デフォルト | 範囲 | 説明 |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | 必須 | - | - | マージのベースとして使用される主要なCLIPモデル |
| `clip2` | CLIP | 必須 | - | - | 追加するパッチを提供する2番目のCLIPモデル |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `CLIP` | CLIP | 両方の入力モデルの特徴を組み合わせた統合CLIPモデルを返します |
