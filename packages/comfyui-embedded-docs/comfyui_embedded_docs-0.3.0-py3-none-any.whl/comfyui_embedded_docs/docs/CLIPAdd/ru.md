Узел CLIPAdd объединяет две модели CLIP путем слияния их ключевых патчей. Он создает копию первой модели CLIP, а затем добавляет большинство ключевых патчей из второй модели, исключая параметры идентификаторов позиций и масштаба логитов. Это позволяет объединять функции из различных моделей CLIP, сохраняя структуру первой модели.

## Входы

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | Обязательный | - | - | Основная модель CLIP, которая будет использоваться как база для слияния |
| `clip2` | CLIP | Обязательный | - | - | Вторичная модель CLIP, которая предоставляет дополнительные патчи для добавления |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CLIP` | CLIP | Возвращает объединенную модель CLIP, сочетающую функции обеих входных моделей |
