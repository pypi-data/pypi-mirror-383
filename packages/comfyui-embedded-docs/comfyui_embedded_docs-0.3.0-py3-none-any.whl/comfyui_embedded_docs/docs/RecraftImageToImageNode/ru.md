Этот узел изменяет существующее изображение на основе текстового запроса и параметра силы воздействия. Он использует API Recraft для преобразования входного изображения в соответствии с предоставленным описанием, сохраняя при этом некоторое сходство с исходным изображением в зависимости от настройки силы воздействия.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для модификации |
| `запрос` | STRING | Да | - | Текст-запрос для генерации изображения (по умолчанию: "") |
| `n` | INT | Да | 1-6 | Количество генерируемых изображений (по умолчанию: 1) |
| `степень изменения` | FLOAT | Да | 0.0-1.0 | Определяет степень отличия от исходного изображения, должна находиться в диапазоне [0, 1], где 0 означает почти идентичное изображение, а 1 — минимальное сходство (по умолчанию: 0.5) |
| `seed` | INT | Да | 0-18446744073709551615 | Сид для определения необходимости повторного выполнения узла; фактические результаты не являются детерминированными независимо от сида (по умолчанию: 0) |
| `recraft_style` | STYLEV3 | Нет | - | Опциональный выбор стиля для генерации изображения |
| `негативный запрос` | STRING | Нет | - | Опциональное текстовое описание нежелательных элементов на изображении (по умолчанию: "") |
| `recraft_controls` | CONTROLS | Нет | - | Опциональные дополнительные элементы управления генерацией через узел Recraft Controls |

**Примечание:** Параметр `seed` только запускает повторное выполнение узла, но не гарантирует детерминированных результатов. Параметр силы воздействия внутренне округляется до 2 знаков после запятой.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Сгенерированное изображение(я) на основе входного изображения и запроса |
