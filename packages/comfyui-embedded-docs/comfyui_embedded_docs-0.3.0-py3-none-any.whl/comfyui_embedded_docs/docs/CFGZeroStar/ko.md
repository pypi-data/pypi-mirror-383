CFGZeroStar 노드는 확산 모델에 특화된 가이던스 스케일링 기법을 적용합니다. 이 노드는 조건부 예측과 무조건부 예측 간의 차이를 기반으로 최적화된 스케일 인자를 계산하여 classifier-free 가이던스 과정을 수정합니다. 이 접근 방식은 모델 안정성을 유지하면서 생성 과정에 대한 향상된 제어를 제공하기 위해 최종 출력을 조정합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 타입 | 기본값 | 범위 | 설명 |
|----------|-------------|------------|---------|-------|-------------|
| `모델` | MODEL | 필수 | - | - | CFGZeroStar 가이던스 스케일링 기법이 적용될 확산 모델 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `patched_model` | MODEL | CFGZeroStar 가이던스 스케일링이 적용된 수정된 모델 |
