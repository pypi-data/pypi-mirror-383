Узел ByteDance Image Edit позволяет изменять изображения с использованием моделей искусственного интеллекта ByteDance через API. Вы предоставляете исходное изображение и текстовый запрос, описывающий желаемые изменения, а узел обрабатывает изображение в соответствии с вашими инструкциями. Узел автоматически управляет взаимодействием с API и возвращает отредактированное изображение.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `model` | MODEL | COMBO | seededit_3 | Image2ImageModelName options | Название модели |
| `image` | IMAGE | IMAGE | - | - | Базовое изображение для редактирования |
| `prompt` | STRING | STRING | "" | - | Инструкция для редактирования изображения |
| `seed` | INT | INT | 0 | 0-2147483647 | Сид (начальное значение) для генерации |
| `guidance_scale` | FLOAT | FLOAT | 5.5 | 1.0-10.0 | Более высокое значение заставляет изображение точнее следовать запросу |
| `watermark` | BOOLEAN | BOOLEAN | True | - | Добавлять ли водяной знак "Сгенерировано ИИ" на изображение |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Отредактированное изображение, возвращённое из API ByteDance |
