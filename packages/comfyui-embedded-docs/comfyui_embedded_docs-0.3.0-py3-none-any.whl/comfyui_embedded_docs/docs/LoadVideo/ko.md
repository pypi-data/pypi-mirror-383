Load Video 노드는 입력 디렉토리에서 비디오 파일을 불러와 워크플로우에서 처리할 수 있도록 합니다. 지정된 입력 폴더에서 비디오 파일을 읽어들이고, 다른 비디오 처리 노드에 연결할 수 있는 비디오 데이터로 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `파일` | STRING | 예 | 여러 옵션 사용 가능 | 입력 디렉토리에서 불러올 비디오 파일 |

**참고:** `file` 매개변수에 사용 가능한 옵션들은 입력 디렉토리에 있는 비디오 파일들로부터 동적으로 채워집니다. 지원되는 콘텐츠 타입을 가진 비디오 파일만 표시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 다른 비디오 처리 노드로 전달할 수 있는 불러온 비디오 데이터 |
