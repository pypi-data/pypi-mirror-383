Узел Load Video загружает видеофайлы из входной директории и делает их доступными для обработки в рабочем процессе. Он считывает видеофайлы из указанной входной папки и выводит их в виде видеоданных, которые могут быть подключены к другим узлам обработки видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `файл` | STRING | Да | Доступно несколько вариантов | Видеофайл для загрузки из входной директории |

**Примечание:** Доступные варианты для параметра `file` динамически заполняются из видеофайлов, присутствующих во входной директории. Отображаются только видеофайлы с поддерживаемыми типами контента.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Загруженные видеоданные, которые могут быть переданы другим узлам обработки видео |
