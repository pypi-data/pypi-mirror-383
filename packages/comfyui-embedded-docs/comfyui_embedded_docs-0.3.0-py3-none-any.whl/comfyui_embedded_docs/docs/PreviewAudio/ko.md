PreviewAudio 노드는 인터페이스에 표시할 수 있는 임시 오디오 미리보기 파일을 생성합니다. 이 노드는 SaveAudio를 상속하지만, 임시 디렉터리에 무작위 파일 이름 접두사를 사용하여 파일을 저장합니다. 이를 통해 사용자는 영구적인 파일을 생성하지 않고도 오디오 출력을 빠르게 미리볼 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `오디오` | AUDIO | 예 | - | 미리볼 오디오 데이터 |
| `prompt` | PROMPT | 아니오 | - | 내부 사용을 위한 숨겨진 매개변수 |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니오 | - | 내부 사용을 위한 숨겨진 매개변수 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `ui` | UI | 인터페이스에 오디오 미리보기를 표시합니다 |
