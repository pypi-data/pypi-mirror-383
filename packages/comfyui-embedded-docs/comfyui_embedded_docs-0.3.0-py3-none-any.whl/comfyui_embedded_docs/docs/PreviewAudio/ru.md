Узел PreviewAudio создает временный файл предварительного просмотра аудио, который может отображаться в интерфейсе. Он наследуется от SaveAudio, но сохраняет файлы во временный каталог со случайным префиксом имени. Это позволяет пользователям быстро просматривать аудиовыходы без создания постоянных файлов.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `аудио` | AUDIO | Да | - | Аудиоданные для предварительного просмотра |
| `prompt` | PROMPT | Нет | - | Скрытый параметр для внутреннего использования |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Скрытый параметр для внутреннего использования |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `ui` | UI | Отображает предварительный просмотр аудио в интерфейсе |
