WanCameraEmbeddingノードは、カメラモーションパラメータに基づいてプラッカー埋め込みを使用してカメラ軌道の埋め込みを生成します。さまざまなカメラの動きをシミュレートする一連のカメラポーズを作成し、ビデオ生成パイプラインに適した埋め込みテンソルに変換します。
