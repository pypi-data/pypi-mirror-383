Узел LatentFlip предназначен для манипуляции латентными представлениями путём их отражения по вертикали или горизонтали. Эта операция позволяет преобразовывать латентное пространство, потенциально раскрывая новые вариации или перспективы в данных.

## Входы

| Параметр     | Тип данных    | Описание |
|--------------|---------------|-------------|
| `образцы`    | `LATENT`      | Параметр 'samples' представляет латентные представления, которые будут отражены. Операция отражения изменяет эти представления либо по вертикали, либо по горизонтали, в зависимости от параметра 'flip_method', таким образом преобразуя данные в латентном пространстве. |
| `метод_отражения`| COMBO[STRING] | Параметр 'flip_method' определяет ось, вдоль которой будут отражены латентные сэмплы. Может принимать значения 'x-axis: vertically' (по оси X: вертикально) или 'y-axis: horizontally' (по оси Y: горизонтально), определяя направление отражения и, следовательно, характер преобразования, применяемого к латентным представлениям. |

## Выходы

| Параметр | Тип данных | Описание |
|----------|------------|-------------|
| `latent` | `LATENT`   | Выходные данные представляют собой модифицированную версию входных латентных представлений, которые были отражены в соответствии с указанным методом. Это преобразование может внести новые вариации в латентном пространстве. |
