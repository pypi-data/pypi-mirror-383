PrimitiveIntノードは、ワークフロー内で整数値を扱うためのシンプルな方法を提供します。整数入力を受け取り、同じ値を出力するため、ノード間で整数パラメータを渡したり、他の操作のために特定の数値を設定したりするのに便利です。

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `値` | INT | はい | -9223372036854775807 から 9223372036854775807 | 出力する整数値 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `output` | INT | 変更されずに通過した入力整数値 |
