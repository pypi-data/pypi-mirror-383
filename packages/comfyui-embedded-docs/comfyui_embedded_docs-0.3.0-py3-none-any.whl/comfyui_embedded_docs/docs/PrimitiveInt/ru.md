Узел PrimitiveInt предоставляет простой способ работы с целочисленными значениями в вашем рабочем процессе. Он принимает целочисленный вход и выводит то же значение, что делает его полезным для передачи целочисленных параметров между узлами или установки конкретных числовых значений для других операций.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `значение` | INT | Да | -9223372036854775807 до 9223372036854775807 | Целочисленное значение для вывода |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | INT | Входное целочисленное значение, переданное без изменений |
