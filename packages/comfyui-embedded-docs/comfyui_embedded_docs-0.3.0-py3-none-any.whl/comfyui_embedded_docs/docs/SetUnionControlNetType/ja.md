SetUnionControlNetTypeノードは、コンディショニングに使用するコントロールネットワークのタイプを指定することができます。既存のコントロールネットワークを受け取り、選択に基づいてそのコントロールタイプを設定し、指定されたタイプ設定で変更されたコントロールネットワークのコピーを作成します。

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `control_net` | CONTROL_NET | はい | - | 新しいタイプ設定で変更するコントロールネットワーク |
| `タイプ` | STRING | はい | `"auto"`<br>利用可能なすべてのUNION_CONTROLNET_TYPESキー | 適用するコントロールネットワークのタイプ。自動タイプ検出には"auto"を使用するか、利用可能なオプションから特定のコントロールネットワークタイプを選択します |
