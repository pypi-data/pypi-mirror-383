Wan Image to Imageノードは、1つまたは2つの入力画像とテキストプロンプトから画像を生成します。入力画像を提供された説明に基づいて変換し、元の入力画像のアスペクト比を維持した新しい画像を作成します。出力画像は入力サイズに関係なく、160万ピクセルで固定されます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | はい | "wan2.5-i2i-preview" | 使用するモデル（デフォルト: "wan2.5-i2i-preview"）。 |
| `image` | IMAGE | はい | - | 単一画像編集または複数画像融合、最大2画像まで。 |
| `prompt` | STRING | はい | - | 要素と視覚的特徴を説明するためのプロンプト、英語/中国語をサポート（デフォルト: 空）。 |
| `negative_prompt` | STRING | いいえ | - | 避けるべき要素を導くためのネガティブテキストプロンプト（デフォルト: 空）。 |
| `seed` | INT | いいえ | 0 から 2147483647 | 生成に使用するシード値（デフォルト: 0）。 |
| `watermark` | BOOLEAN | いいえ | - | 結果に「AI生成」の透かしを追加するかどうか（デフォルト: true）。 |

**注意:** このノードは正確に1つまたは2つの入力画像を受け付けます。2つ以上の画像を提供した場合、または画像がまったく提供されなかった場合、ノードはエラーを返します。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `image` | IMAGE | 入力画像とテキストプロンプトに基づいて生成された画像。 |
