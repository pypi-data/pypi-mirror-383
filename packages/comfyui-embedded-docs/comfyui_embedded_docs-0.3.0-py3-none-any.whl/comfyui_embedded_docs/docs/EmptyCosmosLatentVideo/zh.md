| 参数 | 数据类型 | 必需 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `宽度` | INT | 是 | 16 至 MAX_RESOLUTION | 潜视频的宽度（单位：像素），默认值：1280，必须能被16整除 |
| `高度` | INT | 是 | 16 至 MAX_RESOLUTION | 潜视频的高度（单位：像素），默认值：704，必须能被16整除 |
| `长度` | INT | 是 | 1 至 MAX_RESOLUTION | 潜视频的帧数，默认值：121 |
| `批量大小` | INT | 否 | 1 至 4096 | 单批次生成的潜视频数量，默认值：1 |
