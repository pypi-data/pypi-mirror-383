Узел EmptyCosmosLatentVideo создает пустой латентный видеотензор с заданными размерами. Он генерирует латентное представление, заполненное нулями, которое может использоваться в качестве отправной точки для рабочих процессов генерации видео, с настраиваемыми параметрами ширины, высоты, длины и размера пакета.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `ширина` | INT | Да | 16 до MAX_RESOLUTION | Ширина латентного видео в пикселях (по умолчанию: 1280, должна быть кратна 16) |
| `высота` | INT | Да | 16 до MAX_RESOLUTION | Высота латентного видео в пикселях (по умолчанию: 704, должна быть кратна 16) |
| `длина` | INT | Да | 1 до MAX_RESOLUTION | Количество кадров в латентном видео (по умолчанию: 121) |
| `размер_пакета` | INT | Нет | 1 до 4096 | Количество латентных видео для генерации в пакете (по умолчанию: 1) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `samples` | LATENT | Сгенерированный пустой латентный видеотензор с нулевыми значениями |
