Pikadditions 노드를 사용하면 비디오에 어떤 객체나 이미지든 추가할 수 있습니다. 비디오를 업로드하고 추가하려는 내용을 지정하여 완벽하게 통합된 결과물을 생성합니다. 이 노드는 Pika API를 사용하여 이미지를 비디오에 자연스럽게 통합된 모습으로 삽입합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `비디오` | VIDEO | 예 | - | 이미지를 추가할 대상 비디오입니다. |
| `이미지` | IMAGE | 예 | - | 비디오에 추가할 이미지입니다. |
| `프롬프트 텍스트` | STRING | 예 | - | 비디오에 추가할 내용에 대한 텍스트 설명입니다. |
| `부정 프롬프트` | STRING | 예 | - | 비디오에서 피해야 할 내용에 대한 텍스트 설명입니다. |
| `시드` | INT | 예 | 0부터 4294967295까지 | 재현 가능한 결과를 위한 랜덤 시드 값입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 이미지가 삽입된 처리된 비디오입니다. |
