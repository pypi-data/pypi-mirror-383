Улучшает направленность на детализированную структуру, используя другой набор негативного CFG с пропущенными слоями. Эта универсальная версия SkipLayerGuidance может использоваться на каждой модели DiT и вдохновлена Perturbed Attention Guidance. Оригинальная экспериментальная реализация была создана для SD3.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, к которой применяется направленность с пропуском слоёв |
| `двойные_слои` | STRING | Да | - | Номера слоёв для двойных блоков, разделённые запятыми, которые нужно пропустить (по умолчанию: "7, 8, 9") |
| `одиночные_слои` | STRING | Да | - | Номера слоёв для одиночных блоков, разделённые запятыми, которые нужно пропустить (по умолчанию: "7, 8, 9") |
| `масштаб` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштаба направленности (по умолчанию: 3.0) |
| `начальный_процент` | FLOAT | Да | 0.0 - 1.0 | Начальный процент для применения направленности (по умолчанию: 0.01) |
| `конечный_процент` | FLOAT | Да | 0.0 - 1.0 | Конечный процент для применения направленности (по умолчанию: 0.15) |
| `масштаб_перемасштабирования` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштаба перемасштабирования (по умолчанию: 0.0) |

**Примечание:** Если оба параметра `double_layers` и `single_layers` пусты (не содержат номеров слоёв), узел возвращает исходную модель без применения какой-либо направленности.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с применённой направленностью с пропуском слоёв |
