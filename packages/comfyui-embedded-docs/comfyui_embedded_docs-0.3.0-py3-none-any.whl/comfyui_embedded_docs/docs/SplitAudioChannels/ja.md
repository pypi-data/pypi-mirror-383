SplitAudioChannelsノードは、ステレオ音声を個別の左チャンネルと右チャンネルに分離します。2チャンネルのステレオ音声入力を受け取り、左チャンネル用と右チャンネル用の2つの独立した音声ストリームを出力します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | はい | - | チャンネルに分離するステレオ音声入力 |

**注意:** 入力音声は正確に2チャンネル（ステレオ）である必要があります。入力音声が1チャンネルのみの場合、ノードはエラーを発生させます。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `left` | AUDIO | 分離された左チャンネルの音声 |
| `right` | AUDIO | 分離された右チャンネルの音声 |
