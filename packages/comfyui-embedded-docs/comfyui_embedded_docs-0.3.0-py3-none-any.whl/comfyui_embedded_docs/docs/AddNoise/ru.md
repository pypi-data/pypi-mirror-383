# AddNoise

Этот узел добавляет контролируемый шум к латентному изображению, используя заданные параметры шума и значения сигмы. Он обрабатывает входные данные через систему сэмплирования модели для применения масштабирования шума, соответствующего заданному диапазону сигмы.

## Как это работает

Узел принимает латентное изображение и применяет к нему шум на основе предоставленного генератора шума и значений сигмы. Сначала он проверяет, предоставлены ли какие-либо сигмы - если нет, возвращает исходное латентное изображение без изменений. Затем узел использует систему сэмплирования модели для обработки латентного изображения и применения масштабированного шума. Масштабирование шума определяется разницей между первым и последним значениями сигмы, когда предоставлено несколько сигм, или одиночным значением сигмы, когда доступно только одно. Пустые латентные изображения (содержащие только нули) не смещаются во время обработки. Конечный результат - новое латентное представление с примененным шумом, где любые значения NaN или бесконечности преобразуются в нули для стабильности.

## Входы

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `модель` | MODEL | Обязательный | - | - | Модель, содержащая параметры сэмплирования и функции обработки |
| `шум` | NOISE | Обязательный | - | - | Генератор шума, который создает базовый шаблон шума |
| `сигмы` | SIGMAS | Обязательный | - | - | Значения сигмы, управляющие интенсивностью масштабирования шума |
| `латентное_изображение` | LATENT | Обязательный | - | - | Входное латентное представление, к которому будет добавлен шум |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Модифицированное латентное представление с добавленным шумом |
