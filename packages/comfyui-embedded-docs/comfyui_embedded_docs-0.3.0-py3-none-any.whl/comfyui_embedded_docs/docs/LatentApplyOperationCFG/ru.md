Узел LatentApplyOperationCFG применяет латентную операцию для модификации процесса управления условиями в модели. Он работает путем перехвата выходных данных кондиционирования в процессе семплинга с классификатор-фри гайденсом (CFG) и применения указанной операции к латентным представлениям перед их использованием для генерации.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, к которой будет применена операция CFG |
| `операция` | LATENT_OPERATION | Да | - | Латентная операция, применяемая в процессе CFG семплинга |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с примененной операцией CFG к ее процессу семплинга |
