Узел Combine Hooks [8] объединяет до восьми различных групп хуков в одну комбинированную группу хуков. Он принимает несколько входных хуков и объединяет их с использованием функциональности комбинирования хуков ComfyUI. Это позволяет консолидировать несколько конфигураций хуков для оптимизированной обработки в сложных рабочих процессах.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `hooks_A` | HOOKS | optional | None | - | Первая группа хуков для объединения |
| `hooks_B` | HOOKS | optional | None | - | Вторая группа хуков для объединения |
| `hooks_C` | HOOKS | optional | None | - | Третья группа хуков для объединения |
| `hooks_D` | HOOKS | optional | None | - | Четвертая группа хуков для объединения |
| `hooks_E` | HOOKS | optional | None | - | Пятая группа хуков для объединения |
| `hooks_F` | HOOKS | optional | None | - | Шестая группа хуков для объединения |
| `hooks_G` | HOOKS | optional | None | - | Седьмая группа хуков для объединения |
| `hooks_H` | HOOKS | optional | None | - | Восьмая группа хуков для объединения |

**Примечание:** Все входные параметры являются необязательными. Узел объединит только те группы хуков, которые предоставлены, игнорируя любые пустые входы. Вы можете предоставить от одной до восьми групп хуков для объединения.

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | Единая комбинированная группа хуков, содержащая все предоставленные конфигурации хуков |
