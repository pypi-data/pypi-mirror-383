Узел SamplerEulerAncestralCFGPP создает специализированный сэмплер для генерации изображений с использованием метода Эйлера-Анцестрального с классификатор-фри гайдингом. Этот сэмплер сочетает методы анцестрального сэмплирования с управляющей кондиционированием для создания разнообразных вариаций изображений при сохранении связности. Он позволяет тонко настраивать процесс сэмплирования через параметры, управляющие шумом и корректировками размера шага.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `eta` | FLOAT | Да | 0.0 - 1.0 | Управляет размером шага во время сэмплирования, при этом более высокие значения приводят к более агрессивным обновлениям (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 10.0 | Регулирует количество шума, добавляемого в процессе сэмплирования (по умолчанию: 1.0) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Возвращает настроенный объект сэмплера, который может быть использован в конвейере генерации изображений |
