SaveWEBMノードは、一連の画像をWEBM動画ファイルとして保存します。複数の入力画像を受け取り、設定可能な品質設定とフレームレートでVP9またはAV1コーデックを使用して動画にエンコードします。生成された動画ファイルは、プロンプト情報を含むメタデータとともに出力ディレクトリに保存されます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `画像` | IMAGE | はい | - | 動画フレームとしてエンコードする入力画像のシーケンス |
| `ファイル名の接頭辞` | STRING | いいえ | - | 出力ファイル名のプレフィックス（デフォルト: "ComfyUI"） |
| `コーデック` | COMBO | はい | "vp9"<br>"av1" | エンコードに使用する動画コーデック |
| `fps` | FLOAT | いいえ | 0.01-1000.0 | 出力動画のフレームレート（デフォルト: 24.0） |
| `crf` | FLOAT | いいえ | 0-63.0 | 品質設定。crfが高いほどファイルサイズは小さいが品質は低く、crfが低いほど品質は高いがファイルサイズは大きくなります（デフォルト: 32.0） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `ui` | PREVIEW | 保存されたWEBMファイルを表示する動画プレビュー |
