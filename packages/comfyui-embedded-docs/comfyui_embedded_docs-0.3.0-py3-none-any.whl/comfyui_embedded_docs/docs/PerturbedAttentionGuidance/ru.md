Узел PerturbedAttentionGuidance применяет метод направленного возмущения внимания к диффузионной модели для повышения качества генерации. Он модифицирует механизм самовнимания модели во время семплирования, заменяя его упрощённой версией, которая фокусируется на проекциях значений. Эта техника помогает улучшить согласованность и качество генерируемых изображений за счёт корректировки процесса условного шумоподавления.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Диффузионная модель, к которой применяется метод направленного возмущения внимания |
| `масштаб` | FLOAT | Нет | 0.0 - 100.0 | Интенсивность эффекта направленного возмущения внимания (по умолчанию: 3.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с применённым методом направленного возмущения внимания |
