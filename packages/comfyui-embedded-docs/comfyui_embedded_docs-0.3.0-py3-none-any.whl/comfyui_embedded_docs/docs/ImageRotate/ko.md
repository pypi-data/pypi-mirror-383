ImageRotate 노드는 입력 이미지를 지정된 각도로 회전시킵니다. 회전 없음, 시계 방향 90도, 180도, 시계 방향 270도의 네 가지 회전 옵션을 지원합니다. 회전은 이미지 데이터 무결성을 유지하는 효율적인 텐서 연산을 사용하여 수행됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 회전시킬 입력 이미지 |
| `rotation` | STRING | 예 | "none"<br>"90 degrees"<br>"180 degrees"<br>"270 degrees" | 이미지에 적용할 회전 각도 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 회전된 출력 이미지 |
