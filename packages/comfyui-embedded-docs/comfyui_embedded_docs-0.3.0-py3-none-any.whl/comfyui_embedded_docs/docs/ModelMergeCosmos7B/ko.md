ModelMergeCosmos7B 노드는 두 개의 AI 모델을 특정 구성 요소의 가중치 혼합을 통해 결합합니다. 위치 임베딩, 트랜스포머 블록, 최종 레이어 등 각 부분에 대한 개별 가중치를 조정하여 모델의 다양한 부분이 어떻게 결합되는지 세밀하게 제어할 수 있습니다.
