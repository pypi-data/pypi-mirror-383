Создает видео синхронно на основе изображения, текстового описания и дополнительных параметров с использованием API MiniMax. Этот узел принимает изображение объекта и текстовое описание для создания видео с помощью сервиса генерации видео от MiniMax.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `subject` | IMAGE | Да | - | Изображение объекта, используемое в качестве ориентира для генерации видео |
| `prompt_text` | STRING | Да | - | Текстовое описание для управления процессом генерации видео (по умолчанию: пустая строка) |
| `model` | COMBO | Нет | "S2V-01"<br> | Модель, используемая для генерации видео (по умолчанию: "S2V-01") |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное начальное число (сид), используемое для создания шума (по умолчанию: 0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе входного изображения объекта и текстового описания |
