Узел LatentCrop предназначен для выполнения операций обрезки латентных представлений изображений. Он позволяет задавать размеры и позицию области обрезки, обеспечивая целенаправленные модификации латентного пространства.

## Входы

| Параметр  | Тип данных | Описание |
|-----------|------------|----------|
| `образцы` | `LATENT`   | Параметр 'samples' представляет латентные представления, которые необходимо обрезать. Он играет ключевую роль в определении данных, над которыми будет выполнена операция обрезки. |
| `ширина`   | `INT`      | Задает ширину области обрезки. Непосредственно влияет на размеры выходного латентного представления. |
| `высота`  | `INT`      | Задает высоту области обрезки, влияя на размер результирующего обрезанного латентного представления. |
| `x`       | `INT`      | Определяет начальную x-координату области обрезки, влияя на позицию обрезки внутри исходного латентного представления. |
| `y`       | `INT`      | Определяет начальную y-координату области обрезки, устанавливая позицию обрезки внутри исходного латентного представления. |

## Выходы

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `latent` | `LATENT`   | Выходные данные представляют собой модифицированное латентное представление с примененной указанной обрезкой. |
