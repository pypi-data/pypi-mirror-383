Узел UNetSelfAttentionMultiply применяет коэффициенты умножения к компонентам запроса, ключа, значения и выхода механизма самовнимания в модели UNet. Он позволяет масштабировать различные части вычисления внимания для экспериментов с тем, как веса внимания влияют на поведение модели.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель UNet для модификации с коэффициентами масштабирования внимания |
| `q` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент умножения для компонента запроса (по умолчанию: 1.0) |
| `k` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент умножения для компонента ключа (по умолчанию: 1.0) |
| `v` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент умножения для компонента значения (по умолчанию: 1.0) |
| `выход` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент умножения для компонента выхода (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `MODEL` | MODEL | Модифицированная модель UNet с масштабированными компонентами внимания |
