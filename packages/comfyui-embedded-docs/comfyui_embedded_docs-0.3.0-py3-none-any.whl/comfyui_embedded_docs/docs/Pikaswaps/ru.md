Узел Pika Swaps позволяет заменять объекты или области в вашем видео новыми изображениями. Вы можете определить области для замены, используя либо маску, либо координаты, и узел бесшовно заменит указанное содержимое на протяжении всей видео последовательности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `видео` | VIDEO | Да | - | Видео, в котором требуется заменить объект. |
| `изображение` | IMAGE | Да | - | Изображение, используемое для замены замаскированного объекта в видео. |
| `маска` | MASK | Да | - | Используйте маску для определения областей в видео, которые нужно заменить. |
| `текстовый запрос` | STRING | Да | - | Текстовый запрос, описывающий желаемую замену. |
| `негативный запрос` | STRING | Да | - | Текстовый запрос, описывающий, чего следует избегать в замене. |
| `seed` | INT | Да | 0 до 4294967295 | Значение случайного seed для воспроизводимых результатов. |

**Примечание:** Этот узел требует указания всех входных параметров. `video`, `image` и `mask` работают вместе для определения операции замены, где маска указывает, какие области видео будут заменены предоставленным изображением.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Обработанное видео с замененным указанным объектом или областью. |
