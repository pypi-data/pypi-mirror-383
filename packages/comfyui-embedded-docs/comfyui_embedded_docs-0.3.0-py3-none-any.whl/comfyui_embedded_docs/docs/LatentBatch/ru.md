Узел LatentBatch предназначен для объединения двух наборов латентных сэмплов в единый пакет, при необходимости изменяя размер одного набора для соответствия размерам другого перед конкатенацией. Эта операция позволяет комбинировать различные латентные представления для последующей обработки или задач генерации.

## Входы

| Параметр    | Тип данных | Описание |
|--------------|-------------|-------------|
| `образцы1`   | `LATENT`    | Первый набор латентных сэмплов для объединения. Играет ключевую роль в определении конечной формы объединенного пакета. |
| `образцы2`   | `LATENT`    | Второй набор латентных сэмплов для объединения. Если его размеры отличаются от первого набора, он масштабируется для обеспечения совместимости перед объединением. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Объединенный набор латентных сэмплов, теперь скомбинированный в единый пакет для дальнейшей обработки. |
