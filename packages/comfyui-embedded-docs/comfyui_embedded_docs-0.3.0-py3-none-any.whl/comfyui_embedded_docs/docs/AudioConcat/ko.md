AudioConcat 노드는 두 개의 오디오 입력을 연결하여 결합합니다. 두 개의 오디오 입력을 받아 지정한 순서대로 연결하며, 두 번째 오디오를 첫 번째 오디오 앞이나 뒤에 배치할 수 있습니다. 이 노드는 모노 오디오를 스테레오로 변환하고 두 입력 간의 샘플링 속도를 일치시켜 서로 다른 오디오 형식을 자동으로 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 타입 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `audio1` | AUDIO | 필수 | - | - | 결합할 첫 번째 오디오 입력 |
| `audio2` | AUDIO | 필수 | - | - | 결합할 두 번째 오디오 입력 |
| `direction` | COMBO | 필수 | after | ['after', 'before'] | audio2를 audio1 뒤에 추가할지 앞에 추가할지 여부 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | 두 입력 오디오 파일이 함께 연결된 결합된 오디오 |
