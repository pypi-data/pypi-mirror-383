Узел AudioConcat объединяет два аудиовхода путем их соединения. Он принимает два аудиовхода и соединяет их в указанном порядке, размещая второе аудио либо до, либо после первого. Узел автоматически обрабатывает различные аудиоформаты, преобразуя моно-аудио в стерео и согласовывая частоты дискретизации между двумя входами.

## Входы

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `audio1` | AUDIO | обязательный | - | - | Первый аудиовход для объединения |
| `audio2` | AUDIO | обязательный | - | - | Второй аудиовход для объединения |
| `direction` | COMBO | обязательный | after | ['after', 'before'] | Определяет, добавлять ли audio2 после или до audio1 |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | Объединенное аудио, содержащее оба входных аудиофайла, соединенных вместе |
