Узел LotusConditioning предоставляет предварительно вычисленные эмбеддинги кондиционирования для модели Lotus. Он использует замороженный энкодер с нулевым кондиционированием и возвращает жёстко заданные эмбеддинги промптов для достижения соответствия с эталонной реализацией без необходимости выполнения вывода или загрузки больших тензорных файлов. Этот узел выводит фиксированный тензор кондиционирования, который может использоваться непосредственно в конвейере генерации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| *Нет входов* | - | - | - | Этот узел не принимает входные параметры. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | Предварительно вычисленные эмбеддинги кондиционирования для модели Lotus, содержащие фиксированные эмбеддинги промптов и пустой словарь. |
