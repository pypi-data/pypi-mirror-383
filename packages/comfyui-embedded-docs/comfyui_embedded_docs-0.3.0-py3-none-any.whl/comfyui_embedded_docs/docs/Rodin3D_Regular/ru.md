Узел Rodin 3D Regular генерирует 3D-ассеты с использованием Rodin API. Он принимает входные изображения и обрабатывает их через сервис Rodin для создания 3D-моделей. Узел управляет всем рабочим процессом — от создания задачи до загрузки окончательных файлов 3D-модели.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | Да | - | Входные изображения, используемые для генерации 3D-модели |
| `Seed` | INT | Да | - | Значение случайного seed для воспроизводимых результатов |
| `Material_Type` | STRING | Да | - | Тип материала для применения к 3D-модели |
| `Polygon_count` | STRING | Да | - | Целевое количество полигонов для генерируемой 3D-модели |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | Путь к файлу сгенерированной 3D-модели |
