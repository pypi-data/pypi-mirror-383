Узел Wan Text to Image генерирует изображения на основе текстовых описаний. Он использует модели искусственного интеллекта для создания визуального контента из текстовых промптов, поддерживая ввод текста как на английском, так и на китайском языке. Узел предоставляет различные элементы управления для настройки размера, качества и стилевых предпочтений выходного изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | "wan2.5-t2i-preview" | Модель для использования (по умолчанию: "wan2.5-t2i-preview") |
| `prompt` | STRING | Да | - | Промпт, используемый для описания элементов и визуальных особенностей, поддерживает английский/китайский языки (по умолчанию: пусто) |
| `negative_prompt` | STRING | Нет | - | Негативный текстовый промпт для указания, чего следует избегать (по умолчанию: пусто) |
| `width` | INT | Нет | 768-1440 | Ширина изображения в пикселях (по умолчанию: 1024, шаг: 32) |
| `height` | INT | Нет | 768-1440 | Высота изображения в пикселях (по умолчанию: 1024, шаг: 32) |
| `seed` | INT | Нет | 0-2147483647 | Сид (начальное значение) для использования при генерации (по умолчанию: 0) |
| `prompt_extend` | BOOLEAN | Нет | - | Следует ли улучшать промпт с помощью ИИ-ассистента (по умолчанию: True) |
| `watermark` | BOOLEAN | Нет | - | Следует ли добавлять водяной знак "Сгенерировано ИИ" к результату (по умолчанию: True) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное изображение на основе текстового промпта |
