MiniMax의 API를 사용하여 프롬프트와 선택적 매개변수를 기반으로 동기적으로 비디오를 생성합니다. 이 노드는 MiniMax의 텍스트-비디오 변환 서비스에 연결하여 텍스트 설명으로부터 비디오 콘텐츠를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `프롬프트 텍스트` | STRING | 예 | - | 비디오 생성을 안내하는 텍스트 프롬프트 |
| `모델` | COMBO | 아니오 | "T2V-01"<br>"T2V-01-Director" | 비디오 생성에 사용할 모델 (기본값: "T2V-01") |
| `시드` | INT | 아니오 | 0 ~ 18446744073709551615 | 노이즈 생성에 사용되는 랜덤 시드 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 입력 프롬프트를 기반으로 생성된 비디오 |
