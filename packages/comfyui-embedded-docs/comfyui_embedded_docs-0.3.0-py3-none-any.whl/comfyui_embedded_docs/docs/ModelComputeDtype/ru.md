Узел ModelComputeDtype позволяет изменить вычислительный тип данных, используемый моделью во время инференса. Он создает копию входной модели и применяет указанную настройку типа данных, что может помочь оптимизировать использование памяти и производительность в зависимости от возможностей вашего оборудования. Это особенно полезно для отладки и тестирования различных настроек точности.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Входная модель для модификации с новым вычислительным типом данных |
| `dtype` | STRING | Да | "default"<br>"fp32"<br>"fp16"<br>"bf16" | Вычислительный тип данных для применения к модели |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с примененным новым вычислительным типом данных |
