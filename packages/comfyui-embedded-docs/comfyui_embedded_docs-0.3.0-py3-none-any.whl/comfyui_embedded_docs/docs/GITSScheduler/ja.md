GITSSchedulerノードは、GITS（Generative Iterative Time Steps）サンプリング手法用のノイズスケジュールシグマを生成します。このノードは係数パラメータとステップ数に基づいてシグマ値を計算し、使用される総ステップ数を削減できるオプションのデノイズ係数を備えています。ノードは事前定義されたノイズレベルと補間を使用して最終的なシグマスケジュールを作成します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `係数` | FLOAT | はい | 0.80 - 1.50 | ノイズスケジュール曲線を制御する係数値（デフォルト: 1.20） |
| `ステップ` | INT | はい | 2 - 1000 | シグマを生成するサンプリングステップの総数（デフォルト: 10） |
| `ノイズ除去` | FLOAT | はい | 0.0 - 1.0 | 使用されるステップ数を削減するデノイズ係数（デフォルト: 1.0） |

**注記:** `denoise`が0.0に設定されている場合、ノードは空のテンソルを返します。`denoise`が1.0未満の場合、実際に使用されるステップ数は`round(steps * denoise)`として計算されます。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | ノイズスケジュール用に生成されたシグマ値 |
