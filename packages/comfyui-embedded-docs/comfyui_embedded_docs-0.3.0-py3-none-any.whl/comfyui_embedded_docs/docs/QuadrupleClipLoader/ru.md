# Quadruple CLIP Loader

QuadrupleCLIPLoader — один из основных узлов ComfyUI, первоначально добавленный для поддержки модели HiDream I1 version. Если этот узел отсутствует, попробуйте обновить ComfyUI до последней версии, чтобы обеспечить его поддержку.

Он требует 4 модели CLIP, соответствующих параметрам `clip_name1`, `clip_name2`, `clip_name3` и `clip_name4`, и предоставляет выходную модель CLIP для последующих узлов.

Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/text_encoders`, а также читает модели из дополнительных путей, настроенных в файле extra_model_paths.yaml. Иногда после добавления моделей может потребоваться **перезагрузить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей в соответствующей папке.
