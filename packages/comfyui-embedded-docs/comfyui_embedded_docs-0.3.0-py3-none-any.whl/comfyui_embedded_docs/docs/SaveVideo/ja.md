SaveVideoノードは、入力された動画コンテンツをComfyUIの出力ディレクトリに保存します。ファイル名のプレフィックス、動画形式、コーデックを指定することができます。このノードはカウンターによるインクリメントを含むファイル名の自動処理を行い、保存された動画にワークフローのメタデータを含めることができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `ビデオ` | VIDEO | はい | - | 保存する動画。 |
| `ファイル名プレフィックス` | STRING | いいえ | - | 保存するファイルのプレフィックス。%date:yyyy-MM-dd% や %Empty Latent Image.width% などのフォーマット情報を含めることができ、ノードからの値を含めます（デフォルト: "video/ComfyUI"）。 |
| `フォーマット` | COMBO | いいえ | 複数のオプション利用可能 | 動画を保存する形式（デフォルト: "auto"）。 |
| `コーデック` | COMBO | いいえ | 複数のオプション利用可能 | 動画に使用するコーデック（デフォルト: "auto"）。 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| *出力なし* | - | このノードは出力データを返しません。 |
