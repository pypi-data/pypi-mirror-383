Узел PrimitiveStringMultiline предоставляет многострочное текстовое поле для ввода и передачи строковых значений через ваш рабочий процесс. Он принимает текстовый ввод с несколькими строками и выводит то же самое строковое значение без изменений. Этот узел полезен, когда вам нужно ввести длинный текстовый контент или форматированный текст, занимающий несколько строк.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `значение` | STRING | Да | N/A | Текстовое входное значение, которое может занимать несколько строк |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | То же самое строковое значение, которое было предоставлено на входе |
