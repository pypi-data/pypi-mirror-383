Узел ByteDance Text to Video создает видео с использованием моделей ByteDance через API на основе текстовых промптов. Он принимает текстовое описание и различные настройки видео в качестве входных данных, а затем создает видео, соответствующее предоставленным спецификациям. Узел обрабатывает взаимодействие с API и возвращает сгенерированное видео в качестве выходных данных.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `model` | STRING | Combo | seedance_1_pro | Text2VideoModelName options | Название модели |
| `prompt` | STRING | String | - | - | Текстовый промпт, используемый для генерации видео. |
| `resolution` | STRING | Combo | - | ["480p", "720p", "1080p"] | Разрешение выходного видео. |
| `aspect_ratio` | STRING | Combo | - | ["16:9", "4:3", "1:1", "3:4", "9:16", "21:9"] | Соотношение сторон выходного видео. |
| `duration` | INT | Int | 5 | 3-12 | Длительность выходного видео в секундах. |
| `seed` | INT | Int | 0 | 0-2147483647 | Сид (начальное значение) для использования при генерации. (Опционально) |
| `camera_fixed` | BOOLEAN | Boolean | False | - | Определяет, следует ли фиксировать камеру. Платформа добавляет инструкцию по фиксации камеры к вашему промпту, но не гарантирует фактический эффект. (Опционально) |
| `watermark` | BOOLEAN | Boolean | True | - | Добавлять ли водяной знак "Сгенерировано ИИ" на видео. (Опционально) |

**Ограничения параметров:**

- Параметр `prompt` должен содержать не менее 1 символа после удаления пробелов
- Параметр `prompt` не может содержать следующие текстовые параметры: "resolution", "ratio", "duration", "seed", "camerafixed", "watermark"
- Параметр `duration` ограничен значениями от 3 до 12 секунд
- Параметр `seed` принимает значения от 0 до 2 147 483 647

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
