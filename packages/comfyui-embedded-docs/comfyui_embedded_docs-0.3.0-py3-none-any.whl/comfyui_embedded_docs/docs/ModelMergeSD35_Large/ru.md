Узел ModelMergeSD35_Large позволяет объединять две модели Stable Diffusion 3.5 Large, регулируя влияние различных компонентов моделей. Он обеспечивает точный контроль над тем, насколько каждая часть второй модели влияет на итоговую объединенную модель — от слоев эмбеддинга до совместных блоков и финальных слоев.
