Runway Image to Video (Gen4 Turbo) 노드는 Runway의 Gen4 Turbo 모델을 사용하여 단일 시작 프레임에서 비디오를 생성합니다. 텍스트 프롬프트와 초기 이미지 프레임을 입력받아 설정된 지속 시간과 화면 비율에 기반하여 비디오 시퀀스를 생성합니다. 이 노드는 시작 프레임을 Runway의 API에 업로드하는 작업을 처리하며 생성된 비디오를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 예 | - | 생성을 위한 텍스트 프롬프트 (기본값: 빈 문자열) |
| `start_frame` | IMAGE | 예 | - | 비디오에 사용될 시작 프레임 |
| `duration` | COMBO | 예 | 사용 가능한 여러 옵션 | 사용 가능한 지속 시간 옵션 중에서 비디오 길이 선택 |
| `ratio` | COMBO | 예 | 사용 가능한 여러 옵션 | 사용 가능한 Gen4 Turbo 화면 비율 옵션 중에서 선택 |
| `seed` | INT | 아니오 | 0부터 4294967295까지 | 생성을 위한 랜덤 시드 (기본값: 0) |

**매개변수 제약 조건:**

- `start_frame` 이미지의 크기는 7999x7999 픽셀을 초과할 수 없음
- `start_frame` 이미지의 화면 비율은 0.5에서 2.0 사이여야 함
- `prompt`는 최소 한 글자 이상을 포함해야 함

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 입력 프레임과 프롬프트를 기반으로 생성된 비디오 |
