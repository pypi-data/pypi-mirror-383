Узел FluxKontextMultiReferenceLatentMethod изменяет данные кондиционирования, устанавливая определённый метод обработки референсных латентов. Он добавляет выбранный метод к входным данным кондиционирования, что влияет на обработку референсных латентов на последующих этапах генерации. Этот узел помечен как экспериментальный и является частью системы кондиционирования Flux.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `conditioning` | CONDITIONING | Да | - | Данные кондиционирования, которые будут изменены с добавлением метода обработки референсных латентов |
| `reference_latents_method` | STRING | Да | `"offset"`<br>`"index"`<br>`"uxo/uno"` | Метод обработки референсных латентов. Если выбрано "uxo" или "uso", значение будет преобразовано в "uxo" |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | Изменённые данные кондиционирования с применённым методом обработки референсных латентов |
