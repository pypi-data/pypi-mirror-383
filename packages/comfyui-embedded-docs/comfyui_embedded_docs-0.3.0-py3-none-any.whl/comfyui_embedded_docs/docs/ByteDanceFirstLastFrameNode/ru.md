Этот узел создает видео с использованием текстового запроса вместе с изображениями первого и последнего кадров. Он принимает ваше описание и два ключевых кадра для создания полной видео последовательности, которая плавно переходит между ними. Узел предоставляет различные опции для управления разрешением, соотношением сторон, продолжительностью и другими параметрами генерации видео.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `model` | COMBO | комбо | seedance_1_lite | seedance_1_lite | Название модели |
| `prompt` | STRING | строка | - | - | Текстовый запрос, используемый для генерации видео. |
| `first_frame` | IMAGE | изображение | - | - | Первый кадр, который будет использоваться для видео. |
| `last_frame` | IMAGE | изображение | - | - | Последний кадр, который будет использоваться для видео. |
| `resolution` | COMBO | комбо | - | 480p, 720p, 1080p | Разрешение выходного видео. |
| `aspect_ratio` | COMBO | комбо | - | adaptive, 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 | Соотношение сторон выходного видео. |
| `duration` | INT | слайдер | 5 | 3-12 | Продолжительность выходного видео в секундах. |
| `seed` | INT | число | 0 | 0-2147483647 | Сид (начальное значение) для использования при генерации. (опционально) |
| `camera_fixed` | BOOLEAN | булевый | False | - | Определяет, следует ли фиксировать камеру. Платформа добавляет инструкцию для фиксации камеры к вашему запросу, но не гарантирует фактический эффект. (опционально) |
| `watermark` | BOOLEAN | булевый | True | - | Добавлять ли водяной знак "Сгенерировано ИИ" на видео. (опционально) |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
