Узел Differential Diffusion изменяет процесс удаления шума, применяя бинарную маску на основе пороговых значений временных шагов. Он создаёт маску, которая смешивает исходную маску удаления шума с бинарной маской на основе порога, позволяя контролируемо регулировать интенсивность процесса диффузии.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель диффузии для модификации |
| `strength` | FLOAT | Нет | 0.0 - 1.0 | Управляет силой смешивания между исходной маской удаления шума и бинарной пороговой маской (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель диффузии с обновлённой функцией маски удаления шума |
