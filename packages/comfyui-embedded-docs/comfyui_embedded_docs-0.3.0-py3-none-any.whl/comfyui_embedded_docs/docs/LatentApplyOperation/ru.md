Узел LatentApplyOperation применяет указанную операцию к латентным сэмплам. Он принимает латентные данные и операцию в качестве входных данных, обрабатывает латентные сэмплы с использованием предоставленной операции и возвращает модифицированные латентные данные. Этот узел позволяет преобразовывать или манипулировать латентными представлениями в вашем рабочем процессе.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `образцы` | LATENT | Да | - | Латентные сэмплы для обработки операцией |
| `операция` | LATENT_OPERATION | Да | - | Операция для применения к латентным сэмплам |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | LATENT | Модифицированные латентные сэмплы после применения операции |
