Узел HunyuanRefinerLatent обрабатывает кондиционирование и латентные входные данные для операций доработки. Он применяет шумовое усиление как к позитивному, так и к негативному кондиционированию, одновременно включая латентные данные изображения, и генерирует новый латентный вывод с определенными размерами для дальнейшей обработки.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Входные данные позитивного кондиционирования для обработки |
| `negative` | CONDITIONING | Да | - | Входные данные негативного кондиционирования для обработки |
| `latent` | LATENT | Да | - | Входные данные латентного представления |
| `noise_augmentation` | FLOAT | Да | 0.0 - 1.0 | Величина применяемого шумового усиления (по умолчанию: 0.10) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Обработанное позитивное кондиционирование с примененным шумовым усилением и конкатенацией латентного изображения |
| `negative` | CONDITIONING | Обработанное негативное кондиционирование с примененным шумовым усилением и конкатенацией латентного изображения |
| `latent` | LATENT | Новый латентный вывод с размерами [batch_size, 32, height, width, channels] |
