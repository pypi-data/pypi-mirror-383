Узел ModelSave сохраняет обученные или изменённые модели в хранилище вашего компьютера. Он принимает модель в качестве входных данных и записывает её в файл с указанным вами именем. Это позволяет сохранить вашу работу и повторно использовать модели в будущих проектах.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель для сохранения на диск |
| `префикс_названия_файла` | STRING | Да | - | Префикс имени файла и пути для сохраняемого файла модели (по умолчанию: "diffusion_models/ComfyUI") |
| `prompt` | PROMPT | Нет | - | Информация о промпте рабочего процесса (предоставляется автоматически) |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Дополнительные метаданные рабочего процесса (предоставляются автоматически) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| *Нет* | - | Этот узел не возвращает никаких выходных значений |
