Узел StringTrim удаляет пробельные символы из начала, конца или обеих сторон текстовой строки. Вы можете выбрать обрезку с левой стороны, правой стороны или с обеих сторон строки. Это полезно для очистки текстовых входных данных путём удаления нежелательных пробелов, табуляций или символов перевода строки.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | - | Текстовая строка для обработки. Поддерживает многострочный ввод. |
| `mode` | COMBO | Да | "Both"<br>"Left"<br>"Right" | Определяет, с какой стороны (сторон) строки производить обрезку. "Both" удаляет пробелы с обоих концов, "Left" удаляет только с начала, "Right" удаляет только с конца. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Обрезанная текстовая строка с пробелами, удалёнными в соответствии с выбранным режимом. |
