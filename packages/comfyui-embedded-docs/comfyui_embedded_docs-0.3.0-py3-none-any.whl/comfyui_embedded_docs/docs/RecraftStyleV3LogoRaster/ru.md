Этот узел позволяет выбрать растровый стиль логотипа и выбрать из доступных подстилей. Он специализируется на создании дизайнов логотипов с параметрами растрового стиля, предоставляя различные визуальные обработки для создания логотипов.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `подстиль` | STRING | Да | Доступно несколько вариантов | Конкретный подстиль растрового логотипа для применения при генерации логотипа |

## Выходы

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное изображение логотипа с применением выбранного растрового стиля |
