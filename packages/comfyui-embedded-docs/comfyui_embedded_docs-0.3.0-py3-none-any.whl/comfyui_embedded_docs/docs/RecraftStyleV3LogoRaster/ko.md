이 노드를 사용하면 로고 래스터 스타일을 선택하고 사용 가능한 서브스타일 중에서 선택할 수 있습니다. 래스터 기반 스타일링 옵션을 사용한 로고 디자인 생성에 특화되어 있으며, 로고 제작을 위한 다양한 시각적 처리를 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `하위 스타일` | STRING | 예 | 여러 옵션 사용 가능 | 로고 생성에 적용할 특정 로고 래스터 서브스타일 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 선택한 래스터 스타일이 적용된 생성된 로고 이미지 |
