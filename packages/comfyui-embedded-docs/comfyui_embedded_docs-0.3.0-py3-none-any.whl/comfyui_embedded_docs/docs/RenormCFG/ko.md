RenormCFG 노드는 분산 모델에서 분류자 없는 지도(CFG) 과정을 조건부 스케일링 및 정규화를 적용하여 수정합니다. 이 노드는 지정된 타임스텝 임계값과 재정규화 계수를 기반으로 노이즈 제거 과정을 조정하여 이미지 생성 중 조건부 예측과 무조건부 예측의 영향을 제어합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | 재정규화된 CFG를 적용할 분산 모델 |
| `cfg_trunc` | FLOAT | 아니오 | 0.0 - 100.0 | CFG 스케일링을 적용하기 위한 타임스텝 임계값 (기본값: 100.0) |
| `renorm_cfg` | FLOAT | 아니오 | 0.0 - 100.0 | 조건부 지도 강도를 제어하기 위한 재정규화 계수 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | 재정규화된 CFG 함수가 적용된 수정된 모델 |
