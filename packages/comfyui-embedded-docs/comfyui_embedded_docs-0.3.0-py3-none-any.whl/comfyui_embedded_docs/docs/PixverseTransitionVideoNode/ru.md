Генерирует видео на основе промпта и размера вывода. Этот узел создает переходные видео между двумя входными изображениями с использованием API PixVerse, позволяя задать качество видео, продолжительность, стиль движения и параметры генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `первый кадр` | IMAGE | Да | - | Начальное изображение для видео-перехода |
| `последний кадр` | IMAGE | Да | - | Конечное изображение для видео-перехода |
| `промпт` | STRING | Да | - | Промпт для генерации видео (по умолчанию: пустая строка) |
| `качество` | COMBO | Да | Доступные варианты качества из перечисления PixverseQuality<br>По умолчанию: res_540p | Настройка качества видео |
| `длительность (секунды)` | COMBO | Да | Доступные варианты продолжительности из перечисления PixverseDuration | Продолжительность видео в секундах |
| `режим движения` | COMBO | Да | Доступные варианты режима движения из перечисления PixverseMotionMode | Стиль движения для перехода |
| `seed` | INT | Да | от 0 до 2147483647 | Сид для генерации видео (по умолчанию: 0) |
| `негативный промпт` | STRING | Нет | - | Необязательное текстовое описание нежелательных элементов на изображении (по умолчанию: пустая строка) |

**Примечание:** При использовании качества 1080p режим движения автоматически устанавливается в normal, а продолжительность ограничивается 5 секундами. Для продолжительности, отличной от 5 секунд, режим движения также автоматически устанавливается в normal.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное переходное видео |
