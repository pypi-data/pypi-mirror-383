InstructPixToPixConditioning 노드는 긍정적 및 부정적 텍스트 프롬프트를 이미지 데이터와 결합하여 InstructPix2Pix 이미지 편집을 위한 조건부 데이터를 준비합니다. 입력 이미지를 VAE 인코더를 통해 처리하여 잠재 표현을 생성하고, 이러한 잠재 표현을 긍정적 및 부정적 조건부 데이터 모두에 첨부합니다. 이 노드는 VAE 인코딩 프로세스와의 호환성을 위해 이미지 크기를 8픽셀의 배수로 자동 조정하여 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `긍정 조건` | CONDITIONING | 예 | - | 원하는 이미지 특성을 위한 텍스트 프롬프트와 설정을 포함하는 긍정적 조건부 데이터 |
| `부정 조건` | CONDITIONING | 예 | - | 원하지 않는 이미지 특성을 위한 텍스트 프롬프트와 설정을 포함하는 부정적 조건부 데이터 |
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 표현으로 인코딩하는 데 사용되는 VAE 모델 |
| `픽셀` | IMAGE | 예 | - | 처리되어 잠재 공간으로 인코딩될 입력 이미지 |

**참고:** 입력 이미지 크기는 VAE 인코딩 프로세스와의 호환성을 보장하기 위해 너비와 높이 모두에서 가장 가까운 8픽셀 배수로 자동 조정됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `부정 조건` | CONDITIONING | 잠재 이미지 표현이 첨부된 긍정적 조건부 데이터 |
| `잠재 이미지` | CONDITIONING | 잠재 이미지 표현이 첨부된 부정적 조건부 데이터 |
| `latent` | LATENT | 인코딩된 이미지와 동일한 크기를 가진 빈 잠재 텐서 |
