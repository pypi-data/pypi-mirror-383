FreeU 노드는 이미지 생성 품질을 향상시키기 위해 모델의 출력 블록에 주파수 영역 수정을 적용합니다. 서로 다른 채널 그룹을 스케일링하고 특정 특징 맵에 푸리에 필터링을 적용하여 생성 과정에서 모델의 동작을 미세하게 제어할 수 있도록 합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | FreeU 수정을 적용할 모델 |
| `b1` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 4 특징에 대한 백본 스케일링 계수 (기본값: 1.1) |
| `b2` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 2 특징에 대한 백본 스케일링 계수 (기본값: 1.2) |
| `s1` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 4 특징에 대한 스킵 연결 스케일링 계수 (기본값: 0.9) |
| `s2` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 2 특징에 대한 스킵 연결 스케일링 계수 (기본값: 0.2) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | FreeU 패치가 적용된 수정된 모델 |
