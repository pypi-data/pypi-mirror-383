| `b1` | FLOAT | はい | 0.0 - 10.0 | model_channels × 4 特徴量に対するバックボーンスケーリング係数（デフォルト: 1.1） |
| `b2` | FLOAT | はい | 0.0 - 10.0 | model_channels × 2 特徴量に対するバックボーンスケーリング係数（デフォルト: 1.2） |
| `s1` | FLOAT | はい | 0.0 - 10.0 | model_channels × 4 特徴量に対するスキップ接続スケーリング係数（デフォルト: 0.9） |
| `s2` | FLOAT | はい | 0.0 - 10.0 | model_channels × 2 特徴量に対するスキップ接続スケーリング係数（デフォルト: 0.2） |
