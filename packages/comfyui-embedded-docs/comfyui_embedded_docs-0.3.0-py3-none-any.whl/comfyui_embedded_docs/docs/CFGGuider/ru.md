Узел CFGGuider создает систему управления для контроля процесса сэмплирования при генерации изображений. Он принимает модель вместе с позитивным и негативным кондиционированием, а затем применяет коэффициент классификатор-фри гайданса (CFG) для направления генерации в сторону желаемого контента, избегая нежелательных элементов. Этот узел выводит объект гайдера, который может использоваться сэмплирующими узлами для управления направлением генерации изображений.

## Входы

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `модель` | MODEL | Обязательный | - | - | Модель, используемая для гайданса |
| `положительный` | CONDITIONING | Обязательный | - | - | Позитивное кондиционирование, направляющее генерацию к желаемому контенту |
| `отрицательный` | CONDITIONING | Обязательный | - | - | Негативное кондиционирование, отводящее генерацию от нежелательного контента |
| `cfg` | FLOAT | Обязательный | 8.0 | 0.0 - 100.0 | Коэффициент классификатор-фри гайданса, который определяет силу влияния кондиционирования на генерацию |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | Объект гайдера, который может быть передан в сэмплирующие узлы для управления процессом генерации |
