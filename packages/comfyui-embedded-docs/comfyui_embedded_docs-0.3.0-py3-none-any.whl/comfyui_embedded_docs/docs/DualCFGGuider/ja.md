DualCFGGuiderノードは、デュアルクラスカーフリーガイダンスサンプリングのためのガイダンスシステムを作成します。2つのポジティブなコンディショニング入力と1つのネガティブなコンディショニング入力を組み合わせ、各コンディショニングペアに異なるガイダンススケールを適用して、生成される出力に対する各プロンプトの影響を制御します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `モデル` | MODEL | はい | - | ガイダンスに使用するモデル |
| `cond1` | CONDITIONING | はい | - | 1つ目のポジティブなコンディショニング入力 |
| `cond2` | CONDITIONING | はい | - | 2つ目のポジティブなコンディショニング入力 |
| `ネガティブ` | CONDITIONING | はい | - | ネガティブなコンディショニング入力 |
| `cfg_conds` | FLOAT | はい | 0.0 - 100.0 | 1つ目のポジティブなコンディショニングのガイダンススケール（デフォルト: 8.0） |
| `cfg_cond2_negative` | FLOAT | はい | 0.0 - 100.0 | 2つ目のポジティブおよびネガティブなコンディショニングのガイダンススケール（デフォルト: 8.0） |
| `style` | COMBO | はい | "regular"<br>"nested" | 適用するガイダンススタイル（デフォルト: "regular"） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | サンプリングで使用するための設定済みガイダンスシステム |
