Узел EmptyAudio генерирует бесшумный аудиоклип с указанной длительностью, частотой дискретизации и конфигурацией каналов. Он создает форму волны, содержащую только нули, производя полную тишину в течение заданной длительности. Этот узел полезен для создания заглушек аудио или генерации тихих сегментов в рабочих процессах обработки звука.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `duration` | FLOAT | Да | 0.0 до 1.8446744073709552e+19 | Длительность пустого аудиоклипа в секундах (по умолчанию: 60.0) |
| `sample_rate` | INT | Да | - | Частота дискретизации пустого аудиоклипа (по умолчанию: 44100) |
| `channels` | INT | Да | 1 до 2 | Количество аудиоканалов (1 для моно, 2 для стерео) (по умолчанию: 2) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | Сгенерированный бесшумный аудиоклип, содержащий данные формы волны и информацию о частоте дискретизации |
