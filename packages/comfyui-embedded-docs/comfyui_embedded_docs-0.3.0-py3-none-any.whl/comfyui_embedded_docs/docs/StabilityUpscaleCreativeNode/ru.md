Увеличивает разрешение изображения с минимальными изменениями до 4K. Этот узел использует технологию творческого увеличения разрешения от Stability AI для улучшения разрешения изображения с сохранением исходного содержания и добавлением тонких творческих деталей.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Исходное изображение для увеличения разрешения |
| `промпт` | STRING | Да | - | То, что вы хотите видеть в выходном изображении. Четкий, описательный промт, который ясно определяет элементы, цвета и объекты, приведет к лучшим результатам. (по умолчанию: пустая строка) |
| `креативность` | FLOAT | Да | 0.1-0.5 | Управляет вероятностью создания дополнительных деталей, не сильно обусловленных исходным изображением. (по умолчанию: 0.3) |
| `стиль` | COMBO | Да | Доступно несколько вариантов | Опциональный желаемый стиль генерируемого изображения. Варианты включают различные стилевые пресеты от Stability AI. |
| `seed` | INT | Да | 0-4294967294 | Случайное начальное значение, используемое для создания шума. (по умолчанию: 0) |
| `негативный промпт` | STRING | Нет | - | Ключевые слова того, что вы не хотите видеть в выходном изображении. Это расширенная функция. (по умолчанию: пустая строка) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Увеличенное изображение с разрешением 4K |
