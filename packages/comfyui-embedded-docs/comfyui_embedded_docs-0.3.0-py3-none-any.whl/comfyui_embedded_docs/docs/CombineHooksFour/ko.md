Combine Hooks [4] 노드는 최대 네 개의 별도 훅 그룹을 단일 결합 훅 그룹으로 병합합니다. 사용 가능한 네 가지 훅 입력 중 임의의 조합을 가져와 ComfyUI의 훅 결합 시스템을 사용하여 결합합니다. 이를 통해 고급 워크플로에서 여러 훅 구성을 통합하여 효율적인 처리를 가능하게 합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 타입 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `hooks_A` | HOOKS | 선택적 | 없음 | - | 결합할 첫 번째 훅 그룹 |
| `hooks_B` | HOOKS | 선택적 | 없음 | - | 결합할 두 번째 훅 그룹 |
| `hooks_C` | HOOKS | 선택적 | 없음 | - | 결합할 세 번째 훅 그룹 |
| `hooks_D` | HOOKS | 선택적 | 없음 | - | 결합할 네 번째 훅 그룹 |

**참고:** 네 가지 훅 입력 모두 선택사항입니다. 노드는 제공된 훅 그룹만 결합하며, 입력이 연결되지 않은 경우 빈 훅 그룹을 반환합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | 제공된 모든 훅 구성을 포함하는 결합된 훅 그룹 |
