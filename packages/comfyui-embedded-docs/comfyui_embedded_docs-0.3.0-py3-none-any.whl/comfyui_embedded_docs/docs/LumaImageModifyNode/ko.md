프롬프트와 종횡비를 기반으로 이미지를 동기적으로 수정합니다. 이 노드는 입력 이미지를 가져와 원본 이미지의 종횡비를 유지하면서 제공된 텍스트 프롬프트에 따라 변환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `이미지` | IMAGE | 예 | - | 수정할 입력 이미지 |
| `프롬프트` | STRING | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: "") |
| `이미지 가중치` | FLOAT | 아니오 | 0.0-0.98 | 이미지 가중치; 1.0에 가까울수록 이미지 수정 정도가 적어집니다 (기본값: 0.1) |
| `모델` | MODEL | 예 | 여러 옵션 사용 가능 | 이미지 수정에 사용할 Luma 모델 |
| `시드` | INT | 아니오 | 0-18446744073709551615 | 노드 재실행 여부를 결정하는 시드 값; 실제 결과는 시드 값과 관계없이 비결정적입니다 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `이미지` | IMAGE | Luma 모델에 의해 생성된 수정된 이미지 |
