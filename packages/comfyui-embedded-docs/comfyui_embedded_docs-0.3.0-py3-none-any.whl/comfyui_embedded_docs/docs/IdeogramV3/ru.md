Узел Ideogram V3 генерирует изображения с использованием модели Ideogram V3. Он поддерживает как обычную генерацию изображений по текстовым промптам, так и редактирование изображений, когда предоставлены и изображение, и маска. Узел предлагает различные настройки для соотношения сторон, разрешения, скорости генерации и опциональных изображений-ссылок на персонажей.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Промпт для генерации или редактирования изображения (по умолчанию: пусто) |
| `image` | IMAGE | Нет | - | Опциональное эталонное изображение для редактирования |
| `mask` | MASK | Нет | - | Опциональная маска для инпейнтинга (белые области будут заменены) |
| `aspect_ratio` | COMBO | Нет | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | Соотношение сторон для генерации изображения. Игнорируется, если разрешение не установлено в Auto (по умолчанию: "1:1") |
| `resolution` | COMBO | Нет | "Auto"<br>"1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | Разрешение для генерации изображения. Если не установлено в Auto, переопределяет настройку aspect_ratio (по умолчанию: "Auto") |
| `magic_prompt_option` | COMBO | Нет | "AUTO"<br>"ON"<br>"OFF" | Определяет, следует ли использовать MagicPrompt при генерации (по умолчанию: "AUTO") |
| `seed` | INT | Нет | 0-2147483647 | Случайное зерно для генерации (по умолчанию: 0) |
| `num_images` | INT | Нет | 1-8 | Количество генерируемых изображений (по умолчанию: 1) |
| `rendering_speed` | COMBO | Нет | "DEFAULT"<br>"TURBO"<br>"QUALITY" | Управляет балансом между скоростью генерации и качеством (по умолчанию: "DEFAULT") |
| `character_image` | IMAGE | Нет | - | Изображение для использования в качестве эталона персонажа |
| `character_mask` | MASK | Нет | - | Опциональная маска для эталонного изображения персонажа |

**Ограничения параметров:**

- При одновременном предоставлении `image` и `mask` узел переключается в режим редактирования
- Если предоставлен только один из параметров `image` или `mask`, произойдет ошибка
- `character_mask` требует наличия `character_image`
- Параметр `aspect_ratio` игнорируется, когда `resolution` не установлен в "Auto"
- Белые области маски будут заменены во время инпейнтинга
- Маска персонажа и изображение персонажа должны быть одного размера

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное или отредактированное изображение(я) |
