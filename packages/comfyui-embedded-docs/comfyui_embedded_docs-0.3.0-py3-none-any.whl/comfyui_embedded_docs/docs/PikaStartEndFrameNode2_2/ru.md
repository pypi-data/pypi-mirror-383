Узел PikaFrames v2.2 создает видео, комбинируя ваш первый и последний кадр. Вы загружаете два изображения, чтобы определить начальную и конечную точки, а искусственный интеллект создает плавный переход между ними для получения готового видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image_start` | IMAGE | Да | - | Первое изображение для комбинирования. |
| `image_end` | IMAGE | Да | - | Последнее изображение для комбинирования. |
| `prompt_text` | STRING | Да | - | Текстовый запрос, описывающий желаемое содержимое видео. |
| `negative_prompt` | STRING | Да | - | Текст, описывающий, чего следует избегать в видео. |
| `seed` | INT | Да | - | Значение случайного сида для воспроизводимости генерации. |
| `resolution` | STRING | Да | - | Разрешение выходного видео. |
| `duration` | INT | Да | - | Длительность генерируемого видео. |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео, объединяющее начальный и конечный кадры с переходами, созданными ИИ. |
