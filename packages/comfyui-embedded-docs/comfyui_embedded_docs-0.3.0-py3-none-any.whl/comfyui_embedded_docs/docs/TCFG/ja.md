TCFG（Tangential Damping CFG）は、条件付き（ポジティブ）予測により良く整合するように、無条件（ネガティブ）予測を洗練するガイダンス技術を実装します。この手法は、参照研究論文2503.18137に基づき、無条件ガイダンスに接線減衰を適用することで出力品質を向上させます。このノードは、クラスサー不要ガイダンスプロセスにおいて無条件予測が処理される方法を調整することで、モデルのサンプリング動作を変更します。
