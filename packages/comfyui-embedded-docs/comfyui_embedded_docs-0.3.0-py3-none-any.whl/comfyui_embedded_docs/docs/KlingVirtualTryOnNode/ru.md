Узел Kling Virtual Try On. Позволяет загрузить изображение человека и изображение одежды, чтобы примерить одежду на человека. Вы можете объединить несколько изображений предметов одежды в одно изображение с белым фоном.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `human_image` | IMAGE | Да | - | Изображение человека, на которого примеряется одежда |
| `cloth_image` | IMAGE | Да | - | Изображение одежды, которую нужно примерить на человека |
| `model_name` | STRING | Да | `"kolors-virtual-try-on-v1"` | Модель для виртуальной примерки (по умолчанию: "kolors-virtual-try-on-v1") |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Результирующее изображение, показывающее человека с надетым предметом одежды |
