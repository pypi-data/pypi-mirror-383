ImageAddNoise 노드는 입력 이미지에 무작위 노이즈를 추가합니다. 지정된 랜덤 시드를 사용하여 일관된 노이즈 패턴을 생성하며, 노이즈 효과의 강도를 제어할 수 있습니다. 결과 이미지는 입력과 동일한 크기를 유지하지만 시각적 텍스처가 추가됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 노이즈를 추가할 입력 이미지 |
| `seed` | INT | 예 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 랜덤 시드 (기본값: 0) |
| `strength` | FLOAT | 예 | 0.0부터 1.0까지 | 노이즈 효과의 강도를 제어합니다 (기본값: 0.5) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 노이즈가 적용된 출력 이미지 |
