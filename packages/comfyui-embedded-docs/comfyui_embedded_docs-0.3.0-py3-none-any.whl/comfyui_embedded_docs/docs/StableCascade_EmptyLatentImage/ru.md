Узел StableCascade_EmptyLatentImage создает пустые латентные тензоры для моделей Stable Cascade. Он генерирует два отдельных латентных представления — одно для стадии C и другое для стадии B — с соответствующими размерностями на основе входного разрешения и настроек сжатия. Этот узел предоставляет отправную точку для конвейера генерации Stable Cascade.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `ширина` | INT | Да | 256 до MAX_RESOLUTION | Ширина выходного изображения в пикселях (по умолчанию: 1024, шаг: 8) |
| `высота` | INT | Да | 256 до MAX_RESOLUTION | Высота выходного изображения в пикселях (по умолчанию: 1024, шаг: 8) |
| `сжатие` | INT | Да | 4 до 128 | Коэффициент сжатия, определяющий латентные размерности для стадии C (по умолчанию: 42, шаг: 1) |
| `размер_пакета` | INT | Нет | 1 до 4096 | Количество латентных образцов для генерации в пакете (по умолчанию: 1) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `этап_b` | LATENT | Латентный тензор стадии C с размерностями [batch_size, 16, height//compression, width//compression] |
| `stage_b` | LATENT | Латентный тензор стадии B с размерностями [batch_size, 4, height//4, width//4] |
