LTXVImgToVideo 노드는 입력 이미지를 비디오 생성 모델을 위한 비디오 잠재 표현으로 변환합니다. 단일 이미지를 가져와 VAE 인코더를 사용하여 프레임 시퀀스로 확장한 후, 비디오 생성 과정에서 원본 이미지 콘텐츠가 보존되는 정도와 수정되는 정도를 결정하는 강도 제어를 통해 조건화를 적용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `긍정 조건` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 긍정적 조건화 프롬프트 |
| `부정 조건` | CONDITIONING | 예 | - | 비디오에서 특정 요소를 배제하기 위한 부정적 조건화 프롬프트 |
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `이미지` | IMAGE | 예 | - | 비디오 프레임으로 변환될 입력 이미지 |
| `너비` | INT | 아니오 | 64 ~ MAX_RESOLUTION | 출력 비디오의 너비(픽셀) (기본값: 768, 단계: 32) |
| `높이` | INT | 아니오 | 64 ~ MAX_RESOLUTION | 출력 비디오의 높이(픽셀) (기본값: 512, 단계: 32) |
| `길이` | INT | 아니오 | 9 ~ MAX_RESOLUTION | 생성될 비디오의 프레임 수 (기본값: 97, 단계: 8) |
| `배치 크기` | INT | 아니오 | 1 ~ 4096 | 동시에 생성할 비디오 수 (기본값: 1) |
| `강도` | FLOAT | 아니오 | 0.0 ~ 1.0 | 비디오 생성 과정에서 원본 이미지가 수정되는 정도를 제어합니다. 1.0은 원본 콘텐츠를 대부분 보존하고 0.0은 최대 수정을 허용합니다 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `부정 조건` | CONDITIONING | 비디오 프레임 마스킹이 적용된 처리된 긍정적 조건화 |
| `잠재 데이터` | CONDITIONING | 비디오 프레임 마스킹이 적용된 처리된 부정적 조건화 |
| `latent` | LATENT | 인코딩된 프레임과 비디오 생성을 위한 노이즈 마스크를 포함하는 비디오 잠재 표현 |
