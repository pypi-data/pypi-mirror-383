Узел Vidu Text To Video Generation создает видео из текстовых описаний. Он использует различные модели генерации видео для преобразования ваших текстовых промптов в видеоконтент с настраиваемыми параметрами продолжительности, соотношения сторон и визуального стиля.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `vidu_q1`<br>*Другие варианты VideoModelName* | Название модели (по умолчанию: vidu_q1) |
| `prompt` | STRING | Да | - | Текстовое описание для генерации видео |
| `duration` | INT | Нет | 5-5 | Продолжительность выходного видео в секундах (по умолчанию: 5) |
| `seed` | INT | Нет | 0-2147483647 | Сид для генерации видео (0 для случайного) (по умолчанию: 0) |
| `aspect_ratio` | COMBO | Нет | `r_16_9`<br>*Другие варианты AspectRatio* | Соотношение сторон выходного видео (по умолчанию: r_16_9) |
| `resolution` | COMBO | Нет | `r_1080p`<br>*Другие варианты Resolution* | Поддерживаемые значения могут различаться в зависимости от модели и продолжительности (по умолчанию: r_1080p) |
| `movement_amplitude` | COMBO | Нет | `auto`<br>*Другие варианты MovementAmplitude* | Амплитуда движения объектов в кадре (по умолчанию: auto) |

**Примечание:** Поле `prompt` является обязательным и не может быть пустым. Параметр `duration` в настоящее время фиксирован на значении 5 секунд.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе текстового промпта |
