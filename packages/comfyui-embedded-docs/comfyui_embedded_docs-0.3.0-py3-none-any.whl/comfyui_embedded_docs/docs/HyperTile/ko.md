HyperTile 노드는 확산 모델의 어텐션 메커니즘에 타일링 기법을 적용하여 이미지 생성 중 메모리 사용을 최적화합니다. 잠재 공간을 더 작은 타일로 분할하여 개별적으로 처리한 후 결과를 재조립합니다. 이를 통해 메모리 부족 없이 더 큰 이미지 크기로 작업할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | HyperTile 최적화를 적용할 확산 모델 |
| `타일 크기` | INT | 아니오 | 1-2048 | 처리에 사용할 목표 타일 크기 (기본값: 256) |
| `스왑 크기` | INT | 아니오 | 1-128 | 처리 중 타일 재배열 방식을 제어합니다 (기본값: 2) |
| `최대 깊이` | INT | 아니오 | 0-10 | 타일링을 적용할 최대 깊이 수준 (기본값: 0) |
| `스케일 깊이` | BOOLEAN | 아니오 | - | 깊이 수준에 따라 타일 크기를 스케일링할지 여부 (기본값: False) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | HyperTile 최적화가 적용된 수정된 모델 |
