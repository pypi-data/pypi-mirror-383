Узел Morphology применяет различные морфологические операции к изображениям — это математические операции, используемые для обработки и анализа форм на изображениях. Он может выполнять такие операции, как эрозия, дилатация, открытие, закрытие и другие, с использованием настраиваемого размера ядра для управления силой эффекта.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для обработки |
| `операция` | STRING | Да | `"erode"`<br>`"dilate"`<br>`"open"`<br>`"close"`<br>`"gradient"`<br>`"bottom_hat"`<br>`"top_hat"` | Применяемая морфологическая операция |
| `размер_ядра` | INT | Нет | 3-999 | Размер ядра структурирующего элемента (по умолчанию: 3) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Обработанное изображение после применения морфологической операции |
