Узел Combine Hooks [2] объединяет две группы хуков в одну комбинированную группу хуков. Он принимает два опциональных входа с хуками и объединяет их с помощью функциональности комбинирования хуков ComfyUI. Это позволяет консолидировать несколько конфигураций хуков для оптимизированной обработки.

## Входы

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `hooks_A` | HOOKS | Опциональный | None | - | Первая группа хуков для объединения |
| `hooks_B` | HOOKS | Опциональный | None | - | Вторая группа хуков для объединения |

**Примечание:** Оба входа являются опциональными, но для работы узла должна быть предоставлена хотя бы одна группа хуков. Если предоставлена только одна группа хуков, она будет возвращена без изменений.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `hooks` | HOOKS | Комбинированная группа хуков, содержащая все хуки из обеих входных групп |
