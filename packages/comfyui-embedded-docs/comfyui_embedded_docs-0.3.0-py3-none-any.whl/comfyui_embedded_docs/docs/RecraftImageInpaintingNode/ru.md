Этот узел изменяет изображения на основе текстового запроса и маски. Он использует API Recraft для интеллектуального редактирования определённых областей изображения, которые вы задаёте с помощью маски, оставляя остальную часть изображения неизменной.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Исходное изображение для изменения |
| `mask` | MASK | Да | - | Маска, определяющая, какие области изображения должны быть изменены |
| `запрос` | STRING | Да | - | Запрос для генерации изображения (по умолчанию: пустая строка) |
| `n` | INT | Да | 1-6 | Количество генерируемых изображений (по умолчанию: 1, минимум: 1, максимум: 6) |
| `seed` | INT | Да | 0-18446744073709551615 | Сид для определения, должен ли узел выполняться повторно; фактические результаты недетерминированы независимо от сида (по умолчанию: 0, минимум: 0, максимум: 18446744073709551615) |
| `recraft_style` | STYLEV3 | Нет | - | Дополнительный параметр стиля для API Recraft |
| `негативный запрос` | STRING | Нет | - | Дополнительное текстовое описание нежелательных элементов на изображении (по умолчанию: пустая строка) |

*Примечание: Параметры `image` и `mask` должны быть предоставлены вместе для работы операции восстановления. Маска будет автоматически масштабирована в соответствии с размерами изображения.*

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Изменённое изображение (изображения), сгенерированное на основе запроса и маски |
