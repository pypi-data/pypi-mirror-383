Set Hook Keyframesノードは、既存のフックグループにキーフレームスケジューリングを適用することができます。このノードはフックグループを受け取り、オプションでキーフレームのタイミング情報を適用して、生成プロセス中に異なるフックがいつ実行されるかを制御します。キーフレームが提供された場合、ノードはフックグループをクローンし、グループ内のすべてのフックにキーフレームタイミングを設定します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `フック` | HOOKS | はい | - | キーフレームスケジューリングが適用されるフックグループ |
| `フック_kf` | HOOK_KEYFRAMES | いいえ | - | フック実行のタイミング情報を含むオプションのキーフレームグループ |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `フック` | HOOKS | キーフレームスケジューリングが適用された修正済みフックグループ（キーフレームが提供された場合はクローンされます） |
