Set Hook Keyframes 노드를 사용하면 기존 훅 그룹에 키프레임 스케줄링을 적용할 수 있습니다. 이 노드는 훅 그룹을 입력받고, 선택적으로 키프레임 타이밍 정보를 적용하여 생성 과정 중 다양한 훅이 실행되는 시점을 제어합니다. 키프레임이 제공되면, 노드는 훅 그룹을 복제하고 그룹 내 모든 훅에 키프레임 타이밍을 설정합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `후크` | HOOKS | 예 | - | 키프레임 스케줄링이 적용될 훅 그룹 |
| `KF 후크` | HOOK_KEYFRAMES | 아니요 | - | 훅 실행을 위한 타이밍 정보를 포함하는 선택적 키프레임 그룹 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `후크` | HOOKS | 키프레임 스케줄링이 적용된 수정된 훅 그룹 (키프레임이 제공된 경우 복제됨) |
