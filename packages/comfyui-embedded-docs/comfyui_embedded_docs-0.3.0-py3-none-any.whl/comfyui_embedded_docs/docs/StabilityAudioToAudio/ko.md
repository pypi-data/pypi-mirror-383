> 이 문서는 AI에 의해 생성되었습니다. 오류를 발견하거나 개선 제안이 있으시면 기여해 주세요! [GitHub에서 편집](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/StabilityAudioToAudio/ko.md)

텍스트 지시를 사용하여 기존 오디오 샘플을 새로운 고품질 작품으로 변환합니다. 이 노드는 입력 오디오 파일을 가져와 사용자의 텍스트 프롬프트를 기반으로 수정하여 새로운 오디오 콘텐츠를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | "stable-audio-2.5"<br> | 오디오 변환에 사용할 AI 모델 |
| `prompt` | STRING | 예 |  | 오디오를 어떻게 변환할지 설명하는 텍스트 지시 (기본값: 빈 값) |
| `audio` | AUDIO | 예 |  | 오디오는 6초에서 190초 사이여야 합니다 |
| `duration` | INT | 아니오 | 1-190 | 생성된 오디오의 지속 시간(초)을 제어합니다 (기본값: 190) |
| `seed` | INT | 아니오 | 0-4294967294 | 생성을 위해 사용되는 랜덤 시드 (기본값: 0) |
| `steps` | INT | 아니오 | 4-8 | 샘플링 단계 수를 제어합니다 (기본값: 8) |
| `strength` | FLOAT | 아니오 | 0.01-1.0 | audio 매개변수가 생성된 오디오에 미치는 영향 정도를 제어하는 매개변수 (기본값: 1.0) |

**참고:** 입력 오디오는 6초에서 190초 사이의 지속 시간이어야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 입력 오디오와 텍스트 프롬프트를 기반으로 생성된 변환된 오디오 |
