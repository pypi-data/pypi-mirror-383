SaveAudioノードは、オーディオデータをFLAC形式のファイルとして保存します。オーディオ入力を受け取り、指定された出力ディレクトリに所定のファイル名プレフィックスで書き込みます。このノードはファイル名の自動処理を行い、後続の使用のためにオーディオが適切に保存されることを保証します。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `オーディオ` | AUDIO | はい | - | 保存するオーディオデータ |
| `ファイル名_プレフィックス` | STRING | いいえ | - | 出力ファイル名のプレフィックス（デフォルト: "audio/ComfyUI"） |

*注: `prompt`および`extra_pnginfo`パラメータは非表示であり、システムによって自動的に処理されます*

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| *なし* | - | このノードは出力データを返しませんが、オーディオファイルを出力ディレクトリに保存します |
