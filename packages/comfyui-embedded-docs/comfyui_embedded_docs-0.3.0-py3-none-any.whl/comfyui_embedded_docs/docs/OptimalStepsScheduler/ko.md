OptimalStepsScheduler 노드는 선택된 모델 유형과 단계 구성을 기반으로 확산 모델의 노이즈 스케줄 시그마 값을 계산합니다. 디노이즈 매개변수에 따라 총 단계 수를 조정하고 요청된 단계 수에 맞게 노이즈 수준을 보간합니다. 이 노드는 확산 샘플링 과정에서 사용되는 노이즈 수준을 결정하는 일련의 시그마 값을 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델 종류` | COMBO | 예 | "FLUX"<br>"Wan"<br>"Chroma" | 노이즈 수준 계산에 사용할 확산 모델의 유형 |
| `스텝` | INT | 예 | 3-1000 | 계산할 샘플링 단계의 총 수 (기본값: 20) |
| `디노이즈` | FLOAT | 아니오 | 0.0-1.0 | 디노이징 강도를 제어하며, 유효 단계 수를 조정합니다 (기본값: 1.0) |

**참고:** `denoise`가 1.0 미만으로 설정된 경우, 노드는 유효 단계를 `steps * denoise`로 계산합니다. `denoise`가 0.0으로 설정된 경우, 노드는 빈 텐서를 반환합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | 확산 샘플링을 위한 노이즈 스케줄을 나타내는 일련의 시그마 값 |
