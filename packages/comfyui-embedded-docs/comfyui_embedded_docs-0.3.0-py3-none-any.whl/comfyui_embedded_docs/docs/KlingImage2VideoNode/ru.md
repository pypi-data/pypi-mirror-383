Узел Kling Image to Video создает видеоконтент из исходного изображения с использованием текстовых промптов. Он принимает опорное изображение и создает видеопоследовательность на основе предоставленных позитивных и негативных текстовых описаний, с различными параметрами конфигурации для выбора модели, длительности и соотношения сторон.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `start_frame` | IMAGE | Да | - | Опорное изображение, используемое для генерации видео. |
| `prompt` | STRING | Да | - | Позитивный текстовый промпт. |
| `negative_prompt` | STRING | Да | - | Негативный текстовый промпт. |
| `model_name` | COMBO | Да | Доступно несколько вариантов | Выбор модели для генерации видео (по умолчанию: "kling-v2-master"). |
| `cfg_scale` | FLOAT | Да | 0.0-1.0 | Параметр масштаба конфигурации (по умолчанию: 0.8). |
| `mode` | COMBO | Да | Доступно несколько вариантов | Выбор режима генерации видео (по умолчанию: std). |
| `aspect_ratio` | COMBO | Да | Доступно несколько вариантов | Соотношение сторон для генерируемого видео (по умолчанию: field_16_9). |
| `duration` | COMBO | Да | Доступно несколько вариантов | Длительность генерируемого видео (по умолчанию: field_5). |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео. |
| `video_id` | STRING | Уникальный идентификатор для сгенерированного видео. |
| `duration` | STRING | Информация о длительности сгенерированного видео. |
