Узел StringCompare сравнивает две текстовые строки, используя различные методы сравнения. Он может проверять, начинается ли одна строка с другой, заканчивается ли она другой, или являются ли обе строки точно одинаковыми. Сравнение может выполняться с учетом или без учета разницы в регистре букв.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string_a` | STRING | Да | - | Первая строка для сравнения |
| `string_b` | STRING | Да | - | Вторая строка, с которой производится сравнение |
| `mode` | COMBO | Да | "Starts With"<br>"Ends With"<br>"Equal" | Используемый метод сравнения |
| `case_sensitive` | BOOLEAN | Нет | - | Учитывать ли регистр букв при сравнении (по умолчанию: true) |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | BOOLEAN | Возвращает true, если условие сравнения выполнено, и false в противном случае |
