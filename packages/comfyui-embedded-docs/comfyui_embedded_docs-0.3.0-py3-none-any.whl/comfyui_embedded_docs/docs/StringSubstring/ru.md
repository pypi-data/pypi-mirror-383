Узел StringSubstring извлекает часть текста из более крупной строки. Он принимает начальную и конечную позиции для определения нужного участка и возвращает текст между этими двумя позициями.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | - | Исходная текстовая строка для извлечения |
| `start` | INT | Да | - | Индекс начальной позиции для подстроки |
| `end` | INT | Да | - | Индекс конечной позиции для подстроки |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Извлеченная подстрока из исходного текста |
