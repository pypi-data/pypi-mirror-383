Узел IdeogramV1 генерирует изображения с использованием модели Ideogram V1 через API. Он принимает текстовые промпты и различные настройки генерации для создания одного или нескольких изображений на основе вашего ввода. Узел поддерживает различные соотношения сторон и режимы генерации для настройки результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Промпт для генерации изображения (по умолчанию: пусто) |
| `turbo` | BOOLEAN | Да | - | Использовать ли турбо-режим (более быстрая генерация, потенциально более низкое качество) (по умолчанию: False) |
| `aspect_ratio` | COMBO | Нет | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | Соотношение сторон для генерации изображения (по умолчанию: "1:1") |
| `magic_prompt_option` | COMBO | Нет | "AUTO"<br>"ON"<br>"OFF" | Определяет, следует ли использовать MagicPrompt в генерации (по умолчанию: "AUTO") |
| `seed` | INT | Нет | 0-2147483647 | Значение случайного сида для генерации (по умолчанию: 0) |
| `negative_prompt` | STRING | Нет | - | Описание того, что следует исключить из изображения (по умолчанию: пусто) |
| `num_images` | INT | Нет | 1-8 | Количество изображений для генерации (по умолчанию: 1) |

**Примечание:** Параметр `num_images` имеет максимальное ограничение в 8 изображений на один запрос генерации.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированные изображения от модели Ideogram V1 |
