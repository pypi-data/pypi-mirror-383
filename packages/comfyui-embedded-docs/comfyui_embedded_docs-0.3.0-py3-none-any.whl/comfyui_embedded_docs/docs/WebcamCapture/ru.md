Узел WebcamCapture захватывает изображения с веб-камеры и преобразует их в формат, который может использоваться в рабочих процессах ComfyUI. Он наследуется от узла LoadImage и предоставляет опции для управления размерами захвата и временными параметрами. При включении узел может захватывать новые изображения каждый раз при обработке очереди рабочего процесса.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | WEBCAM | Да | - | Источник ввода с веб-камеры для захвата изображений |
| `ширина` | INT | Нет | 0 до MAX_RESOLUTION | Желаемая ширина захваченного изображения (по умолчанию: 0, использует собственное разрешение веб-камеры) |
| `высота` | INT | Нет | 0 до MAX_RESOLUTION | Желаемая высота захваченного изображения (по умолчанию: 0, использует собственное разрешение веб-камеры) |
| `захват_в_очереди` | BOOLEAN | Нет | - | При включении захватывает новое изображение каждый раз при обработке очереди рабочего процесса (по умолчанию: True) |

**Примечание:** Когда оба параметра `width` и `height` установлены в 0, узел использует собственное разрешение веб-камеры. Установка любого измерения в ненулевое значение соответствующим образом изменит размер захваченного изображения.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Захваченное изображение с веб-камеры, преобразованное в формат изображения ComfyUI |
