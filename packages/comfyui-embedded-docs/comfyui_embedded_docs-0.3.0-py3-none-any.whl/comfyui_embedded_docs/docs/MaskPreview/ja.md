MaskPreviewノードは、マスクを3チャンネルの画像形式に変換し、一時ファイルとして保存することで、マスクの視覚的なプレビューを生成します。マスク入力を取得し、画像表示に適した形式に再形成した後、ランダムなファイル名プレフィックスを使用して一時ディレクトリに結果を保存します。これにより、ワークフローの実行中にマスクデータを視覚的に確認することができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `mask` | MASK | はい | - | プレビューされ、画像形式に変換されるマスクデータ |
| `filename_prefix` | STRING | いいえ | - | 出力ファイル名のプレフィックス（デフォルト: "ComfyUI"） |
| `prompt` | PROMPT | いいえ | - | メタデータ用のプロンプト情報（自動的に提供されます） |
| `extra_pnginfo` | EXTRA_PNGINFO | いいえ | - | メタデータ用の追加PNG情報（自動的に提供されます） |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `ui` | DICT | 表示用のプレビュー画像情報とメタデータを含みます |
