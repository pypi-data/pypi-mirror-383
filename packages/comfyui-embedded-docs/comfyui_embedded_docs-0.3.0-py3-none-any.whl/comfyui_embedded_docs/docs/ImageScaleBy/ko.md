ImageScaleBy 노드는 다양한 보간 방법을 사용하여 지정된 배율로 이미지를 업스케일링하도록 설계되었습니다. 이는 다양한 업스케일링 요구에 맞춰 이미지 크기를 유연하게 조정할 수 있게 해줍니다.

## 입력

| 매개변수         | 데이터 유형   | 설명                                                                                                                   |
| ---------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `이미지`          | `IMAGE`       | 업스케일링할 입력 이미지입니다. 이 매개변수는 업스케일링 과정을 거칠 기본 이미지를 제공합니다.                         |
| `확대 방법` | COMBO[STRING] | 업스케일링에 사용할 보간 방법을 지정합니다. 방법의 선택은 업스케일링된 이미지의 품질과 특성에 영향을 미칠 수 있습니다. |
| `배율`       | `FLOAT`       | 이미지가 업스케일링될 배율을 나타냅니다. 이는 입력 이미지에 비해 출력 이미지의 크기 증가를 결정합니다.                 |

## 출력

| 매개변수 | 데이터 유형 | 설명                                                                         |
| -------- | ----------- | ---------------------------------------------------------------------------- |
| `이미지`  | `IMAGE`     | 지정된 배율과 보간 방법에 따라 입력 이미지보다 큰 업스케일링된 이미지입니다. |
