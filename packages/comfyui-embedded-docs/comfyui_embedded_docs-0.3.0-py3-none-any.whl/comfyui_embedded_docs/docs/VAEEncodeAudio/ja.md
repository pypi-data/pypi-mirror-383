# VAEEncodeAudio

VAEEncodeAudioノードは、変分オートエンコーダー（VAE）を使用してオーディオデータを潜在表現に変換します。このノードはオーディオ入力を取り、VAEを通して処理し、さらなるオーディオ生成や操作タスクに使用できる圧縮された潜在サンプルを生成します。エンコード前に必要に応じてオーディオを44100 Hzに自動的にリサンプリングします。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `オーディオ` | AUDIO | はい | - | エンコードするオーディオデータ（波形とサンプルレート情報を含む） |
| `vae` | VAE | はい | - | オーディオを潜在空間にエンコードするために使用する変分オートエンコーダーモデル |

**注記:** 元のサンプルレートが44100 Hzと異なる場合、オーディオ入力は自動的に44100 Hzにリサンプリングされます。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | 潜在空間におけるエンコードされたオーディオ表現（圧縮されたサンプルを含む） |
