# /help 命令文档

## 用途
显示可用命令和用法。

## 触发条件
用户请求帮助时使用此命令。

## AI执行逻辑
直接执行帮助命令。

## AI执行脚本
当AI识别到用户输入 `/help` 命令时，需要在命令行执行以下操作：

```bash
# 显示帮助信息
mkf-help

# 或者使用完整路径执行
.moke-core/bin/mkf-help
```

**执行说明：**
- 使用 `mkf-help` 命令显示所有可用命令和用法
- 命令会列出工作流引擎支持的所有命令及其描述
- 提供详细的命令使用示例和参数说明

## 示例
```
/help
```
